A React hook that monitors real-time connection status to the OpenFrame platform, including authentication state and AI configuration loading.

## Key Components

- **`useConnectionStatus()`** - Main hook that returns connection status, server URL, AI configuration, and loading state
- **`ConnectionStatus`** - Type defining connection states: 'connected', 'disconnected', 'connecting'
- **`AIConfiguration`** - Interface for AI model configuration including provider, model name, and API key status
- **`UseConnectionStatusReturn`** - Return type interface with all hook outputs

## Usage Example

```typescript
import { useConnectionStatus } from './hooks/useConnectionStatus'

function ConnectionIndicator() {
  const { status, serverUrl, aiConfiguration, isFullyLoaded } = useConnectionStatus()
  
  return (
    <div>
      <div>Status: {status}</div>
      <div>Server: {serverUrl}</div>
      {isFullyLoaded && aiConfiguration && (
        <div>AI Model: {aiConfiguration.modelName} ({aiConfiguration.provider})</div>
      )}
    </div>
  )
}
```

The hook manages token authentication, establishes NATS WebSocket connections for real-time communication, fetches AI configuration from the server, and provides a unified interface for monitoring the complete connection state of the OpenFrame platform.
