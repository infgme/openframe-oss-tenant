A React hook that manages the selection and loading of Mingo AI dialog conversations, handling both dialog metadata and message pagination with real-time background message integration.

## Key Components

- **`useMingoDialogSelection`** - Main hook that orchestrates dialog selection, message loading, and state management
- **Dialog Query** - Fetches dialog metadata using React Query with caching
- **Messages Infinite Query** - Handles paginated message loading with automatic pagination
- **Select Dialog Mutation** - Manages dialog switching with proper state cleanup
- **Message Merging Logic** - Combines fetched messages with background/real-time messages

## Usage Example

```typescript
import { useMingoDialogSelection } from './hooks/use-mingo-dialog-selection'

function DialogViewer() {
  const {
    selectDialog,
    isSelectingDialog,
    isLoadingDialog,
    isLoadingMessages,
    rawMessagesCount,
    dialogError,
    messagesError,
    refetchDialog,
    refetchMessages
  } = useMingoDialogSelection()

  const handleDialogSelect = (dialogId: string) => {
    selectDialog(dialogId)
  }

  if (isLoadingDialog) return <div>Loading dialog...</div>
  if (dialogError) return <div>Error: {dialogError}</div>

  return (
    <div>
      <p>Messages loaded: {rawMessagesCount}</p>
      <button 
        onClick={() => handleDialogSelect('dialog-123')}
        disabled={isSelectingDialog}
      >
        Select Dialog
      </button>
    </div>
  )
}
```

The hook automatically handles message deduplication, merges background messages with fetched ones, and maintains proper chronological ordering while providing loading states and error handling.
