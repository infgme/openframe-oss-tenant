A modal dialog component that prompts users to resend email verification when their email address hasn't been verified yet. Built using AlertDialog components from the OpenFrame UI library with custom Flamingo design system styling.

## Key Components

- **EmailVerificationModal** - Main modal component that displays verification prompt
- **EmailVerificationModalProps** - Interface defining required props including open state, email address, and submission handler
- **AlertDialog components** - Uses AlertDialog, AlertDialogContent, AlertDialogHeader, AlertDialogTitle, AlertDialogDescription, AlertDialogFooter, AlertDialogCancel, and AlertDialogAction from the UI library

## Usage Example

```typescript
import { EmailVerificationModal } from './email-verification-modal'
import { useState } from 'react'

function MyComponent() {
  const [isModalOpen, setIsModalOpen] = useState(false)
  const [isSending, setIsSending] = useState(false)
  
  const handleResendVerification = async () => {
    setIsSending(true)
    try {
      // Call API to resend verification email
      await resendVerificationEmail()
    } finally {
      setIsSending(false)
      setIsModalOpen(false)
    }
  }

  return (
    <EmailVerificationModal
      open={isModalOpen}
      onOpenChange={setIsModalOpen}
      userEmail="user@example.com"
      onSubmit={handleResendVerification}
      isSending={isSending}
    />
  )
}
```

The modal features custom Flamingo design tokens (ods-*) for consistent styling, highlights the user's email address in warning color, and provides loading state handling for the resend operation.
