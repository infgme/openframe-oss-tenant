A data structure for deserializing API responses from agent registration endpoints, containing authentication credentials and machine identification.

## Key Components

- **AgentRegistrationResponse** - Main response struct with camelCase JSON field mapping
- **machine_id** - Unique identifier for the registered machine
- **client_id** - OAuth client identifier for authentication
- **client_secret** - OAuth client secret for secure authentication

## Usage Example

```rust
use serde_json;

// Deserialize JSON response from registration API
let json_response = r#"
{
    "machineId": "machine-12345",
    "clientId": "client-abc123",
    "clientSecret": "secret-xyz789"
}
"#;

let response: AgentRegistrationResponse = serde_json::from_str(json_response)?;

println!("Registered machine: {}", response.machine_id);
println!("Client ID: {}", response.client_id);
// Note: Don't log client_secret in production
```

The struct automatically maps camelCase JSON fields to snake_case Rust field names through the `#[serde(rename_all = "camelCase")]` attribute, making it easy to work with typical REST API responses while following Rust naming conventions.
