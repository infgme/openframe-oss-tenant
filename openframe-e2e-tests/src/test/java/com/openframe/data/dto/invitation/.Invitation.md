Data Transfer Object for managing user invitations in the OpenFrame platform, providing invitation lifecycle management with role assignments and expiration tracking.

## Key Components

- **Fields**:
  - `id` - Unique invitation identifier
  - `email` - Target user's email address
  - `roles` - List of roles to be assigned upon acceptance
  - `expiresAt` - Invitation expiration timestamp
  - `status` - Current invitation state (defaults to PENDING)
  - `createdAt` / `updatedAt` - Audit timestamps

- **Annotations**:
  - `@Data` - Generates getters, setters, equals, hashCode, and toString
  - `@Builder` - Enables fluent builder pattern
  - `@NoArgsConstructor` / `@AllArgsConstructor` - Default and full constructors

## Usage Example

```java
// Create a new invitation with builder pattern
Invitation invitation = Invitation.builder()
    .email("user@example.com")
    .roles(List.of(UserRole.VIEWER, UserRole.TECHNICIAN))
    .expiresAt(Instant.now().plusSeconds(604800)) // 7 days
    .status(InvitationStatus.PENDING)
    .createdAt(Instant.now())
    .build();

// Update invitation status
invitation.setStatus(InvitationStatus.ACCEPTED);
invitation.setUpdatedAt(Instant.now());

// Check if invitation is expired
boolean isExpired = invitation.getExpiresAt().isBefore(Instant.now());
```
