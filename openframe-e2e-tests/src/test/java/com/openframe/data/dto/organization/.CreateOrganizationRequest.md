Data Transfer Object for creating new organizations in the OpenFrame platform. This request DTO contains all necessary fields for organization creation, with the organization ID automatically generated as a UUID on the backend.

## Key Components

- **`name`** - Organization name (required field)
- **`category`** - Business category classification
- **`numberOfEmployees`** - Employee count for sizing purposes
- **`websiteUrl`** - Organization's website URL
- **`notes`** - Additional notes or comments
- **`contactInformation`** - Embedded contact details (ContactInformationDto)
- **`monthlyRevenue`** - Revenue information as string
- **`contractStartDate`** - Contract start date
- **`contractEndDate`** - Contract end date

## Usage Example

```java
CreateOrganizationRequest request = CreateOrganizationRequest.builder()
    .name("Acme Corporation")
    .category("Technology")
    .numberOfEmployees(150)
    .websiteUrl("https://acme.com")
    .notes("High-priority client with 24/7 support needs")
    .contactInformation(contactInfo)
    .monthlyRevenue("50000")
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2025, 12, 31))
    .build();
```

The DTO uses Lombok annotations for automatic getter/setter generation and builder pattern support, making it easy to construct requests with only the required fields populated.
