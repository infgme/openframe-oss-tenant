A specialized service for handling initial configuration during installation, with support for both local and production environments including automatic SSL certificate detection.

## Key Components

- **`InstallationInitialConfigService`** - Main service that wraps InitialConfigurationService for installation-specific logic
- **`InstallConfigParams`** - Configuration parameters structure containing server URL, initial key, org ID, and local mode flag
- **`build_and_save()`** - Core method that validates parameters, builds configuration, and optionally resolves mkcert SSL certificates for local development

## Usage Example

```rust
use crate::platform::DirectoryManager;

// Initialize the installation config service
let directory_manager = DirectoryManager::new()?;
let install_service = InstallationInitialConfigService::new(directory_manager)?;

// Configure for production environment
let prod_params = InstallConfigParams {
    server_url: Some("https://api.flamingo.com".to_string()),
    initial_key: Some("prod-key-123".to_string()),
    org_id: Some("org-456".to_string()),
    local_mode: false,
};

install_service.build_and_save(prod_params)?;

// Configure for local development with SSL
let local_params = InstallConfigParams {
    server_url: Some("https://localhost:8080".to_string()),
    initial_key: Some("dev-key-789".to_string()),
    org_id: Some("dev-org".to_string()),
    local_mode: true, // Automatically resolves mkcert SSL certificates
};

install_service.build_and_save(local_params)?;
```
