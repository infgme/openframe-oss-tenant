A service for managing installed tools data persistence in JSON format, providing CRUD operations for tool storage and retrieval.

## Key Components

- **InstalledToolsService** - Main service struct that manages tool persistence to a JSON file in the secured directory
- **save()** - Saves or updates an installed tool, replacing existing entries with matching tool_agent_id
- **get_by_tool_agent_id()** - Retrieves a specific tool by its identifier
- **get_all()** - Returns all installed tools from storage
- **delete_by_tool_agent_id()** - Removes a tool from storage and returns whether deletion occurred
- **persist()** - Private helper for writing tool data to JSON file

## Usage Example

```rust
use crate::services::InstalledToolsService;
use crate::models::InstalledTool;
use crate::platform::directories::DirectoryManager;

// Initialize the service
let dir_manager = DirectoryManager::new()?;
let tools_service = InstalledToolsService::new(dir_manager)?;

// Save a new tool
let tool = InstalledTool {
    tool_agent_id: "my-tool-123".to_string(),
    // ... other fields
};
tools_service.save(tool).await?;

// Retrieve a specific tool
let found_tool = tools_service
    .get_by_tool_agent_id("my-tool-123")
    .await?;

// Get all installed tools
let all_tools = tools_service.get_all().await?;

// Delete a tool
let was_deleted = tools_service
    .delete_by_tool_agent_id("my-tool-123")
    .await?;
```

The service automatically handles file creation, JSON serialization/deserialization, and ensures the secured directory exists before operations.
