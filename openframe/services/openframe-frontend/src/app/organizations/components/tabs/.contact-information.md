A React form component for capturing comprehensive contact information including primary, billing, and technical contacts plus address details.

## Key Components

- **`ContactInfoState`** - TypeScript interface defining the complete contact information data structure with fields for three contact types and address information
- **`ContactInformationTab`** - Main form component that renders organized contact input fields with automatic mailing address synchronization
- **`set`** - Helper function for partial state updates using the spread operator pattern

## Usage Example

```typescript
import { ContactInformationTab, ContactInfoState } from './contact-information'

function CustomerForm() {
  const [contactInfo, setContactInfo] = useState<ContactInfoState>({
    primaryName: '',
    primaryTitle: '',
    primaryPhone: '',
    primaryEmail: '',
    billingName: '',
    billingTitle: '',
    billingPhone: '',
    billingEmail: '',
    technicalName: '',
    technicalTitle: '',
    technicalPhone: '',
    technicalEmail: '',
    physicalAddress: '',
    mailingAddress: '',
    mailingSameAsPhysical: false
  })

  return (
    <ContactInformationTab 
      value={contactInfo} 
      onChange={setContactInfo} 
    />
  )
}
```

The component features a responsive grid layout that automatically syncs the mailing address with the physical address when the "same as physical" checkbox is enabled. All form inputs use consistent OpenFrame UI styling with proper accessibility labels and placeholder text.
