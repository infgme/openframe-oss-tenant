A reusable modal component for confirming the removal of expired user invitations with destructive action styling.

## Key Components

- **ConfirmRemoveInvitationModal** - Main modal component that displays a confirmation dialog
- **ConfirmRemoveInvitationModalProps** - TypeScript interface defining the component's props
- **AlertDialog components** - UI primitives from the OpenFrame design system for dialog structure

## Props

| Prop | Type | Description |
|------|------|-------------|
| `open` | `boolean` | Controls modal visibility state |
| `onOpenChange` | `(open: boolean) => void` | Callback for open state changes |
| `userEmail` | `string` | Email address to display in confirmation message |
| `onConfirm` | `() => Promise<void> \| void` | Callback executed when user confirms removal |

## Usage Example

```typescript
import { ConfirmRemoveInvitationModal } from './confirm-remove-invitation-modal'
import { useState } from 'react'

function InvitationManager() {
  const [showModal, setShowModal] = useState(false)
  const [selectedEmail, setSelectedEmail] = useState('')

  const handleRemoveInvitation = async () => {
    await removeInvitationAPI(selectedEmail)
    setShowModal(false)
    // Refresh invitation list
  }

  return (
    <>
      <button onClick={() => {
        setSelectedEmail('user@example.com')
        setShowModal(true)
      }}>
        Remove Invitation
      </button>
      
      <ConfirmRemoveInvitationModal
        open={showModal}
        onOpenChange={setShowModal}
        userEmail={selectedEmail}
        onConfirm={handleRemoveInvitation}
      />
    </>
  )
}
```

The modal features destructive styling with red error colors for the confirm button and highlighted email text to emphasize the permanent nature of the action.
