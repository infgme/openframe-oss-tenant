A MongoDB collection wrapper that provides static methods for querying user documents from the "users" collection with various filtering options.

## Key Components

- **findUser()** - Retrieves the first user document without filters
- **findUser(String id)** - Finds a user by their unique ID
- **findUser(UserStatus status)** - Finds an active admin user with the specified status
- **findUser(UserRole role)** - Finds an active user with the specified role
- **findUser(UserStatus status, UserRole role)** - Finds a user matching both status and role criteria

## Usage Example

```java
// Find any user
AuthUser anyUser = UsersCollection.findUser();

// Find user by ID
AuthUser specificUser = UsersCollection.findUser("user123");

// Find active admin user
AuthUser adminUser = UsersCollection.findUser(UserStatus.ACTIVE);

// Find user by role
AuthUser moderator = UsersCollection.findUser(UserRole.MODERATOR);

// Find user with specific status and role
AuthUser activeAdmin = UsersCollection.findUser(UserStatus.ACTIVE, UserRole.ADMIN);
```

The class uses MongoDB's `Filters` API to construct queries and returns `AuthUser` objects. All methods return `null` if no matching user is found. The collection leverages the static `getCollection` method from the MongoDB utility class for database access.
