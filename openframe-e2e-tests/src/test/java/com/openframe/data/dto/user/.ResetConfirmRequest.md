A data transfer object (DTO) that represents a password reset confirmation request containing a token and new password for completing the password reset process.

## Key Components

- **token** - The password reset token that validates the reset request
- **newPassword** - The new password to be set for the user account
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a reset confirmation request using builder pattern
ResetConfirmRequest request = ResetConfirmRequest.builder()
    .token("eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...")
    .newPassword("newSecurePassword123!")
    .build();

// Creating using constructor
ResetConfirmRequest request = new ResetConfirmRequest(
    "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "newSecurePassword123!"
);

// Accessing fields
String resetToken = request.getToken();
String password = request.getNewPassword();

// Setting fields
request.setToken("new-token-value");
request.setNewPassword("anotherPassword");
```

This DTO is typically used in REST API endpoints for handling the second step of a password reset flow, where users submit their reset token along with their desired new password.
