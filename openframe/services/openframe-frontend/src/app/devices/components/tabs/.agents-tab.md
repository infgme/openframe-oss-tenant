A React component that displays management agents installed on a device in a responsive grid layout with detailed agent information and connection status.

## Key Components

- **AgentsTab**: Main component that renders agent information cards in a responsive grid
- **agentTypeToToolType**: Mapping object that converts agent types to standardized tool types
- **Agent data merging**: Combines installed agents with tool connections to show complete agent status
- **Responsive grid**: Displays agents in 1-3 column layout based on screen size
- **Agent cards**: Shows agent ID, version, and connection status with tooltips

## Usage Example

```typescript
import { AgentsTab } from './agents-tab'

// Device with agents and tool connections
const deviceData = {
  id: 'device-123',
  installedAgents: [
    {
      agentType: 'tacticalrmm-agent',
      version: '2.5.0'
    }
  ],
  toolConnections: [
    {
      toolType: 'TACTICAL_RMM',
      agentToolId: 'agent-456'
    }
  ]
}

export function DeviceDetail() {
  return (
    <div className="device-tabs">
      <AgentsTab device={deviceData} />
    </div>
  )
}
```

The component automatically handles cases where agents are installed but not connected, or connected but not showing in installed agents list. Each agent card displays a tool badge, info tooltip, and relevant details like version and agent ID.
