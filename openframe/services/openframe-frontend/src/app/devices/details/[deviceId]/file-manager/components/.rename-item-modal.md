A modal component for renaming items in the Flamingo OpenFrame interface. Provides a simple input field with keyboard shortcuts and loading states for item renaming operations.

## Key Components

- **RenameItemModal** - Main modal component that handles item renaming workflow
- **RenameItemModalProps** - TypeScript interface defining component props including state handlers and callbacks
- **Keyboard Event Handler** - useEffect hook that enables Enter key submission when modal is open

## Usage Example

```typescript
import { RenameItemModal } from './rename-item-modal'

    }
  }

  return (
    <RenameItemModal
      isOpen={isRenameOpen}
      value={itemName}
      submitting={isSubmitting}
      onChange={setItemName}
      onSubmit={handleRename}
      onClose={() => setIsRenameOpen(false)}
    />
  )
}
```

The component includes automatic focus management, Enter key submission, input validation (requires non-empty trimmed value), and proper loading states during the rename operation.
