A React modal component that provides remote shell access to devices through MeshCentral tunnels with an integrated xterm.js terminal interface.

## Key Components

- **RemoteShellModal**: Main modal component that establishes secure shell connections to remote devices
- **MeshTunnel**: Handles WebSocket communication with MeshCentral for remote command execution
- **MeshControlClient**: Manages authentication and session control for MeshCentral connections
- **Terminal Integration**: Uses xterm.js with FitAddon for a full-featured terminal experience
- **PowerShell Support**: Automatic detection and initialization of PowerShell sessions on Windows devices

## Usage Example

```typescript
import { RemoteShellModal } from './remote-shell-modal'

function DeviceManagement() {
  const [showShell, setShowShell] = useState(false)
  const [selectedDevice, setSelectedDevice] = useState<string>('')

  const openShell = (deviceId: string) => {
    setSelectedDevice(deviceId)
    setShowShell(true)
  }

  return (
    <div>
      <button onClick={() => openShell('device-123')}>
        Open Remote Shell
      </button>
      
      <RemoteShellModal
        isOpen={showShell}
        onClose={() => setShowShell(false)}
        deviceId={selectedDevice}
        deviceLabel="Production Server"
        shellType="powershell" // or "cmd"
      />
    </div>
  )
}
```

The component automatically handles terminal resizing, session management, and provides real-time feedback on connection status. It supports both Command Prompt and PowerShell environments with automatic PowerShell initialization when specified.
