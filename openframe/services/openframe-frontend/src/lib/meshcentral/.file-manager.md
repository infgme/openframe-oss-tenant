A TypeScript class that provides comprehensive file management capabilities for MeshCentral, handling file operations, transfers, and directory navigation through secure tunnels.

## Key Components

- **MeshCentralFileManager** - Main class orchestrating file operations
- **File Operations** - Directory listing, file operations, search functionality
- **Transfer Management** - Upload/download handling with progress tracking
- **Tunnel Management** - Secure connection establishment and maintenance
- **State Management** - Connection states and operation tracking
- **Error Handling** - Comprehensive error management and user feedback

## Usage Example

```typescript
import { MeshCentralFileManager } from './file-manager'

// Initialize file manager for remote device
const fileManager = new MeshCentralFileManager({
  isRemote: true,
  nodeId: 'device-123',
  authCookie: 'auth-token',
  controlClient: meshControlClient,
  onStateChange: (state) => console.log('Connection state:', state),
  onTransferProgress: (progress) => console.log('Transfer progress:', progress),
  onError: (error) => console.error('File operation error:', error)
})

// Connect and navigate
await fileManager.connect()
await fileManager.loadDirectory('/C:/Users')

// Search for files
const results = await fileManager.searchFiles('*.pdf')

// Upload/download files
await fileManager.uploadFile(file, '/remote/path/')
await fileManager.downloadFile('/remote/file.txt')
```

The file manager supports both local and remote file operations, handles binary protocols for efficient transfers, and provides comprehensive error handling with real-time progress updates for all file operations.
