A simple data transfer object representing a tag entity used for categorizing and organizing devices within the OpenFrame platform.

## Key Components

- **Tag Class**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **Core Properties**:
  - `id` - Unique identifier for the tag
  - `name` - Display name of the tag
  - `description` - Optional description text
  - `color` - Color code for visual representation
  - `organizationId` - Links tag to specific organization
  - `createdAt` - Timestamp of creation
  - `createdBy` - User who created the tag

## Usage Example

```java
// Create a new tag using builder pattern
Tag deviceTag = Tag.builder()
    .id("tag-001")
    .name("Production Server")
    .description("Servers used in production environment")
    .color("#FF5722")
    .organizationId("org-123")
    .createdAt("2024-01-15T10:30:00Z")
    .createdBy("admin@company.com")
    .build();

// Access properties
String tagName = deviceTag.getName();
String tagColor = deviceTag.getColor();

// Create using constructor
Tag simpleTag = new Tag("tag-002", "Development", "Dev environment", "#4CAF50", "org-123", "2024-01-15T11:00:00Z", "dev@company.com");
```

This DTO is typically used for REST API responses and database entity mapping in device management operations.
