A React hook that manages dialog status updates with API communication and user feedback through toast notifications.

## Key Components

- **useDialogStatus()** - Main hook that provides dialog status management functionality
- **updateDialogStatus()** - Core function that updates dialog status via API and handles responses
- **putOnHold()** - Convenience method to set dialog status to 'ON_HOLD'
- **resolve()** - Convenience method to set dialog status to 'RESOLVED'
- **activate()** - Convenience method to set dialog status to 'ACTIVE'
- **isUpdating** - Boolean state that tracks whether an update operation is in progress

The hook includes error handling, prevents concurrent updates, and provides user feedback through toast messages. It integrates with the Flamingo API client and toast notification system.

## Usage Example

```typescript
import { useDialogStatus } from './hooks/use-dialog-status'

function DialogActions({ dialogId }: { dialogId: string }) {
  const { putOnHold, resolve, activate, isUpdating } = useDialogStatus()

  const handleResolve = async () => {
    const success = await resolve(dialogId)
    if (success) {
      // Dialog was successfully resolved
      console.log('Dialog resolved!')
    }
  }

  return (
    <div>
      <button onClick={() => putOnHold(dialogId)} disabled={isUpdating}>
        Put on Hold
      </button>
      <button onClick={handleResolve} disabled={isUpdating}>
        Resolve
      </button>
      <button onClick={() => activate(dialogId)} disabled={isUpdating}>
        Activate
      </button>
    </div>
  )
}
```
