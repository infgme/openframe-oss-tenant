TypeScript type definitions for the Flamingo chat messaging system, providing interfaces for messages, pagination, and API responses.

## Key Components

- **`Message`** - Core message interface with dialog association, chat type, owner information, and flexible message data
- **`MessageConnection`** - GraphQL-style connection interface for paginated message lists with cursor-based edges
- **`MessagesResponse`** - API response wrapper for message queries
- **`MessagePage`** - Simplified pagination interface for processed message data

## Usage Example

```typescript
import type { Message, MessagePage, MessagesResponse } from './message.types'

// Creating a new message
const message: Message = {
  id: 'msg_123',
  dialogId: 'dialog_456',
  chatType: 'SUPPORT',
  dialogMode: 'standard',
  createdAt: '2024-01-15T10:30:00Z',
  owner: {
    type: 'USER',
    model: 'gpt-4'
  },
  messageData: {
    text: 'Hello, I need help with my account',
    attachments: []
  }
}

// Processing API response
const processMessages = (response: MessagesResponse): MessagePage => ({
  messages: response.data.messages.edges.map(edge => edge.node),
  pageInfo: response.data.messages.pageInfo
})
```
