A React component that provides a comprehensive dashboard overview of system logs with statistics cards and a recent logs list.

## Key Components

- **LogsOverviewSection**: Main component displaying log statistics and recent entries
- **ExtendedLogEntry**: Type extending base LogEntry with navigation data
- **Statistics Cards**: Info, Warning, and Critical log counters with percentages
- **Recent Logs List**: Displays the 10 most recent log entries with click navigation
- **Responsive Layout**: Two-column grid layout that adapts to screen size

## Usage Example

```typescript
import { LogsOverviewSection } from './logs-overview'

// Use in dashboard page
function DashboardPage() {
  return (
    <div className="dashboard">
      <LogsOverviewSection />
    </div>
  )
}

// The component automatically:
// - Fetches log data using useLogsOverview and useLogs hooks
// - Displays total log count and severity breakdowns
// - Shows recent logs with click-to-navigate functionality
// - Handles loading states and responsive design
```

The component leverages the OpenFrame frontend core for UI components (`DashboardInfoCard`, `LogsList`) and utilities, while providing seamless navigation to detailed log views through the integrated log navigation system.
