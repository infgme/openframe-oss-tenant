Main page component for the Mingo AI chat interface in Flamingo's OpenFrame platform, providing a real-time conversational AI experience with multi-dialog management and NATS-based messaging.

## Key Components

- **Mingo()** - Primary page component managing the complete chat interface
- **Dialog Management** - Multiple custom hooks for creating, selecting, and managing chat dialogs
- **Real-time Processing** - NATS subscription system for live message streaming and background updates
- **State Management** - Zustand stores for dialog details and background message handling
- **Message System** - Optimistic updates, welcome messages, and typing indicators

## Usage Example

```typescript
// Rendered as a Next.js page at /mingo route
export default function Mingo() {
  // Component automatically handles:
  // - Dialog creation and selection via URL params
  // - Real-time message streaming via NATS
  // - Background dialog notifications
  // - Optimistic message updates
  // - Welcome message injection
}

// URL structure: /mingo?dialogId=abc123
// Keyboard shortcuts: ESC to close current dialog
// SaaS tenant mode required for access
```

The component integrates multiple specialized hooks and stores to provide a seamless chat experience with features like persistent NATS subscriptions per dialog, automatic welcome messages, unread count management, and optimistic UI updates for better perceived performance.
