A React component that displays detailed log information in a structured card layout with labeled rows for the OpenFrame platform.

## Key Components

- **FullInformationSection**: Main component that renders log entry details in a formatted card
- **LogEntry**: TypeScript interface defining the structure of log data including tool events, timestamps, severity levels, and metadata
- **FullInformationSectionProps**: Component props interface accepting optional log details
- **formatTimestamp**: Utility function that safely converts timestamp strings to ISO format

## Usage Example

```typescript
import { FullInformationSection } from './full-information-section'

const logData = {
  toolEventId: "evt_12345",
  eventType: "ERROR", 
  ingestDay: "2024-01-15",
  toolType: "monitoring",
  severity: "high",
  userId: "user_789",
  deviceId: "device_456",
  message: "System alert detected",
  timestamp: "2024-01-15T10:30:00Z",
  details: "Additional context information"
}

function LogViewer() {
  return (
    <div className="p-4">
      <FullInformationSection logDetails={logData} />
    </div>
  )
}
```

The component uses OpenFrame's design system components (`InfoRow`, `ToolBadge`) and styling tokens for consistent presentation. It gracefully handles missing data by showing conditional fields and a fallback state when no log details are provided.
