A file manager container component that provides remote file access through Flamingo's OpenFrame platform with full CRUD operations and real-time progress tracking.

## Key Components

- **FileManagerContainer** - Main container component that orchestrates file operations for remote devices
- **useMeshFileManager** - Custom hook for managing file operations through MeshCentral agents
- **Modal Components** - NewFolderModal, RenameItemModal, DeleteConfirmationModal for user interactions
- **Progress Tracking** - Real-time upload/download progress with cancellation support
- **Keyboard Shortcuts** - Standard file manager shortcuts (Ctrl+C, Ctrl+V, Delete, etc.)

## Usage Example

```typescript
// Basic implementation for remote device file management
<FileManagerContainer
  deviceId="device-123"
  meshcentralAgentId="agent-456"
  hostname="Production-Server-01"
  organizationName="Acme Corp"
/>

// Minimal setup for device file access
<FileManagerContainer
  deviceId="device-789"
  meshcentralAgentId="agent-101"
/>
```

## Key Features

- **Remote File Operations** - Full CRUD operations through MeshCentral integration
- **Real-time Progress** - Upload/download progress with cancellation capabilities  
- **Search & Navigation** - File search with breadcrumb navigation
- **Clipboard Operations** - Copy, cut, paste with visual feedback
- **Multi-selection** - Batch operations on selected files
- **Connection Management** - Automatic reconnection and state handling
- **Responsive Design** - Mobile-optimized file management interface

The component integrates with Flamingo's device management system to provide secure remote file access for MSP operations.
