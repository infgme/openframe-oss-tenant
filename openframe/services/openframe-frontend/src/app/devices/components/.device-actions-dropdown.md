A React client component that renders an actions dropdown menu for device management with context-aware options and confirmation dialogs.

## Key Components

- **DeviceActionsDropdown** - Main dropdown component with configurable actions based on device type and context
- **Action Handlers** - Methods for remote control, script execution, file management, archival, and deletion
- **Confirmation Dialogs** - AlertDialog components for destructive actions (archive/delete)
- **Uninstall Modal** - Modal with command display and copy functionality
- **Context-Aware Rendering** - Different action sets for table vs detail view contexts

## Usage Example

```typescript
import { DeviceActionsDropdown } from './device-actions-dropdown'

function DeviceTable() {
  const devices = useDevices()

  return (
    <div>
      {devices.map(device => (
        <div key={device.id} className="device-row">
          <span>{device.name}</span>
          <DeviceActionsDropdown 
            device={device}
            context="table"
            onActionComplete={() => refetchDevices()}
            onRemoteControl={() => openRemoteControlModal(device)}
            onRunScript={() => openScriptModal(device)}
            onRemoteShell={(type) => openShellModal(device, type)}
          />
        </div>
      ))}
    </div>
  )
}

// Usage in device detail view
function DeviceDetailPage({ device }) {
  return (
    <div className="device-header">
      <h1>{device.name}</h1>
      <DeviceActionsDropdown 
        device={device}
        context="detail"
        onActionComplete={() => router.push('/devices')}
      />
    </div>
  )
}
```

The component dynamically shows/hides actions based on device capabilities (MeshCentral agent availability, platform type, etc.) and provides different action sets for table rows versus detail pages. All destructive actions include confirmation dialogs with proper loading states.
