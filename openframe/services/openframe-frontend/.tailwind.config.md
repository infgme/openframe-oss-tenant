Tailwind CSS configuration file that extends the OpenFrame core preset with custom fonts and animation support for a Flamingo Stack application.

## Key Components

- **OpenFrame Core Preset**: Imports and uses the base Tailwind configuration from `@flamingo-stack/openframe-frontend-core`
- **Content Paths**: Configures Tailwind to scan both local source files and OpenFrame core components
- **Custom Font Family**: Extends the theme with DM Sans body font and Azeret Mono monospace font using CSS variables
- **Animation Plugin**: Includes `tailwindcss-animate` for enhanced animation utilities

## Usage Example

This configuration automatically provides all OpenFrame Design System (ODS) colors and utilities while adding custom typography:

```typescript
// In your React components, you can use:
<div className="font-body text-primary bg-surface-elevated">
  <h1 className="font-mono">OpenFrame App</h1>
  <p className="animate-fade-in">Content with custom fonts</p>
</div>
```

The preset system ensures consistency across Flamingo Stack applications while allowing project-specific customizations. The content paths include both local source files and the OpenFrame core package, enabling Tailwind to generate styles for all components used in the application.
