Manages real-time message processing for Mingo AI chat dialogs, handling streaming messages, tool executions, and approval workflows across active and background conversations.

## Key Components

- **`useMingoRealtimeProcessor`** - Main hook that processes real-time chunks from NATS messages
- **`processChunk`** - Processes incoming chunk data and routes to appropriate dialog
- **`processDialogChunk`** - Handles chunk processing for specific dialogs with state management
- **`DialogState`** - Tracks streaming state for each dialog
- **`resetDialog`** - Cleans up dialog state when conversation ends
- **`cleanup`** - Removes all dialog states on unmount

## Usage Example

```typescript
import { useMingoRealtimeProcessor } from './use-mingo-realtime-processor'

function ChatComponent() {
  const processor = useMingoRealtimeProcessor({
    activeDialogId: 'dialog-123',
    onActiveStreamStart: () => setIsStreaming(true),
    onActiveStreamEnd: () => setIsStreaming(false),
    onActiveError: (error) => showError(error),
    onBackgroundStreamStart: (dialogId) => markAsActive(dialogId),
    onBackgroundStreamEnd: (dialogId) => markAsInactive(dialogId),
    onBackgroundUnreadIncrement: (dialogId) => incrementUnread(dialogId),
  })

  // Process incoming NATS messages
  useNatsSubscription((chunk, messageType) => {
    processor.processChunk(chunk, messageType)
  })

  return <ChatInterface />
}
```

The hook distinguishes between active (currently viewed) and background dialogs, managing streaming states, message routing, and unread counts. It processes various action types including text messages, tool executions, and approval requests/results.
