A modal component for creating new folders with name input validation and keyboard shortcuts.

## Key Components

- **NewFolderModal**: Main modal component that handles folder creation workflow
- **NewFolderModalProps**: TypeScript interface defining component props for state management
- **Keyboard handling**: Enter key submission with validation checks
- **UI components**: Uses OpenFrame UI library components (Modal, Button, Input)

## Usage Example

```typescript
import { NewFolderModal } from './new-folder-modal'

function FileManager() {
  const [isModalOpen, setIsModalOpen] = useState(false)
  const [folderName, setFolderName] = useState('')
  const [isSubmitting, setIsSubmitting] = useState(false)

  const handleCreateFolder = async () => {
    setIsSubmitting(true)
    try {
      await createFolder(folderName)
      setIsModalOpen(false)
      setFolderName('')
    } finally {
      setIsSubmitting(false)
    }
  }

  return (
    <NewFolderModal
      isOpen={isModalOpen}
      folderName={folderName}
      submitting={isSubmitting}
      onChange={setFolderName}
      onSubmit={handleCreateFolder}
      onClose={() => setIsModalOpen(false)}
    />
  )
}
```

The modal automatically focuses the input field, validates folder names (requires non-empty trimmed text), and supports Enter key submission. The create button shows loading state during submission and is disabled when the folder name is invalid or while processing.
