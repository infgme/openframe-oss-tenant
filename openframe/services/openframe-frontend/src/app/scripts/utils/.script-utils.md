Utility functions for mapping platform identifiers between different naming conventions used in scripts and device filtering systems.

## Key Components

- `mapPlatformsToOsTypes()` - Converts script platform names to device filter OS types
- `mapPlatformsForDisplay()` - Converts platform names to user-friendly display formats

## Usage Example

```typescript
import { mapPlatformsToOsTypes, mapPlatformsForDisplay } from './script-utils';

// Convert script platforms to filter values
const scriptPlatforms = ['windows', 'darwin', 'linux'];
const osTypes = mapPlatformsToOsTypes(scriptPlatforms);
// Returns: ['WINDOWS', 'MAC_OS']

// Convert platforms for UI display
const displayNames = mapPlatformsForDisplay(scriptPlatforms);
// Returns: ['Windows', 'macOS', 'Linux']

// Handle case-insensitive input
const mixedCase = ['Windows', 'DARWIN'];
const normalized = mapPlatformsForDisplay(mixedCase);
// Returns: ['Windows', 'macOS']
```

The functions handle case-insensitive input and filter out unsupported platforms. Note that Linux is supported for display formatting but filtered out in OS type mapping since the device filter system doesn't include Linux as a filterable option.
