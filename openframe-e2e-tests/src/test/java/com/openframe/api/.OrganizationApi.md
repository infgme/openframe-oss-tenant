REST API test utility for organization management operations using RestAssured framework. Provides methods for CRUD operations on organizations through both GraphQL and REST endpoints.

## Key Components

### Static Methods

- **`getOrganizationNames()`** - Fetches all organization names via GraphQL query
- **`getOrganizationIds()`** - Retrieves all organization IDs via GraphQL query  
- **`retrieveOrganization(String id)`** - Gets full organization details by ID using GraphQL
- **`createOrganization(CreateOrganizationRequest request)`** - Creates new organization via REST POST
- **`updateOrganization(String id, CreateOrganizationRequest request)`** - Updates existing organization via REST PUT
- **`deleteOrganization(Organization organization)`** - Deletes organization via REST DELETE

### Constants

- **`GRAPHQL`** - GraphQL endpoint path (`"api/graphql"`)
- **`ORGANIZATIONS`** - REST API endpoint path (`"api/organizations"`)

## Usage Example

```java
// Get all organization names
List<String> orgNames = OrganizationApi.getOrganizationNames();

// Create a new organization
CreateOrganizationRequest request = new CreateOrganizationRequest("Acme Corp");
Organization newOrg = OrganizationApi.createOrganization(request);

// Retrieve organization details
Organization org = OrganizationApi.retrieveOrganization(newOrg.getId());

// Update organization
CreateOrganizationRequest updateRequest = new CreateOrganizationRequest("Acme Corporation");
Organization updatedOrg = OrganizationApi.updateOrganization(org.getId(), updateRequest);

// Delete organization
OrganizationApi.deleteOrganization(updatedOrg);
```

The class uses `RequestSpecHelper.getAuthorizedSpec()` for authentication and imports GraphQL queries from `OrganizationQueries` class. All methods return appropriate response objects or void for deletion operations.
