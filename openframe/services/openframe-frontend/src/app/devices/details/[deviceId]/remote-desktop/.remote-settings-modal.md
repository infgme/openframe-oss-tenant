A React modal component for configuring remote desktop connection settings including quality, scaling, frame rate, and input preferences.

## Key Components

- **RemoteSettingsModal** - Main modal component with form controls for remote desktop settings
- **RemoteSettingsModalProps** - Interface defining component props including settings, connection refs, and callbacks
- **Form Controls** - Quality, scaling, frame rate selectors and mouse/keyboard checkboxes
- **Settings Management** - Integration with `RemoteDesktopSettings` class for applying configurations
- **Connection Validation** - Ensures settings are only applied when remote connection is established

## Usage Example

```typescript
import { RemoteSettingsModal } from './remote-settings-modal'
import { RemoteSettingsConfig } from '@lib/meshcentral/remote-settings'

function RemoteDesktopComponent() {
  const [settingsOpen, setSettingsOpen] = useState(false)
  const [currentSettings, setCurrentSettings] = useState<RemoteSettingsConfig>({
    quality: 60,
    scaling: 100,
    frameRate: 'auto',
    swapMouseButtons: false,
    useRemoteKeyboardMap: false
  })
  const desktopRef = useRef<MeshDesktop | null>(null)
  const tunnelRef = useRef<MeshTunnel | null>(null)
  const [connectionState, setConnectionState] = useState(0)

  return (
    <RemoteSettingsModal
      open={settingsOpen}
      onOpenChange={setSettingsOpen}
      currentSettings={currentSettings}
      desktopRef={desktopRef}
      tunnelRef={tunnelRef}
      connectionState={connectionState}
      onSettingsChange={setCurrentSettings}
    />
  )
}
```

The modal validates connection state before applying settings and provides user feedback through toast notifications. Settings include bandwidth estimation and are immediately applied to the active remote desktop session.
