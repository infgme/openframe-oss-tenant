A data transfer object representing an authenticated user in the OpenFrame system, containing essential user information, authentication details, and authorization data.

## Key Components

**Core Fields:**
- `id` - Unique user identifier
- `tenantId` - Multi-tenant organization identifier
- `email`, `firstName`, `lastName` - Basic user profile information

**Authentication Fields:**
- `passwordHash` - Encrypted password storage
- `emailVerified` - Email verification status flag
- `loginProvider` - Authentication provider (e.g., local, OAuth)

**Authorization Fields:**
- `roles` - List of assigned user roles for permission management
- `status` - Current user account status

**Lombok Annotations:**
- `@Data` - Generates getters, setters, toString, equals, and hashCode
- `@Builder` - Provides builder pattern for object creation
- `@NoArgsConstructor/@AllArgsConstructor` - Generates constructors

## Usage Example

```java
// Create authenticated user with builder pattern
AuthUser user = AuthUser.builder()
    .id("user123")
    .tenantId("tenant456")
    .email("john.doe@example.com")
    .firstName("John")
    .lastName("Doe")
    .emailVerified(true)
    .loginProvider("local")
    .roles(Arrays.asList(UserRole.ADMIN, UserRole.USER))
    .status(UserStatus.ACTIVE)
    .build();

// Access user information
String userEmail = user.getEmail();
boolean isVerified = user.getEmailVerified();
List<UserRole> userRoles = user.getRoles();
```
