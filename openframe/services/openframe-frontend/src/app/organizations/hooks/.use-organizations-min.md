A React hook that fetches and manages minimal organization data with search functionality, designed for lightweight use cases like dropdowns or lists.

## Key Components

- **`OrganizationMin` interface**: Defines the minimal organization data structure with essential fields
- **`useOrganizationsMin` hook**: Custom React hook that manages organization fetching state and operations
- **`fetch` function**: Memoized callback for executing GraphQL queries with search and pagination support

## Usage Example

```typescript
import { useOrganizationsMin } from './use-organizations-min'

function OrganizationSelector() {
  const { items, isLoading, error, fetch } = useOrganizationsMin(20)

  // Fetch organizations on component mount
  useEffect(() => {
    fetch()
  }, [fetch])

  // Search organizations
  const handleSearch = (searchTerm: string) => {
    fetch(searchTerm)
  }

  if (isLoading) return <div>Loading organizations...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <select>
      {items.map(org => (
        <option key={org.id} value={org.organizationId}>
          {org.name} {org.isDefault && '(Default)'}
        </option>
      ))}
    </select>
  )
}
```

The hook returns organization data with essential fields (id, name, default status, image URL) and handles loading states, error management, and GraphQL edge-node response mapping automatically.
