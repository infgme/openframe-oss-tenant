A utility class that provides string manipulation methods for common operations in the OpenFrame platform.

## Key Components

- **extractQueryParam(String url, String paramName)** - Extracts a specific query parameter value from a URL string

## Usage Example

```java
import com.openframe.util.StringUtils;

// Extract parameter from URL
String url = "https://api.example.com/users?id=123&name=john&active=true";
String userId = StringUtils.extractQueryParam(url, "id");
// Returns: "123"

String userName = StringUtils.extractQueryParam(url, "name"); 
// Returns: "john"

String status = StringUtils.extractQueryParam(url, "status");
// Returns: null (parameter not found)

// Handle URLs without query parameters
String simpleUrl = "https://api.example.com/users";
String result = StringUtils.extractQueryParam(simpleUrl, "id");
// Returns: null
```

The utility handles edge cases like missing query strings, malformed parameters, and URL-encoded values. Returns `null` when the parameter is not found or the URL has no query string.
