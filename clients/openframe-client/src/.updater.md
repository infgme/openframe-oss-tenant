A Rust module that provides automated application updates using the Velopack framework, supporting update checking, downloading, and application restart functionality.

## Key Components

- **`VelopackUpdater`** - Main updater struct that manages the update process with configurable update channels and URLs
- **`UpdateInfo`** - Data structure containing update metadata including version, download URL, and release notes
- **`new()`** - Constructor that initializes the updater with version info, update channel, and optional update URL
- **`check_for_updates()`** - Checks for available updates from the configured source
- **`download_and_apply_update()`** - Downloads and applies updates, then restarts the application
- **`restart_to_apply_update()`** - Handles application restart to complete the update process

## Usage Example

```rust
use semver::Version;
use anyhow::Result;

// Initialize the updater
let current_version = Version::parse("1.0.0")?;
let updater = VelopackUpdater::new(
    current_version,
    "stable".to_string(),
    Some("https://updates.myapp.com".to_string()),
)?;

// Check for updates
if let Some(update_info) = updater.check_for_updates()? {
    println!("Update available: {}", update_info.version);
    
    // Download and apply the update
    if updater.download_and_apply_update(&update_info)? {
        println!("Update applied, application will restart");
    }
}

// Alternative: restart to apply pending updates
updater.restart_to_apply_update()?;
```

The updater gracefully handles cases where no update URL is configured by creating a dummy manager that always returns no available updates.
