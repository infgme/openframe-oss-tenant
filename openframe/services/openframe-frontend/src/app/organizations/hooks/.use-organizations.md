A React hook that manages organization data fetching, search, pagination, and state management for the Flamingo platform. It provides comprehensive organization management capabilities with filtering, pagination, and error handling.

## Key Components

- **useOrganizations** - Main hook that manages organization state and data operations
- **OrganizationsFilterInput** - Type for filtering organizations by tier and industry
- **PageInfo** - Interface for pagination metadata
- **fetchOrganizations** - Core function to retrieve organization data from GraphQL API
- **Pagination functions** - Methods to handle first page, next page navigation
- **Search functionality** - Real-time search with state management

## Usage Example

```typescript
import { useOrganizations } from './hooks/use-organizations'

function OrganizationsView() {
  const filters = {
    tiers: ['Premium', 'Basic'],
    industries: ['Technology', 'Healthcare']
  }

  const {
    organizations,
    search,
    isLoading,
    error,
    pageInfo,
    fetchFirstPage,
    fetchNextPage,
    searchOrganizations,
    markInitialLoadDone
  } = useOrganizations(filters)

  useEffect(() => {
    fetchFirstPage('').then(() => {
      markInitialLoadDone()
    })
  }, [])

  const handleSearch = (term: string) => {
    searchOrganizations(term)
  }

  const loadMore = () => {
    if (pageInfo?.hasNextPage) {
      fetchNextPage(search)
    }
  }

  return (
    <div>
      <SearchInput onSearch={handleSearch} />
      {isLoading ? <Skeleton /> : <OrganizationsList data={organizations} />}
      {pageInfo?.hasNextPage && <LoadMoreButton onClick={loadMore} />}
    </div>
  )
}
```

The hook integrates with the organizations store for state persistence and provides comprehensive error handling with toast notifications. It supports cursor-based pagination and automatic refetching when filters change.
