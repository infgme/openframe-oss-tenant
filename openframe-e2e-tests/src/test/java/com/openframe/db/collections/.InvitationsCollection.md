MongoDB collection operations utility for managing user invitations within the OpenFrame platform.

## Key Components

- **findInvitation(String email)** - Retrieves the first invitation matching the specified email address
- **findInvitation(InvitationStatus status)** - Retrieves the first invitation with the specified status (e.g., PENDING, ACCEPTED, EXPIRED)

## Usage Example

```java
// Find invitation by email
Invitation userInvite = InvitationsCollection.findInvitation("user@example.com");

// Find invitation by status
Invitation pendingInvite = InvitationsCollection.findInvitation(InvitationStatus.PENDING);

// Check if invitation exists
if (userInvite != null) {
    System.out.println("Found invitation for: " + userInvite.getEmail());
}
```

The class provides static methods for querying the MongoDB "invitations" collection using MongoDB filters. It leverages the generic `getCollection()` method from the MongoDB utility class to return properly typed Invitation objects. Both methods return the first matching document or `null` if no match is found.
