This class provides API test methods for user-related operations in the OpenFrame platform, utilizing RestAssured for HTTP testing with both authorized and unauthorized request specifications.

## Key Components

- **me()** - Retrieves current user information, returns MeResponse object
- **deleteUser()** - Deletes a user by ID, returns HTTP status code
- **resetPassword()** - Initiates password reset process for a user email
- **confirmReset()** - Confirms password reset using reset token and new password
- **Static endpoint constants** - Predefined API paths for user operations

## Usage Example

```java
// Get current user information
MeResponse currentUser = UserApi.me();
System.out.println("User: " + currentUser.getUsername());

// Delete a specific user
int statusCode = UserApi.deleteUser("user123");
if (statusCode == 204) {
    System.out.println("User deleted successfully");
}

// Reset password flow
User user = new User();
user.setEmail("user@example.com");
UserApi.resetPassword(user);

// Confirm password reset
ResetConfirmRequest resetRequest = new ResetConfirmRequest();
resetRequest.setToken("reset-token");
resetRequest.setNewPassword("newPassword123");
UserApi.confirmReset(resetRequest);
```

The class leverages RequestSpecHelper for handling authentication requirements and follows a clean API design pattern for test automation in the OpenFrame ecosystem.
