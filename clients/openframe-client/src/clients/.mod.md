A module declaration file that exports HTTP client implementations for different service endpoints in the Flamingo platform, providing a unified interface for authentication, registration, tool management, and file operations.

## Key Components

- **AuthClient** - Handles authentication operations and token management
- **RegistrationClient** - Manages user and service registration workflows  
- **ToolAgentFileClient** - Provides file upload/download capabilities for tool agents
- **ToolApiClient** - Interfaces with the tool API for executing commands and retrieving data

## Usage Example

```rust
use crate::clients::{AuthClient, RegistrationClient, ToolAgentFileClient, ToolApiClient};

// Initialize clients for different services
let auth_client = AuthClient::new("https://api.flamingo.com");
let registration_client = RegistrationClient::new("https://api.flamingo.com");
let file_client = ToolAgentFileClient::new("https://api.flamingo.com");
let tool_client = ToolApiClient::new("https://api.flamingo.com");

// Use clients for respective operations
let token = auth_client.authenticate("user", "password").await?;
let registration = registration_client.register_agent(agent_config).await?;
let file_url = file_client.upload_file("config.json", file_data).await?;
let result = tool_client.execute_command("system_info").await?;
```

This module structure allows for clean separation of concerns while providing easy access to all HTTP client functionality through a single import point.
