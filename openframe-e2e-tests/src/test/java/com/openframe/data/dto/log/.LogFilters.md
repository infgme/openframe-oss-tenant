A data transfer object that defines filtering criteria for log queries, supporting multiple filter dimensions including tool types, event types, severities, and organizational scoping.

## Key Components

- **toolTypes** - List of tool type filters to apply to log searches
- **eventTypes** - List of event type filters for categorizing log entries
- **severities** - List of severity level filters (e.g., ERROR, WARN, INFO)
- **organizations** - List of organization filter options for multi-tenant log access

## Usage Example

```java
// Create comprehensive log filters
LogFilters filters = LogFilters.builder()
    .toolTypes(List.of("MONITORING", "BACKUP", "SECURITY"))
    .eventTypes(List.of("ALERT", "SYSTEM_EVENT", "USER_ACTION"))
    .severities(List.of("ERROR", "CRITICAL"))
    .organizations(List.of(
        OrganizationFilterOption.builder()
            .organizationId("org-123")
            .includeSubOrgs(true)
            .build()
    ))
    .build();

// Use in log service query
LogSearchRequest request = LogSearchRequest.builder()
    .filters(filters)
    .dateRange(dateRange)
    .build();
```

This DTO enables flexible log filtering across the OpenFrame platform, supporting both simple attribute-based filtering and complex organizational hierarchies for multi-tenant log management.
