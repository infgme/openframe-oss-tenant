A lightweight wrapper service that provides a simplified interface for publishing serialized messages to NATS subjects through the connection manager.

## Key Components

- **`NatsMessagePublisher`** - Main struct that encapsulates NATS publishing functionality
- **`new()`** - Constructor that accepts a `NatsConnectionManager` instance
- **`publish()`** - Generic async method that serializes and publishes payloads to specified NATS subjects

## Usage Example

```rust
use serde::Serialize;

#[derive(Serialize)]
struct UserEvent {
    user_id: String,
    action: String,
}

// Initialize the publisher
let connection_manager = NatsConnectionManager::new().await?;
let publisher = NatsMessagePublisher::new(connection_manager);

// Publish a message
let event = UserEvent {
    user_id: "user_123".to_string(),
    action: "login".to_string(),
};

publisher.publish("user.events", event).await?;
```

The publisher automatically handles JSON serialization of any `Serialize`-compatible type and provides clear error context for both serialization and publishing failures. It leverages the connection manager to obtain NATS clients, ensuring proper connection handling and reuse.
