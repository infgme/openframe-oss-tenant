A data transfer object (DTO) representing a tag filter option used in device filtering interfaces, containing the tag value, display label, and occurrence count.

## Key Components

- **value** - The actual tag value used for filtering operations
- **label** - Human-readable display name for the tag option
- **count** - Number of devices associated with this tag (optional)

## Usage Example

```java
// Creating a tag filter option for device categories
TagFilterOption serverTag = TagFilterOption.builder()
    .value("server")
    .label("Server Equipment")
    .count(25)
    .build();

// Using in a filter dropdown
List<TagFilterOption> filterOptions = Arrays.asList(
    TagFilterOption.builder()
        .value("workstation")
        .label("Workstations")
        .count(150)
        .build(),
    TagFilterOption.builder()
        .value("printer")
        .label("Printers")
        .count(12)
        .build()
);

// Accessing properties
String filterValue = serverTag.getValue(); // "server"
String displayText = serverTag.getLabel(); // "Server Equipment"
Integer deviceCount = serverTag.getCount(); // 25
```

This DTO is typically used in filtering UIs where users can select from available device tags, with the count indicating how many devices match each filter option.
