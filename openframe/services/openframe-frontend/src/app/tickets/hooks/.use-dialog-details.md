A React hook for fetching and managing dialog details from the Flamingo chat API using GraphQL queries.

## Key Components

- **useDialogDetails** - Main hook that manages dialog state, loading, and error handling
- **fetchDialog** - Callback function that performs the GraphQL query to retrieve dialog data
- **State management** - Handles `dialog`, `isLoading`, and `error` states with proper cleanup
- **GraphQL integration** - Uses typed responses with error handling for API communication

## Usage Example

```typescript
import { useDialogDetails } from './use-dialog-details'

function DialogComponent({ dialogId }: { dialogId: string }) {
  const { dialog, isLoading, error, refresh } = useDialogDetails(dialogId)

  if (isLoading) return <div>Loading dialog...</div>
  if (error) return <div>Error: {error}</div>
  if (!dialog) return <div>Dialog not found</div>

  return (
    <div>
      <h2>Dialog: {dialog.title}</h2>
      <button onClick={refresh}>Refresh</button>
      {/* Render dialog content */}
    </div>
  )
}
```

The hook automatically fetches dialog data when the `dialogId` changes and provides a `refresh` function for manual updates. It includes comprehensive error handling and loading states for a smooth user experience.
