This file serves as the main export barrel for the policies and queries tab system, providing a centralized interface for tab configuration, components, and content management.

## Key Components

- **POLICIES_AND_QUERIES_TABS** - Configuration object containing all available tab definitions
- **getPoliciesAndQueriesTab()** - Function to retrieve specific tab configuration by identifier
- **getTabComponent()** - Function to get the appropriate React component for a given tab
- **PoliciesAndQueriesTabContent** - Main content component that renders the selected tab's interface

## Usage Example

```typescript
import { 
  POLICIES_AND_QUERIES_TABS, 
  getPoliciesAndQueriesTab, 
  getTabComponent,
  PoliciesAndQueriesTabContent 
} from './policies-and-queries'

// Get all available tabs
const allTabs = POLICIES_AND_QUERIES_TABS

// Get a specific tab configuration
const policyTab = getPoliciesAndQueriesTab('policies')

// Get the component for a tab
const TabComponent = getTabComponent('queries')

// Render the main content
<PoliciesAndQueriesTabContent 
  activeTab="policies" 
  onTabChange={handleTabChange} 
/>
```

This barrel export pattern provides a clean API surface while keeping the internal file structure organized across `policies-and-queries-tabs` and `content` modules.
