A configuration class that manages API base URL settings for the OpenFrame platform, providing flexible URL configuration through system properties or environment variables.

## Key Components

- **DEFAULT_BASE_URL** - Default localhost API endpoint
- **GRAPHQL** - GraphQL endpoint path constant
- **getBaseUrl()** - Dynamic base URL resolver with fallback hierarchy

## Configuration Priority

The base URL is determined using the following priority order:

1. System property: `api.base.url`
2. Environment variable: `API_BASE_URL`
3. Default: `https://localhost/`

## Usage Example

```java
// Get the configured base URL
String apiUrl = ApiConfig.getBaseUrl();

// Build GraphQL endpoint
String graphqlEndpoint = ApiConfig.getBaseUrl() + ApiConfig.GRAPHQL;
// Results in: "https://localhost/api/graphql" (default)

// Configure via system property
System.setProperty("api.base.url", "https://api.flamingo.run");
String customUrl = ApiConfig.getBaseUrl();
// Results in: "https://api.flamingo.run/"

// Configure via environment variable
// export API_BASE_URL=https://staging.openframe.ai
// String envUrl = ApiConfig.getBaseUrl();
// Results in: "https://staging.openframe.ai/"
```

The class automatically ensures URLs end with a trailing slash and provides debug logging for troubleshooting configuration issues.
