Vue.js WebSocket utilities for Flamingo's real-time dashboard and CLI communication, providing singleton connections with automatic token authentication and JSON parsing.

## Key Components

- **`getWSUrl()`** - Constructs WebSocket URLs with authentication tokens
- **`useDashWSConnection()`** - Singleton WebSocket connection for dashboard real-time updates
- **`useCliWSConnection()`** - Singleton WebSocket connection for terminal/CLI interactions
- **`WSReturn`** - TypeScript interface defining the standard WebSocket message format

## Usage Example

```typescript
// Dashboard connection for real-time updates
const dashWS = useDashWSConnection();

// Listen for status changes
watch(dashWS.status, (status) => {
  console.log('Dashboard WS status:', status);
});

// Handle incoming data
watch(dashWS.data, (message) => {
  if (message.action === 'system_update') {
    console.log('System data:', message.data);
  }
});

// CLI connection for terminal operations
const cliWS = useCliWSConnection();

// Send command to terminal
cliWS.send(JSON.stringify({
  action: 'execute',
  data: { command: 'ls -la' }
}));

// Close connections when done
dashWS.close();
cliWS.close();
```

The module ensures singleton patterns to prevent multiple connections and automatically handles authentication via the auth store. Both connections parse incoming JSON data and provide reactive Vue refs for status monitoring.
