A React component that renders device status information as a styled badge with color-coded variants based on the device's current state.

## Key Components

- **DeviceStatusBadge**: Main component that displays device status with appropriate styling
- **getStatusColors**: Internal function that maps status variants to Tailwind CSS color classes
- **DeviceStatusBadgeProps**: TypeScript interface defining component props

## Usage Example

```typescript
import { DeviceStatusBadge } from './device-status-badge'

// Basic usage
<DeviceStatusBadge status="online" />

// With custom styling
<DeviceStatusBadge 
  status="offline" 
  className="ml-2 text-sm" 
/>

// Different status variants
<DeviceStatusBadge status="maintenance" />
<DeviceStatusBadge status="critical" />
<DeviceStatusBadge status="warning" />
```

The component automatically applies color-coded styling based on the status variant:
- **Success**: Green background for healthy/online states
- **Error/Critical**: Red background for offline/failed states  
- **Warning**: Yellow background for attention-needed states
- **Info**: Blue background for informational states

The badge uses uppercase text styling with rounded corners and semi-transparent backgrounds for visual consistency across the Flamingo platform.
