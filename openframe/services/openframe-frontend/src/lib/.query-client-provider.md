A React provider component that configures TanStack Query (React Query) for client-side data fetching with optimized default settings for the Flamingo platform.

## Key Components

- **QueryClientProvider** - Main provider component that wraps the application with TanStack Query functionality
- **QueryClient instance** - Configured with custom defaults for stale time and window focus behavior
- **Client-side only** - Uses 'use client' directive for Next.js App Router compatibility

## Usage Example

Wrap your application root or layout with the provider:

```typescript
import { QueryClientProvider } from './query-client-provider'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html>
      <body>
        <QueryClientProvider>
          {children}
        </QueryClientProvider>
      </body>
    </html>
  )
}
```

The provider configures queries with a 60-second stale time and disables refetching on window focus, optimizing performance for the MSP platform's data fetching patterns. Components can then use TanStack Query hooks like `useQuery` and `useMutation` throughout the application.
