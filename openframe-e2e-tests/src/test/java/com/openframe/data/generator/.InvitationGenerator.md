A utility class for generating test data and responses related to invitation functionality in the OpenFrame platform, using the Faker library to create realistic mock data for testing scenarios.

## Key Components

- **newUserInvitationRequest()** - Creates invitation requests with random email addresses for new users
- **existingUserInvitationRequest(AuthUser)** - Creates invitation requests using existing user's email
- **acceptInvitationRequest(Invitation)** - Generates acceptance requests with fake user details and correct password
- **alreadyAcceptedResponse()** - Creates conflict responses for already processed invitations
- **invitationRevokedResponse()** - Creates conflict responses for revoked invitations  
- **userAlreadyExistsResponse(AuthUser)** - Creates conflict responses when user already exists in tenant

## Usage Example

```java
// Generate invitation for new user
InvitationRequest newUserInvite = InvitationGenerator.newUserInvitationRequest();

// Generate invitation for existing user
AuthUser existingUser = // ... get user
InvitationRequest existingUserInvite = InvitationGenerator.existingUserInvitationRequest(existingUser);

// Generate acceptance request
Invitation invitation = // ... get invitation
AcceptInvitationRequest acceptRequest = InvitationGenerator.acceptInvitationRequest(invitation);

// Generate conflict responses
InvitationConflictResponse conflictResponse = InvitationGenerator.userAlreadyExistsResponse(existingUser);
InvitationConflictResponse revokedResponse = InvitationGenerator.invitationRevokedResponse();
```

The class centralizes invitation-related test data generation, ensuring consistent mock data across test suites while supporting various invitation workflow scenarios including conflicts and error conditions.
