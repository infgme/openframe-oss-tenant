A React component that provides tab navigation for application settings pages, organizing different configuration sections into a unified tabbed interface.

## Key Components

- **`SETTINGS_TABS`** - Static array defining all available settings tabs with their IDs, labels, icons, and associated components
- **`SettingsTabNavigation`** - Main navigation component that renders the tab interface
- **`getSettingsTabs()`** - Utility function to retrieve the settings tabs configuration
- **`BoldShieldCheckIcon`** - Custom icon wrapper with increased stroke width for the AI Settings tab

## Tab Configuration

The navigation includes six main settings sections:

| Tab | Purpose | Icon |
|-----|---------|------|
| AI Settings | AI configuration and preferences | ShieldCheck (bold) |
| Architecture | System architecture settings | Network |
| Company & Users | User and organization management | UsersGroup |
| API Keys | API key management | ShieldKey |
| SSO Configuration | Single sign-on setup | SSOConfiguration |
| Profile | User profile settings | User |

## Usage Example

```typescript
import { SettingsTabNavigation, SETTINGS_TABS } from './navigation'

function SettingsPage() {
  const [activeTab, setActiveTab] = useState('ai-settings')

  return (
    <div>
      <SettingsTabNavigation
        activeTab={activeTab}
        onTabChange={setActiveTab}
      />
      {/* Tab content renders automatically based on component property */}
    </div>
  )
}

// Access tab configuration
const tabs = getSettingsTabs()
const aiSettingsTab = tabs.find(tab => tab.id === 'ai-settings')
```
