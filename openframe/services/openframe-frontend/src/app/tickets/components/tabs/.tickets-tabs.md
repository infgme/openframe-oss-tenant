A React component that provides tabbed navigation for tickets/chats management with URL synchronization and tab-specific content rendering.

## Key Components

- **`TICKETS_TABS`** - Array defining available tabs (Current Chats, Archived Chats) with their icons and components
- **`getTicketsTab()`** - Utility function to find a tab by its ID
- **`getTabComponent()`** - Returns the React component associated with a specific tab
- **`TicketsTabNavigation`** - Main navigation component with URL sync and clean tab switching

## Usage Example

```typescript
import { TicketsTabNavigation, getTabComponent } from './tickets-tabs'

// Use the navigation component
function TicketsPage() {
  return (
    <div>
      <TicketsTabNavigation />
      {/* Tab content is handled by TabNavigation component internally */}
    </div>
  )
}

// Get specific tab component programmatically
const CurrentChatsComponent = getTabComponent('current')
if (CurrentChatsComponent) {
  return <CurrentChatsComponent />
}
```

The component implements URL-based tab state management, automatically clearing tab-specific parameters when switching between tabs to ensure a clean state. It integrates with Next.js routing and uses the OpenFrame core UI components for consistent styling across the Flamingo platform.
