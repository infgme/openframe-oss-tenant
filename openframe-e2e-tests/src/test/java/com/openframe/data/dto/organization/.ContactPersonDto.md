Data Transfer Object representing a contact person with basic contact information fields for organization-related operations.

## Key Components

- **ContactPersonDto**: Main DTO class with Lombok annotations for automatic getter/setter generation
- **contactName**: String field for the contact person's full name
- **title**: String field for the person's job title or position
- **phone**: String field for phone number
- **email**: String field for email address

## Usage Example

```java
// Creating a new contact person
ContactPersonDto contact = ContactPersonDto.builder()
    .contactName("John Smith")
    .title("IT Manager")
    .phone("555-0123")
    .email("john.smith@company.com")
    .build();

// Using constructor
ContactPersonDto contact = new ContactPersonDto(
    "Jane Doe", 
    "System Administrator", 
    "555-0124", 
    "jane.doe@company.com"
);

// Accessing fields (thanks to @Data annotation)
String name = contact.getContactName();
contact.setEmail("newemail@company.com");
```

The class uses Lombok annotations to automatically generate boilerplate code including constructors, getters, setters, toString, equals, and hashCode methods. The builder pattern is available for flexible object construction.
