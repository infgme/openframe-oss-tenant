A React component that manages SSO (Single Sign-On) provider configurations with support for both custom OAuth providers and shared OpenFrame SSO integration.

## Key Components

- **SsoConfigurationTab**: Main component rendering SSO provider management interface
- **UIProviderRow**: Type definition for provider table rows with status, configuration, and domain settings
- **Modal State Management**: Handles edit modal for individual provider configuration
- **Feature Flag Integration**: Conditional domain allowlist functionality based on feature flags
- **Shared SSO Integration**: Special handling for tenant domain-based Google/Microsoft SSO

## Usage Example

```typescript
import { SsoConfigurationTab } from './sso-configuration'

// Render SSO configuration management
function AdminSettings() {
  return (
    <div className="admin-panel">
      <SsoConfigurationTab />
    </div>
  )
}

// The component automatically:
// - Loads available OAuth providers (Google, Microsoft, etc.)
// - Displays provider status and configuration state
// - Handles shared SSO for tenant domains
// - Provides edit modal for provider configuration
// - Manages auto-provisioning settings
```

The component integrates with custom hooks (`useSsoConfig`, `useTenantDomain`) to manage provider configurations, supports search filtering, and includes a special card for shared OpenFrame SSO providers that auto-provisions users from verified tenant domains. The interface adapts based on feature flags and displays appropriate loading states and error handling throughout the user experience.
