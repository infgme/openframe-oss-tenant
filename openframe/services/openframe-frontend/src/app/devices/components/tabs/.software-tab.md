A React component that displays device software information in a tabular format with sortable columns and security vulnerability indicators.

## Key Components

- **SoftwareTab**: Main component that renders a table of installed software for a device
- **formatDate**: Utility function to format timestamps for display
- **columns**: Table configuration with custom renderers for software info, source badges, security status, and last opened dates
- **SoftwareInfo**: Displays software name, vendor, and version information
- **SoftwareSourceBadge**: Shows the source of software installation
- **StatusTag**: Renders security vulnerability status with appropriate styling

## Usage Example

```typescript
import { SoftwareTab } from './software-tab'
import { Device } from '../../types/device.types'

function DeviceDetails({ device }: { device: Device }) {
  return (
    <div className="device-details">
      <h2>Device Software</h2>
      <SoftwareTab device={device} />
    </div>
  )
}

// The component handles null devices gracefully
<SoftwareTab device={null} />

// Displays software with vulnerability counts and last opened dates
<SoftwareTab device={deviceWithSoftware} />
```

The component features responsive design, handles empty states, and provides sorting capabilities across all columns. Security vulnerabilities are highlighted with color-coded status tags, and dates are formatted consistently with other device information sections.
