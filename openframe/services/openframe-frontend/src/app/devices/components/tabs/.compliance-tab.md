A React component that displays comprehensive device compliance information including patch management, Windows update policies, and security checks from integrated tools like Tactical RMM and Fleet MDM.

## Key Components

- **ComplianceTab** - Main component that renders compliance data in organized sections
- **InfoCard** - Reusable card component for displaying structured data with tooltips
- **TooltipProvider/Tooltip** - Provides contextual help information for each section
- **formatScheduleDays()** - Helper function to convert numeric day arrays to readable format

## Usage Example

```typescript
import { ComplianceTab } from './compliance-tab'

// Device object with compliance data from RMM/MDM systems
const deviceData = {
  patches_last_installed: '2024-01-15T10:30:00Z',
  has_patches_pending: false,
  winupdatepolicy: [{
    critical: 'approve',
    important: 'approve',
    run_time_days: [1, 3, 5],
    run_time_hour: 2
  }],
  checks: {
    total: 25,
    passing: 23,
    failing: 1,
    warning: 1
  },
  applied_policies: {
    agent_policy: 'Standard Workstation',
    site_policy: 'Office Security'
  },
  issues: {
    failing_policies_count: 2,
    total_issues_count: 5
  }
}

function DeviceDetailsPage() {
  return (
    <div>
      <ComplianceTab device={deviceData} />
    </div>
  )
}
```

The component organizes compliance data into sections: Patch Management (OS updates), Windows Update Policies (severity and scheduling), Policy Compliance (inheritance and applied policies), Compliance Checks (automated monitoring), Fleet Policies (MDM status), and Fleet Labels (device categorization). Each section includes helpful tooltips explaining the data sources and meanings.
