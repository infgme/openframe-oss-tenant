A React hook that fetches and manages the current release version from the API, providing loading states and error handling with toast notifications.

## Key Components

- **`useReleaseVersion`** - Main hook that manages release version state and API calls
- **`ReleaseVersionResponse`** - Type definition for the API response structure
- **`DEFAULT_RELEASE_VERSION`** - Fallback version string ('latest') used when API fails
- **State management** - Tracks `releaseVersion`, `isLoading`, and `error` states
- **`fetchReleaseVersion`** - Memoized callback function that handles the API request
- **Error handling** - Displays toast notifications on failures and falls back to default version

## Usage Example

```typescript
import { useReleaseVersion } from './use-release-version'

function AppHeader() {
  const { releaseVersion, isLoading, error, refetch } = useReleaseVersion()

  return (
    <div className="header">
      <h1>OpenFrame</h1>
      {isLoading ? (
        <span>Loading version...</span>
      ) : error ? (
        <button onClick={refetch}>Retry loading version</button>
      ) : (
        <span className="version">v{releaseVersion}</span>
      )}
    </div>
  )
}
```

The hook automatically fetches the release version on mount and provides a `refetch` function for manual retries. It gracefully handles errors by showing toast notifications while maintaining a working fallback state.
