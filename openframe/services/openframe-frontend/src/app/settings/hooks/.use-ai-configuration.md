A React hook for managing AI provider configuration in the Flamingo platform, handling CRUD operations for AI models from OpenAI, Anthropic, and Google Gemini providers.

## Key Components

- **`useAIConfiguration`** - Main hook that provides AI configuration management functionality
- **`AIConfiguration`** - Interface defining the structure of AI provider settings
- **`ModelInfo`** - Interface describing available AI model specifications  
- **`SupportedModels`** - Interface mapping providers to their available models
- **`UpdateAIConfigurationParams`** - Interface for configuration update parameters

## Usage Example

```typescript
import { useAIConfiguration } from './use-ai-configuration'

function AISettingsPanel() {
  const {
    configuration,
    supportedModels,
    isLoading,
    isSaving,
    updateConfiguration
  } = useAIConfiguration()

  const handleSaveSettings = async () => {
    try {
      await updateConfiguration({
        provider: 'OPENAI',
        modelName: 'gpt-4',
        apiKey: 'sk-...'
      })
    } catch (error) {
      // Error handling is done automatically via toast notifications
      console.error('Failed to save configuration:', error)
    }
  }

  if (isLoading) return <div>Loading AI configuration...</div>

  return (
    <div>
      <h2>AI Configuration</h2>
      <p>Current Provider: {configuration?.provider}</p>
      <p>Model: {configuration?.modelName}</p>
      <button onClick={handleSaveSettings} disabled={isSaving}>
        {isSaving ? 'Saving...' : 'Save Configuration'}
      </button>
    </div>
  )
}
```

The hook automatically fetches configuration and supported models on mount, provides loading states, and handles error notifications through toast messages. It supports all major AI providers and manages API key validation status.
