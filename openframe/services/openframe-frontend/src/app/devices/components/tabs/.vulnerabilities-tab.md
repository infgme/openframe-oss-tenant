A React component that displays security vulnerabilities found in software installed on devices, with sortable table view and severity-based categorization.

## Key Components

- **VulnerabilitiesTab**: Main component that renders a table of vulnerabilities with contextual software information
- **VulnerabilityWithSoftware**: Extended interface combining vulnerability data with software metadata
- **vulnerabilities**: Memoized array that flattens all vulnerabilities from device software with unique identifiers
- **columns**: Table column definitions with custom renderers for CVE links, software info, severity badges, and dates
- **severityCounts**: Computed vulnerability counts grouped by severity level (critical, high, medium, low)
- **formatDate**: Utility function for consistent date formatting
- **getSeverity**: Heuristic function that determines vulnerability severity based on CVE year

## Usage Example

```typescript
import { VulnerabilitiesTab } from './vulnerabilities-tab'
import { Device } from '../../types/device.types'

function DeviceDetailsPage({ device }: { device: Device }) {
  return (
    <div className="space-y-6">
      <h2>Device Security Overview</h2>
      <VulnerabilitiesTab device={device} />
    </div>
  )
}

// Example with no vulnerabilities
<VulnerabilitiesTab device={deviceWithNoVulns} />
// Displays: "No Vulnerabilities Found" with success badge

// Example with vulnerabilities
<VulnerabilitiesTab device={deviceWithVulns} />
// Displays: Sortable table with CVE links, software info, severity tags
```

The component automatically handles empty states, flattens nested vulnerability data from multiple software packages, and provides interactive sorting with severity-based visual indicators. Each vulnerability row includes clickable CVE links and detailed software context.
