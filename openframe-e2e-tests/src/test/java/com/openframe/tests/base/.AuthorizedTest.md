Base test class that provides authentication and database setup for OpenFrame integration tests, automatically configuring REST Assured and MongoDB connections.

## Key Components

- **AuthorizedTest**: Abstract base class for authorized test scenarios
- **@BeforeAll config()**: Sets up REST Assured configuration and opens MongoDB connection
- **@AfterAll close()**: Cleanly closes MongoDB connection after test execution

## Usage Example

```java
public class UserServiceTest extends AuthorizedTest {
    
    @Test
    public void testCreateUser() {
        // Test runs with pre-configured authentication
        // and active MongoDB connection
        given()
            .contentType("application/json")
            .body(createUserRequest)
        .when()
            .post("/api/users")
        .then()
            .statusCode(201);
    }
    
    @Test
    public void testGetUserProfile() {
        // MongoDB connection is already established
        // REST Assured is pre-configured with auth
        given()
            .pathParam("userId", "123")
        .when()
            .get("/api/users/{userId}")
        .then()
            .statusCode(200)
            .body("name", notNullValue());
    }
}
```

This class eliminates boilerplate setup code by providing common test infrastructure. All test classes extending `AuthorizedTest` automatically get authenticated API access and database connectivity without manual configuration.
