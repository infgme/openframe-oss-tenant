A GraphQL edge wrapper for device/machine pagination that follows the Relay connection specification, containing a machine node and its cursor position.

## Key Components

- **`node`** - The `Machine` object representing the actual device data
- **`cursor`** - String identifier for pagination positioning within the connection
- **Lombok annotations** - Auto-generates boilerplate code (getters, setters, constructors, builder)

## Usage Example

```java
// Create a device edge for pagination
DeviceEdge edge = DeviceEdge.builder()
    .node(machine)
    .cursor("Y3Vyc29yOjE=")
    .build();

// Used in GraphQL connection responses
DeviceConnection connection = DeviceConnection.builder()
    .edges(List.of(edge))
    .pageInfo(pageInfo)
    .build();

// Access the machine data
Machine device = edge.getNode();
String position = edge.getCursor();
```

This DTO is typically used in GraphQL APIs implementing cursor-based pagination for device listings, where each edge represents one device in the paginated result set along with its cursor for subsequent navigation.
