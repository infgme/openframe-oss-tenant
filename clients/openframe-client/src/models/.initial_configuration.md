A serializable configuration struct that defines the initial setup parameters for connecting to the Flamingo/OpenFrame platform, including server connection details and authentication credentials.

## Key Components

- **`InitialConfiguration`** - Main configuration struct with serialization support
- **Fields:**
  - `server_host` - Target server hostname or URL
  - `initial_key` - Authentication key for initial setup
  - `local_mode` - Boolean flag for local development mode
  - `org_id` - Organization identifier (optional, defaults to empty)
  - `local_ca_cert_path` - Path to local CA certificate file (optional)

## Usage Example

```rust
use serde_json;

// Create default configuration
let config = InitialConfiguration::default();

// Create configuration for production setup
let prod_config = InitialConfiguration {
    server_host: "api.flamingo.com".to_string(),
    initial_key: "sk-1234567890abcdef".to_string(),
    local_mode: false,
    org_id: "org_flamingo_123".to_string(),
    local_ca_cert_path: "/etc/ssl/certs/flamingo-ca.pem".to_string(),
};

// Serialize to JSON for storage
let json = serde_json::to_string(&prod_config)?;

// Deserialize from JSON
let loaded_config: InitialConfiguration = serde_json::from_str(&json)?;
```

The struct uses `#[serde(default)]` attributes on optional fields, allowing partial configuration loading where missing fields will use their default values.
