A standardized error response DTO that encapsulates API error information with error codes and descriptive messages.

## Key Components

- **code** - String field containing the error type identifier (e.g., "BAD_REQUEST", "VALIDATION_ERROR")
- **message** - String field containing the human-readable error description
- **@Data, @Builder, @NoArgsConstructor, @AllArgsConstructor** - Lombok annotations providing standard object methods and builder pattern support

## Usage Example

```java
// Creating an error response using builder pattern
ErrorResponse error = ErrorResponse.builder()
    .code("BAD_REQUEST")
    .message("Registration is closed for this organization")
    .build();

// Creating an error response using constructor
ErrorResponse validationError = new ErrorResponse(
    "VALIDATION_ERROR", 
    "Invalid email format provided"
);

// Accessing error details
String errorCode = error.getCode();
String errorMessage = error.getMessage();

// Typical usage in exception handling
@ExceptionHandler(ValidationException.class)
public ResponseEntity<ErrorResponse> handleValidation(ValidationException ex) {
    ErrorResponse response = ErrorResponse.builder()
        .code("VALIDATION_ERROR")
        .message(ex.getMessage())
        .build();
    return ResponseEntity.badRequest().body(response);
}
```
