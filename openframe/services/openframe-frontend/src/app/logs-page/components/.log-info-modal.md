Modal component that displays detailed information about a log entry, including device details, raw data, and metadata in a slide-out panel from the right side of the screen.

## Key Components

- **LogInfoModal** - Main modal component with slide-in animation from right
- **InfoField** - Utility component for displaying label-value pairs
- **LogInfoModalProps** - Interface defining modal props including log data and fetch function
- **DetailedLogData** - Interface for extended log information fetched from API

## Key Features

- **Device Integration**: Automatically fetches and displays full device details when deviceId is available
- **Progressive Loading**: Shows basic log data immediately, then fetches detailed information asynchronously
- **Raw Data Display**: Formatted JSON view of log details and metadata
- **Keyboard Navigation**: ESC key support and click-outside-to-close functionality
- **Status Visualization**: StatusTag components for log severity and device status
- **Tool Integration**: ToolBadge component showing log source with appropriate styling

## Usage Example

```typescript
import { LogInfoModal } from './log-info-modal'

function LogsPage() {
  const [selectedLog, setSelectedLog] = useState(null)
  const [isModalOpen, setIsModalOpen] = useState(false)

  const handleFetchLogDetails = async (logEntry: any) => {
    // Fetch detailed log information from API
    const response = await api.getLogDetails({
      toolEventId: logEntry.toolEventId,
      ingestDay: logEntry.ingestDay,
      toolType: logEntry.toolType,
      eventType: logEntry.eventType
    })
    return response.data
  }

  return (
    <>
      <LogInfoModal
        isOpen={isModalOpen}
        onClose={() => setIsModalOpen(false)}
        log={selectedLog}
        fetchLogDetails={handleFetchLogDetails}
      />
    </>
  )
}
```

The modal requires a log object with basic information and an `originalLogEntry` field containing the full log data needed for API calls. It automatically handles device lookups, loading states, and provides a comprehensive view of log information with proper error handling.
