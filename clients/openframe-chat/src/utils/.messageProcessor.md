Processes GraphQL historical messages into a standardized chat display format for the OpenFrame frontend, specifically configured for Fae (client-facing AI assistant) interactions.

## Key Components

- **`processHistoricalMessages`** - Main function that transforms GraphQL messages into chat UI format
- **Message transformation pipeline** - Converts GraphQL message structure to frontend Message interface
- **Fae assistant configuration** - Pre-configured with Fae avatar and client chat type filtering
- **Error handling integration** - Uses core library's error processing capabilities

## Usage Example

```typescript
import { processHistoricalMessages } from './messageProcessor'
import type { Message as GraphQLMessage } from '../services/dialogGraphQLService'

// Transform GraphQL messages for chat display
const graphqlMessages: GraphQLMessage[] = await fetchDialogHistory(dialogId)

const chatMessages = processHistoricalMessages(
  graphqlMessages,
  async (requestId) => {
    // Handle approval action
    await approveRequest(requestId)
  },
  async (requestId) => {
    // Handle rejection action
    await rejectRequest(requestId)
  }
)

// Use in chat component
<ChatInterface messages={chatMessages} />
```

The processor specifically filters for client-type messages and ensures all assistant messages display with the Fae avatar, providing consistent branding for the client-facing AI experience.
