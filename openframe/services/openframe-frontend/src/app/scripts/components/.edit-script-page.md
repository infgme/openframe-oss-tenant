A React component for creating and editing script configurations in the OpenFrame MSP platform, providing a comprehensive form interface for script management with platform support, shell configuration, and code editing capabilities.

## Key Components

- **EditScriptPage** - Main component handling script creation/editing with form validation and API integration
- **ScriptData interface** - Type definition for script configuration including metadata, execution parameters, and code
- **Platform Selection** - Multi-select interface for choosing supported operating systems (Windows, macOS, Linux)
- **Code Editor** - Textarea with line numbers and syntax support for script body editing
- **Dynamic Form Fields** - Handlers for managing script arguments and environment variables

## Usage Example

```typescript
import { EditScriptPage } from './edit-script-page'

// Create new script
function CreateScript() {
  return <EditScriptPage scriptId={null} />
}

// Edit existing script
function EditScript() {
  return <EditScriptPage scriptId="script-123" />
}

// Script data structure
const scriptData: ScriptData = {
  name: 'System Cleanup',
  shell: 'powershell',
  default_timeout: 90,
  args: [{ name: 'verbose', value: 'true' }],
  script_body: 'Get-Process | Where-Object...',
  run_as_user: false,
  env_vars: [{ name: 'LOG_LEVEL', value: 'INFO' }],
  description: 'Automated system cleanup script',
  supported_platforms: ['windows', 'linux'],
  category: 'System Maintenance'
}
```

The component integrates with the Tactical RMM API for script persistence and provides real-time validation, platform-specific shell type selection, and a rich editing experience with features like argument management and environment variable configuration.
