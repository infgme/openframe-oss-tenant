A React component that displays script metadata and configuration details in a structured card layout within the OpenFrame platform.

## Key Components

- **ScriptInfoSection** - Main component that renders script information including description, shell type, supported platforms, category, and timeout settings
- **ShellTypeBadge** - Badge component for displaying the script's shell type (imported from core)
- **OSTypeBadgeGroup** - Component group for showing supported operating systems (imported from core)

## Usage Example

```typescript
import { ScriptInfoSection } from './script-info-section'

// Example script object
const scriptData = {
  description: "System health check script",
  shell: "bash",
  supported_platforms: ["linux", "macos"],
  category: "monitoring",
  default_timeout: 300
}

// Render the component
<ScriptInfoSection script={scriptData} />

// Handle null/empty state
<ScriptInfoSection script={null} />
```

The component automatically handles empty states by displaying a "No script data available" message when no script is provided. It uses OpenFrame's design system (ODS) classes for consistent styling and responsive grid layout that adapts from single column on mobile to four columns on desktop.
