A React hook that manages the fetching and state of detailed log entry information using GraphQL queries, providing error handling and loading states for log detail operations.

## Key Components

- **useLogDetails**: Main hook that provides log details fetching functionality
- **fetchLogDetailsByID**: Async function to retrieve log details by ID with multiple identifying parameters
- **clearLogDetails**: Function to reset log details state
- **State management**: Tracks `logDetails`, `isLoading`, and `error` states
- **Error handling**: Comprehensive error handling with toast notifications for user feedback

## Usage Example

```typescript
import { useLogDetails } from './hooks/use-log-details'

function LogDetailsComponent() {
  const { 
    logDetails, 
    isLoading, 
    error, 
    fetchLogDetailsByID, 
    clearLogDetails 
  } = useLogDetails()

  const handleFetchDetails = async () => {
    try {
      await fetchLogDetailsByID(
        'log-123',
        '2024-01-15', 
        'application',
        'error',
        '2024-01-15T10:30:00Z'
      )
    } catch (error) {
      // Error already handled by hook
      console.log('Fetch failed')
    }
  }

  if (isLoading) return <div>Loading log details...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      {logDetails ? (
        <pre>{JSON.stringify(logDetails, null, 2)}</pre>
      ) : (
        <button onClick={handleFetchDetails}>
          Fetch Log Details
        </button>
      )}
      <button onClick={clearLogDetails}>Clear</button>
    </div>
  )
}
```
