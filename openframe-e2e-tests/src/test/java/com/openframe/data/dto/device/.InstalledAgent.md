A data transfer object (DTO) representing an installed agent on a device within the OpenFrame MSP platform, containing agent identification, versioning, and timestamp information.

## Key Components

- **InstalledAgent**: Main DTO class with Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern
- **id**: Unique identifier for the installed agent instance
- **machineId**: Reference to the device/machine where the agent is installed
- **agentType**: Classification of the agent (e.g., monitoring, security, backup)
- **version**: Current version of the installed agent
- **createdAt/updatedAt**: Timestamp fields for tracking agent installation and updates

## Usage Example

```java
// Create a new installed agent record
InstalledAgent agent = InstalledAgent.builder()
    .id("agent-12345")
    .machineId("machine-67890")
    .agentType("monitoring")
    .version("2.1.0")
    .createdAt("2024-01-15T10:30:00Z")
    .updatedAt("2024-01-15T10:30:00Z")
    .build();

// Update agent version
agent.setVersion("2.1.1");
agent.setUpdatedAt("2024-01-20T14:45:00Z");

// Access agent information
String agentInfo = String.format("Agent %s (v%s) on machine %s", 
    agent.getId(), agent.getVersion(), agent.getMachineId());
```

This DTO is typically used for API responses, database mapping, and internal data transfer when managing agent deployments across client devices in the OpenFrame platform.
