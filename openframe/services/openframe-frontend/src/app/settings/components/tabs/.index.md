This is the main export file for the settings module, providing a centralized interface to all settings-related components and utilities.

## Key Components

- **SettingsTabNavigation** - Navigation component for settings tabs
- **getSettingsTabs** - Function to retrieve available settings tabs
- **SETTINGS_TABS** - Constants defining settings tab configuration
- **ArchitectureTab** - Component for system architecture settings
- **CompanyAndUsersTab** - Component for managing company and user settings
- **ApiKeysTab** - Component for API key management
- **SsoConfigurationTab** - Component for SSO configuration
- **AISettingsTab** - Component for AI-related settings
- **ProfileTab** - Component for user profile settings

## Usage Example

```typescript
import { 
  SettingsTabNavigation,
  getSettingsTabs,
  ArchitectureTab,
  ApiKeysTab 
} from './settings'

// Get available tabs
const availableTabs = getSettingsTabs()

// Use in a settings page component
function SettingsPage() {
  return (
    <div>
      <SettingsTabNavigation tabs={availableTabs} />
      <ArchitectureTab />
      <ApiKeysTab />
    </div>
  )
}
```

This barrel export pattern allows consumers to import any settings component from a single location, improving developer experience and maintaining clean import statements throughout the application.
