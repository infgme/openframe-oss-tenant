A React component that displays user account information for devices in the Flamingo OpenFrame system, showing the currently logged-in user and all system users with their associated metadata.

## Key Components

- **UsersTab**: Main component that renders user information in cards
- **InfoCard**: Used to display user details in a structured format  
- **Tooltip**: Provides contextual help for user data sources
- **Device interface**: Type definition for device data structure
- **User data rendering**: Displays UID, group, shell, and account type information

## Usage Example

```typescript
import { UsersTab } from './users-tab'
import { Device } from '../../types/device.types'

// Example device data with users
const deviceData: Device = {
  users: [
    {
      username: 'john.doe',
      uid: 1001,
      groupname: 'users',
      shell: '/bin/bash',
      type: 'person',
      isLoggedIn: true
    },
    {
      username: 'postgres',
      uid: 999,
      groupname: 'postgres',
      shell: '/bin/bash',
      type: 'service',
      isLoggedIn: false
    }
  ]
}

// Render the users tab
<UsersTab device={deviceData} />
```

The component automatically handles cases where no device data is available and intelligently determines the logged-in user from Fleet MDM and Tactical RMM data. It displays user information in a responsive grid layout with tooltips explaining the data sources and user account types.
