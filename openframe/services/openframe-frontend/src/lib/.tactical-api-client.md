A specialized API client that extends the base API functionality to interact with Tactical RMM endpoints, providing methods for agent management, script execution, and system monitoring.

## Key Components

- **TacticalApiClient**: Main class extending base API client with Tactical RMM-specific methods
- **HTTP Methods**: Standard REST operations (GET, POST, PUT, PATCH, DELETE) with automatic JSON serialization
- **Agent Management**: Methods for retrieving agents, running scripts, and bulk operations
- **System Monitoring**: Endpoints for logs, checks, services, processes, and system information
- **Script Management**: CRUD operations for Tactical RMM scripts
- **URL Building**: Automatic construction of Tactical-specific URLs from tenant configuration

## Usage Example

```typescript
import { tacticalApiClient } from './tactical-api-client'

// Get all agents
const agents = await tacticalApiClient.getAgents()

// Get specific agent details
const agent = await tacticalApiClient.getAgent('agent-123')

// Run a script on an agent
const scriptResult = await tacticalApiClient.runScript('agent-123', {
  output: 'console',
  emails: ['admin@company.com'],
  emailMode: 'onError',
  custom_field: null,
  save_all_output: true,
  script: 456,
  args: ['--verbose'],
  env_vars: [],
  timeout: 300,
  run_as_user: false,
  run_on_server: false
})

// Create a new script
const newScript = await tacticalApiClient.createScript({
  name: 'System Check',
  shell: 'powershell',
  default_timeout: 120,
  args: [],
  script_body: 'Get-ComputerInfo',
  run_as_user: false,
  env_vars: [],
  description: 'Basic system information check',
  supported_platforms: ['windows'],
  category: 'monitoring'
})

// Get agent logs with pagination
const logs = await tacticalApiClient.getAgentLogs('agent-123', {
  limit: 50,
  offset: 0,
  search: 'error'
})
```
