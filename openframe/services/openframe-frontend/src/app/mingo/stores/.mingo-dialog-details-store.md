A Zustand store that manages the detailed state of individual Mingo AI chat dialogs, including message history, typing indicators, and real-time updates for the admin interface.

## Key Components

### State Properties
- **Dialog State**: `currentDialogId`, `currentDialog`, `adminMessages`
- **Loading States**: `isLoadingDialog`, `isLoadingMessages`
- **Error States**: `dialogError`, `messagesError`
- **Pagination**: `hasMoreMessages`, `messagesCursor`, `newestMessageCursor`
- **Typing Indicators**: `dialogTypingStates` per dialog

### Core Actions
- **Dialog Management**: `setCurrentDialog()`, `clearCurrent()`
- **Message Operations**: `setAdminMessages()`, `addRealtimeMessage()`, `removeWelcomeMessages()`
- **Typing Management**: `ensureTypingMessage()`, `updateLastAssistantMessage()`, `setDialogTyping()`
- **State Management**: Loading and error setters, pagination controls

## Usage Example

```typescript
import { useMingoDialogDetailsStore } from './mingo-dialog-details-store'

function ChatDialog({ dialogId }: { dialogId: string }) {
  const {
    currentDialog,
    adminMessages,
    isLoadingMessages,
    setCurrentDialogId,
    addRealtimeMessage,
    ensureTypingMessage,
    updateLastAssistantMessage
  } = useMingoDialogDetailsStore()

  // Set active dialog
  useEffect(() => {
    setCurrentDialogId(dialogId)
  }, [dialogId])

  // Handle real-time message streaming
  const handleStreamingMessage = (partialMessage: string) => {
    ensureTypingMessage(dialogId)
    updateLastAssistantMessage(dialogId, partialMessage)
  }

  return (
    <div>
      {isLoadingMessages ? (
        <div>Loading messages...</div>
      ) : (
        adminMessages.map(message => (
          <MessageComponent key={message.id} message={message} />
        ))
      )}
    </div>
  )
}
```
