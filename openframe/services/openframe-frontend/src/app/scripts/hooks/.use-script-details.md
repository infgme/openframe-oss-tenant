A React hook for fetching and managing script details from the Tactical RMM API with comprehensive error handling and loading states.

## Key Components

- **ScriptDetails Interface**: TypeScript interface defining the complete structure of script data including execution parameters, metadata, and platform support
- **useScriptDetails Hook**: Custom React hook that manages script fetching state and provides automatic refetch capabilities
- **State Management**: Tracks script data, loading status, and error states with proper cleanup
- **API Integration**: Leverages the tactical API client for standardized script retrieval

## Usage Example

```typescript
import { useScriptDetails } from './hooks/use-script-details'

function ScriptDetailsPage({ scriptId }: { scriptId: string }) {
  const { scriptDetails, isLoading, error, refetch } = useScriptDetails(scriptId)

  if (isLoading) return <div>Loading script details...</div>
  if (error) return <div>Error: {error}</div>
  if (!scriptDetails) return <div>Script not found</div>

  return (
    <div>
      <h1>{scriptDetails.name}</h1>
      <p>{scriptDetails.description}</p>
      <p>Shell: {scriptDetails.shell}</p>
      <p>Platforms: {scriptDetails.supported_platforms.join(', ')}</p>
      <button onClick={refetch}>Refresh</button>
    </div>
  )
}
```

The hook automatically fetches script details when the scriptId changes and provides a refetch function for manual updates. It handles loading states, error conditions, and validates that a script ID is provided before making API calls.
