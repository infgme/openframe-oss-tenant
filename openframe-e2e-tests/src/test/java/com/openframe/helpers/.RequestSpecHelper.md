A utility class that creates pre-configured RequestAssured request specifications for API testing, providing both authorized and unauthorized request configurations with Allure reporting integration.

## Key Components

- **`getAuthorizedSpec()`** - Creates a RequestSpecification with authentication cookies, JSON content type, and Allure reporting filter
- **`getUnAuthorizedSpec()`** - Creates a RequestSpecification without authentication for testing unauthorized access scenarios
- **AllureRestAssured filter** - Automatically captures request/response details for test reporting
- **AuthHelper integration** - Leverages existing authentication helper to retrieve session cookies

## Usage Example

```java
import static com.openframe.helpers.RequestSpecHelper.*;
import static io.restassured.RestAssured.given;

// Test authorized endpoint
given()
    .spec(getAuthorizedSpec())
    .when()
    .get("/api/users")
    .then()
    .statusCode(200);

// Test unauthorized access
given()
    .spec(getUnAuthorizedSpec())
    .when()
    .get("/api/protected-resource")
    .then()
    .statusCode(401);
```

This helper standardizes API test setup across the OpenFrame test suite, ensuring consistent authentication handling and comprehensive test reporting through Allure integration.
