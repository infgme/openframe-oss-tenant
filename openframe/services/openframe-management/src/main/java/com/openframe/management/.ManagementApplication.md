Main entry point for the OpenFrame Management application, a Spring Boot service that provides management capabilities for the OpenFrame MSP platform while excluding Cassandra health monitoring.

## Key Components

- **ManagementApplication** - Spring Boot application class with main method
- **@SpringBootApplication** - Enables auto-configuration, component scanning, and configuration
- **@ComponentScan** - Scans multiple OpenFrame packages while filtering out Cassandra components
- **Component exclusion** - Explicitly excludes `CassandraHealthIndicator` from the application context

## Usage Example

```java
// Start the management application
public class ManagementApplication {
    public static void main(String[] args) {
        // Launches Spring Boot application with management services
        SpringApplication.run(ManagementApplication.class, args);
    }
}

// The application scans these packages:
// - com.openframe.management (management services)
// - com.openframe.data (data layer components) 
// - com.openframe.core (core utilities)

// But excludes:
// - CassandraHealthIndicator (Cassandra health monitoring)
```

The application uses Spring's component scanning to automatically discover and register beans from the OpenFrame management, data, and core packages, making it easy to build a comprehensive management service while maintaining selective component inclusion for deployment flexibility.
