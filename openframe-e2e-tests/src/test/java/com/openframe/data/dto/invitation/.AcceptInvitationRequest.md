Data Transfer Object for handling invitation acceptance requests in the OpenFrame platform, containing user registration information and invitation reference.

## Key Components

- **invitationId** - Unique identifier for the invitation being accepted
- **firstName** - User's first name for account creation
- **lastName** - User's last name for account creation  
- **password** - New password for the user account
- **switchTenant** - Boolean flag to control tenant switching behavior (defaults to false)

## Usage Example

```java
// Accept an invitation with user registration details
AcceptInvitationRequest request = AcceptInvitationRequest.builder()
    .invitationId("inv-12345")
    .firstName("John")
    .lastName("Doe")
    .password("securePassword123")
    .switchTenant(true)
    .build();

// Alternative construction
AcceptInvitationRequest request = new AcceptInvitationRequest(
    "inv-12345", 
    "John", 
    "Doe", 
    "securePassword123", 
    false
);
```

The class uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it ideal for REST API request handling and data validation in invitation acceptance workflows.
