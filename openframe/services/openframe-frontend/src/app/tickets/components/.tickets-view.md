A client-side React component that renders the main tickets view interface with tabbed navigation, reading the active tab state from URL search parameters.

## Key Components

- **TicketsView** - Main component that orchestrates the tickets interface layout
- **TicketsTabNavigation** - Navigation component for switching between ticket tabs
- **TicketsTabContent** - Content area that displays ticket data based on active tab
- **useSearchParams** - Next.js hook for reading URL query parameters

## Usage Example

```typescript
import { TicketsView } from './tickets-view'

// In a page or parent component
export default function TicketsPage() {
  return (
    <div className="container mx-auto p-4">
      <h1>Support Tickets</h1>
      <TicketsView />
    </div>
  )
}

// URL examples that affect the view:
// /tickets -> Shows 'current' tab (default)
// /tickets?tab=current -> Shows current tickets
// /tickets?tab=closed -> Shows closed tickets
// /tickets?tab=pending -> Shows pending tickets
```

The component uses URL-based state management, allowing users to bookmark specific ticket views and enabling proper browser navigation. The layout uses flexbox for responsive design across different screen sizes.
