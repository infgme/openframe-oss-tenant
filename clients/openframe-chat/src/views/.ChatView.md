A React component that provides a complete chat interface for the Flamingo AI assistant (Fae), featuring message history, quick actions, resumable dialogs, and real-time connection status monitoring.

## Key Components

- **ChatView** - Main chat interface component with conditional layout based on message state
- **State Management** - Manages current AI model, resumable dialog, and chat metadata
- **Connection Monitoring** - Real-time connection status with disconnect handling
- **Dialog Resume** - Allows users to continue previous conversations
- **Quick Actions** - Pre-defined help shortcuts for common user requests
- **Model Display** - Shows current AI model and provider information

## Usage Example

```typescript
import { ChatView } from './components/ChatView'

function App() {
  return (
    <div className="app">
      <ChatView />
    </div>
  )
}

// The component automatically:
// - Loads resumable dialogs on mount
// - Handles message sending/receiving
// - Shows connection status
// - Displays appropriate UI based on chat state
```

The component renders different layouts depending on whether messages exist: an empty state with quick actions and resumable dialogs, or a full message list with chat history. It integrates with the OpenFrame chat system using NATS messaging when enabled and provides visual feedback for connection status, typing indicators, and model information.
