A React hook that exchanges development tickets for authentication tokens and handles the complete authentication flow including user data retrieval and dashboard redirection.

## Key Components

- **exchangeTicket**: Async function that exchanges a dev ticket for auth tokens, fetches user data, updates auth store, and redirects to dashboard
- **Error Handling**: Uses `useToast` for user feedback following the mandatory pattern from CLAUDE.md
- **Token Management**: Integrates with `useTokenStorage` to securely store authentication tokens
- **User State**: Updates `useAuthStore` with complete user profile and organization data
- **Navigation**: Automatically redirects to dashboard on successful authentication

## Usage Example

```typescript
import { useDevTicketExchange } from './hooks/use-dev-ticket-exchange'

function DevLoginPage() {
  const { exchangeTicket } = useDevTicketExchange()

  const handleTicketSubmit = async (ticket: string) => {
    const result = await exchangeTicket(ticket)
    
    if (result.success) {
      // User automatically redirected to dashboard
      console.log('Authentication successful')
    } else {
      // Error toast already shown by the hook
      console.log('Authentication failed')
    }
  }

  return (
    <form onSubmit={(e) => {
      e.preventDefault()
      const formData = new FormData(e.currentTarget)
      handleTicketSubmit(formData.get('ticket') as string)
    }}>
      <input name="ticket" placeholder="Enter dev ticket" required />
      <button type="submit">Exchange Ticket</button>
    </form>
  )
}
```

The hook handles the complete authentication workflow: ticket exchange → token storage → user data fetch → auth store update → dashboard redirect, with comprehensive error handling and user feedback throughout the process.
