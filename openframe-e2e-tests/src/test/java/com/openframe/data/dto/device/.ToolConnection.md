A data transfer object (DTO) that represents a connection between a monitoring tool/agent and a device in the OpenFrame platform, tracking connection lifecycle and synchronization status.

## Key Components

- **Connection Identity**: `id`, `machineId`, and `agentToolId` for unique identification
- **Tool Information**: `toolType` field to categorize the connected monitoring tool
- **Connection Status**: `status` field using `ConnectionStatus` enum to track current state
- **Metadata Storage**: `metadata` field for additional tool-specific configuration data
- **Timestamp Tracking**: `connectedAt`, `lastSyncAt`, and `disconnectedAt` for connection lifecycle monitoring
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic code generation

## Usage Example

```java
// Create a new tool connection
ToolConnection connection = ToolConnection.builder()
    .id("conn-123")
    .machineId("machine-456")
    .toolType("RMM_AGENT")
    .agentToolId("agent-789")
    .status(ConnectionStatus.CONNECTED)
    .metadata("{\"version\": \"1.2.3\", \"features\": [\"monitoring\", \"patching\"]}")
    .connectedAt("2024-01-15T10:30:00Z")
    .lastSyncAt("2024-01-15T11:00:00Z")
    .build();

// Update connection status
connection.setStatus(ConnectionStatus.DISCONNECTED);
connection.setDisconnectedAt("2024-01-15T12:00:00Z");
```
