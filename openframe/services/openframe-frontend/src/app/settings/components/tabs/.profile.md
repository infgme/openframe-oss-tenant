A React component that renders a user profile tab with display information, editing capabilities, and email verification functionality.

## Key Components

- **ProfileTab**: Main component displaying user profile information including avatar, name, email, roles, and verification status
- **Profile Display**: Shows user avatar (image or initials), display name, email, and role badges
- **Edit Functionality**: Integrates with EditProfileModal for updating first/last name
- **Email Verification**: Handles unverified email states with resend verification option
- **Loading States**: Skeleton loading for initial profile fetch and error handling

## Usage Example

```typescript
import { ProfileTab } from './profile'

export function UserSettings() {
  return (
    <div className="space-y-6">
      <h1>Account Settings</h1>
      <ProfileTab />
    </div>
  )
}
```

The component automatically:
- Fetches full profile data on mount
- Displays user roles as badges
- Shows email verification warnings for unverified accounts
- Handles profile updates with toast notifications
- Provides fallback initials when no profile image exists
