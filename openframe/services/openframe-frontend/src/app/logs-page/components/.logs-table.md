A React table component that displays and manages system logs with filtering, search, and pagination capabilities. It provides both standalone and embedded views for different use cases.

## Key Components

- **LogsTable**: Main table component with cursor pagination and real-time filtering
- **UILogEntry**: TypeScript interface defining the structure of log entries for display
- **LogsTableRef**: Imperative interface exposing refresh functionality
- **LogInfoModal**: Modal for viewing detailed log information

## Usage Example

```typescript
import { LogsTable, type LogsTableRef } from './logs-table'
import { useRef } from 'react'

// Standalone logs table
function LogsPage() {
  const tableRef = useRef<LogsTableRef>(null)
  
  const handleRefresh = () => {
    tableRef.current?.refresh()
  }
  
  return (
    <div>
      <button onClick={handleRefresh}>Refresh Logs</button>
      <LogsTable ref={tableRef} />
    </div>
  )
}

// Embedded device-specific logs
function DeviceLogsSection({ deviceId }: { deviceId: string }) {
  return (
    <LogsTable 
      deviceId={deviceId} 
      embedded={true} 
    />
  )
}
```

The component features:
- **Cursor-based pagination** with server-side data fetching
- **Real-time filtering** by severity, tool type, and organization
- **Responsive design** that adapts columns based on screen size
- **Row actions** for opening detailed log views in new tabs
- **Search functionality** with debounced input
- **Modal preview** for quick log inspection
