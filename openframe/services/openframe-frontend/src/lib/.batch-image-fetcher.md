Utility for batch fetching authenticated images, converting them to blob URLs for secure display in components that need to access protected image resources.

## Key Components

### Functions
- **`batchFetchAuthenticatedImages`** - Fetches multiple images with authentication headers and returns a map of original URLs to blob URLs
- **`useBatchImages`** - React hook for batch fetching images with caching and cleanup

### Features
- URL normalization for different path formats (`/api/`, relative paths, absolute URLs)
- Authentication token injection from localStorage when dev mode is enabled
- Automatic cache busting with timestamps
- Blob URL creation for secure image display
- Memory cleanup with URL.revokeObjectURL

## Usage Example

```typescript
// Batch fetch authenticated images
const imageUrls = ['/api/avatars/user1.jpg', 'https://example.com/image.png']
const blobUrls = await batchFetchAuthenticatedImages(imageUrls)
console.log(blobUrls['/api/avatars/user1.jpg']) // blob:http://localhost:3000/...

// React hook usage
function ImageGallery({ imagePaths }: { imagePaths: string[] }) {
  const fetchedImages = useBatchImages(imagePaths)
  
  return (
    <div>
      {imagePaths.map(path => (
        <img 
          key={path}
          src={fetchedImages[path] || '/placeholder.jpg'} 
          alt="Gallery item"
        />
      ))}
    </div>
  )
}
```

The hook automatically deduplicates URLs, caches results, and handles cleanup to prevent memory leaks from blob URLs.
