Data transfer object representing an organization entity in the OpenFrame MSP platform, containing business information, contract details, and audit fields for tracking company/client data.

## Key Components

- **Entity Fields**: Core organization data including name, category, employee count, and website
- **Contact Information**: Embedded contact details via `ContactInformationDto`
- **Contract Management**: Start/end dates and monthly revenue tracking
- **MongoDB Integration**: BSON annotations for MongoDB document mapping with ObjectId support
- **Audit Trail**: Creation, update, and soft deletion timestamps
- **Lombok Annotations**: Automatic generation of getters, setters, builders, and constructors

## Usage Example

```java
// Create a new organization
Organization mspClient = Organization.builder()
    .organizationId("ORG-001")
    .name("TechCorp Solutions")
    .category("Technology")
    .numberOfEmployees(150)
    .websiteUrl("https://techcorp.com")
    .monthlyRevenue("25000")
    .contractStartDate(LocalDate.of(2024, 1, 1))
    .contractEndDate(LocalDate.of(2025, 12, 31))
    .isDefault(false)
    .createdAt(Instant.now())
    .deleted(false)
    .build();

// Access organization data
String orgName = mspClient.getName();
Integer employeeCount = mspClient.getNumberOfEmployees();
LocalDate contractEnd = mspClient.getContractEndDate();
```

This DTO serves as the primary data structure for managing MSP client organizations within the Flamingo platform's unified interface.
