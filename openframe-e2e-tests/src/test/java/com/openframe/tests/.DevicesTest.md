Comprehensive test suite for device management functionality in the OpenFrame platform, including filtering, searching, archiving, and deletion operations.

## Key Components

- **DevicesTest** - Main test class extending `AuthorizedTest` for authenticated device operations
- **testGetDeviceFilters()** - Validates device filter structure and data availability
- **testGetDevice()** - Tests device retrieval by machine ID
- **testSearchDevices()** - Verifies device search functionality by hostname
- **testFilterDevices()** - Tests filtering devices by status and operating system
- **testArchiveDevice()** - Tests device archival process
- **testDeleteDevice()** - Tests device deletion functionality

## Usage Example

```java
// Running device filter validation
@Test
@DisplayName("Verify device filters")
public void testGetDeviceFilters() {
    DeviceFilters filters = getDeviceFilters();
    assertThat(filters).isNotNull();
    assertThat(filters.getStatuses()).isNotEmpty();
    assertThat(filters.getFilteredCount()).isNotZero();
}

// Testing device search by hostname
@Test
@DisplayName("Search device by hostname")
public void testSearchDevices() {
    List<String> hostnames = getDeviceHostnames(listedDevicesFilter());
    Machine device = searchDevice(listedDevicesFilter(), hostnames.getFirst());
    assertThat(device.getHostname()).isEqualTo(hostnames.getFirst());
}
```

**Note:** All tests are currently disabled with `@Disabled("Tests disabled - waiting for devices")` annotation, indicating they require device infrastructure to be available for execution.
