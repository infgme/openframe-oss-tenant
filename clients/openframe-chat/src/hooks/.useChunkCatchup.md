A React hook that provides chat chunk catchup functionality, wrapping the core chunk catchup system with authentication and API integration for retrieving missed chat chunks.

## Key Components

- **useChunkCatchup**: Main hook that handles chunk retrieval with token-based authentication
- **fetchChunks**: Internal callback function that fetches chunks from the API with proper authentication headers
- **UseChunkCatchupOptions**: Interface defining dialog ID and chunk received callback
- **Re-exported types**: ChunkData, NatsMessageType, UseChunkCatchupReturn from core package

## Usage Example

```typescript
import { useChunkCatchup, ChunkData, NatsMessageType } from './hooks/useChunkCatchup'

function ChatComponent() {
  const handleChunkReceived = useCallback((
    chunk: ChunkData, 
    messageType: NatsMessageType
  ) => {
    console.log('Received chunk:', chunk)
    // Process the received chunk
  }, [])

  const { 
    isConnected, 
    isLoading, 
    error 
  } = useChunkCatchup({
    dialogId: 'dialog-123',
    onChunkReceived: handleChunkReceived
  })

  if (error) return <div>Error: {error}</div>
  if (isLoading) return <div>Loading chunks...</div>
  
  return <div>Chat connected: {isConnected}</div>
}
```

The hook automatically handles token management, ensures authentication readiness, and fetches chunks with proper API authentication headers for the CLIENT chat type.
