Provides system information and device data collection capabilities for the Flamingo agent platform. This module fetches essential device metadata including hostname, agent version, and operating system type.

## Key Components

- **DeviceDataFetcher**: Main struct for collecting device information
- **get_hostname()**: Retrieves the system hostname using the `hostname` crate
- **get_agent_version()**: Returns the current agent version from Cargo package metadata
- **get_os_type()**: Detects the operating system (Windows or macOS)

## Usage Example

```rust
use crate::device_data_fetcher::DeviceDataFetcher;

// Create a new data fetcher instance
let fetcher = DeviceDataFetcher::new();

// Collect device information
if let Some(hostname) = fetcher.get_hostname() {
    println!("Device hostname: {}", hostname);
}

if let Some(version) = fetcher.get_agent_version() {
    println!("Agent version: {}", version);
}

let os_type = fetcher.get_os_type();
println!("Operating system: {}", os_type);
```

The fetcher gracefully handles errors (like hostname retrieval failures) by returning `None` and logging warnings. The agent version is embedded at compile time using the `env!` macro, ensuring version consistency. OS detection uses compile-time target configuration for reliable platform identification.
