A utility class that generates authentication-related data structures and security tokens for OAuth2/PKCE flows and password reset operations.

## Key Components

- **`resetConfirmRequest(String token)`** - Creates a password reset confirmation request with the provided token
- **`clearedCookies()`** - Returns a map of authentication cookies with empty values for logout operations
- **`generateAuthParts(User user)`** - Generates complete OAuth2/PKCE authentication components including code verifier, challenge, and state
- **`generateCodeVerifier()`** - Creates a cryptographically secure code verifier for PKCE flow
- **`generateCodeChallenge(String codeVerifier)`** - Generates SHA-256 based code challenge from verifier
- **`generateState()`** - Creates a random state parameter for CSRF protection

## Usage Example

```java
// Generate OAuth2/PKCE authentication parts for a user
User user = User.builder()
    .email("user@example.com")
    .password("password123")
    .tenantId("tenant-001")
    .build();

AuthParts authParts = AuthGenerator.generateAuthParts(user);
// Use authParts.getCodeVerifier(), authParts.getCodeChallenge(), etc.

// Create password reset request
ResetConfirmRequest resetRequest = AuthGenerator.resetConfirmRequest("reset-token-123");

// Clear authentication cookies for logout
Map<String, String> clearedCookies = AuthGenerator.clearedCookies();
// Apply cleared cookies to HTTP response
```

The class follows OAuth2 PKCE (Proof Key for Code Exchange) specifications for secure authentication flows and provides utilities for common authentication operations like logout and password reset.
