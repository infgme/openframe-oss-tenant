Manages NATS message consumption for OpenFrame client updates with automatic reconnection, retry logic, and machine-specific message filtering.

## Key Components

- **OpenFrameClientUpdateListener**: Main service that listens for client update messages from NATS JetStream
- **start()**: Spawns background task with automatic reconnection loop
- **listen()**: Core message processing loop that handles update messages
- **create_consumer()**: Creates or retrieves NATS consumer with exponential backoff retry
- **build_consumer_configuration()**: Constructs consumer config with machine-specific subjects and durability

## Usage Example

```rust
use openframe_client_update_listener::OpenFrameClientUpdateListener;

// Initialize listener with required services
let listener = OpenFrameClientUpdateListener::new(
    nats_connection_manager,
    openframe_client_update_service,
    config_service
);

// Start listening in background with automatic reconnection
let handle = listener.start().await?;

// The listener will:
// - Create machine-specific consumer for "CLIENT_UPDATE" stream
// - Filter messages for this machine using machine ID
// - Process updates via openframe_client_update_service
// - Acknowledge successful processing or leave unacked for redelivery
// - Automatically reconnect on connection failures
```

The service uses configurable retry policies, exponential backoff for consumer creation, and implements proper NATS message acknowledgment patterns for reliable message processing.
