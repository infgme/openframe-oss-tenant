A React component that displays detailed information about a script, including its properties, arguments, environment variables, and syntax with line numbers. The view supports editing and running scripts through action buttons.

## Key Components

- **ScriptDetailsView** - Main component that renders the script details page
- **useScriptDetails** - Custom hook for fetching script data
- **ScriptInfoSection** - Component displaying basic script information
- **handleCodeScroll** - Synchronized scrolling between line numbers and code content
- **Navigation handlers** - Functions for back, edit, and run actions

## Usage Example

```typescript
import { ScriptDetailsView } from './script-details-view'

// Display script details page
function ScriptPage({ params }: { params: { id: string } }) {
  return <ScriptDetailsView scriptId={params.id} />
}

// Navigate programmatically
const router = useRouter()
router.push('/scripts/details/script-123')

// The component handles loading states, errors, and displays:
// - Script information (name, description, etc.)
// - Script arguments and environment variables in cards
// - Syntax highlighting with line numbers
// - Action buttons for editing and running scripts
```

The component provides a comprehensive view of script details with synchronized scrolling code display, responsive layout for arguments/environment variables, and proper error handling for missing or failed script loads.
