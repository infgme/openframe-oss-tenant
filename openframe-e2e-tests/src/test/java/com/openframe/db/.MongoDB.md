A thread-safe MongoDB connection manager that provides database access and collection retrieval for the OpenFrame platform with automatic POJO codec support.

## Key Components

- **`getDatabase()`** - Returns a configured MongoDatabase instance with POJO codec registry
- **`getCollection(String name, Class<T> docClass)`** - Generic method to retrieve typed MongoDB collections
- **`openConnection()`** - Establishes authenticated MongoDB connection using configuration settings
- **`closeConnection()`** - Properly closes MongoDB client connections and cleans up resources
- **Thread-local storage** - Uses `ThreadLocal<MongoClient>` and `ThreadLocal<MongoDatabase>` for thread safety

## Usage Example

```java
// Get a database instance
MongoDatabase db = MongoDB.getDatabase();

// Get a typed collection for User documents
MongoCollection<User> userCollection = MongoDB.getCollection("users", User.class);

// Insert a user document (POJO automatically encoded)
User newUser = new User("john@example.com", "John Doe");
userCollection.insertOne(newUser);

// Query users
List<User> activeUsers = userCollection.find(eq("status", "active")).into(new ArrayList<>());

// Clean up when done
MongoDB.closeConnection();
```

The class automatically handles MongoDB authentication using credentials from `MongoConfig` and configures POJO codec providers for seamless Java object serialization/deserialization.
