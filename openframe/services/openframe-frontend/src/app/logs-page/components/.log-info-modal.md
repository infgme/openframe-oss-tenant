A React modal component that displays detailed information for log entries, including metadata, raw data, and associated device information.

## Key Components

- **LogInfoModal**: Main modal component with slide-in panel animation from the right
- **InfoField**: Reusable component for displaying label-value pairs with consistent styling
- **Props Interface**: Defines log structure with metadata, device info, and status details
- **Device Integration**: Fetches and displays full device details when deviceId is available
- **Loading States**: Shows loading indicators for both log details and device information

## Usage Example

```typescript

  )
}
```

The modal features keyboard navigation (ESC to close), click-outside closing, responsive design, and integrates with the device management system to show comprehensive log context with associated device information.
