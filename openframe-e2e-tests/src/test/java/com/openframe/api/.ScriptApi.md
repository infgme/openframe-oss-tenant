REST API client for managing automation scripts in the OpenFrame tactical RMM system. Provides methods to list, retrieve, create, and update script configurations.

## Key Components

- **listScripts()** - Retrieves all available scripts as a list
- **getScript(Integer id)** - Fetches a specific script by its ID
- **editScript(Script script)** - Updates an existing script configuration
- **addScript(Script script)** - Creates a new script in the system

## Usage Example

```java
// List all available scripts
List<Script> allScripts = ScriptApi.listScripts();

// Get a specific script by ID
Script script = ScriptApi.getScript(123);

// Create a new script
Script newScript = new Script();
newScript.setName("System Cleanup");
newScript.setContent("echo 'Cleaning system...'");
String response = ScriptApi.addScript(newScript);

// Update an existing script
script.setContent("echo 'Updated script content'");
String updateResponse = ScriptApi.editScript(script);
```

The API uses RestAssured for HTTP operations and integrates with the tactical RMM tools endpoint. All methods include proper authorization through `getAuthorizedSpec()` and return appropriate HTTP status codes (200 for success).
