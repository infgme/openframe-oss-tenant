Zustand store managing background dialog message state for the Mingo AI assistant, including per-dialog message history, unread counts, and typing indicators across multiple concurrent conversations.

## Key Components

**State Properties:**
- `dialogMessages` - Per-dialog message storage with 50 message limit
- `unreadCounts` - Track unread message counts for each dialog
- `backgroundTypingIndicators` - Typing status for background dialogs
- `activeDialogId` - Currently active dialog identifier

**Core Actions:**
- `addBackgroundMessage()` - Add message to background dialog with deduplication
- `setActiveDialogId()` - Switch active dialog context
- `incrementUnreadCount()` / `resetUnreadCount()` - Manage notification badges
- `setBackgroundTyping()` - Update typing indicators
- `mergeDialogMessages()` - Merge new messages avoiding duplicates
- `moveBackgroundToActive()` - Transfer background messages to active state

**Utility Methods:**
- `initializeDialog()` - Set up new dialog with default state
- `clearDialogData()` - Remove all data for a specific dialog
- `getDialogMessages()` / `getUnreadCount()` / `isBackgroundTyping()` - State getters

## Usage Example

```typescript
import { useMingoBackgroundMessagesStore } from './mingo-background-messages-store'

function ChatSidebar() {
  const { 
    dialogMessages, 
    unreadCounts, 
    setActiveDialogId,
    addBackgroundMessage,
    resetUnreadCount 
  } = useMingoBackgroundMessagesStore()

  const handleDialogSwitch = (dialogId: string) => {
    setActiveDialogId(dialogId)
    resetUnreadCount(dialogId)
  }

  const handleNewBackgroundMessage = (dialogId: string, message: Message) => {
    addBackgroundMessage(dialogId, message)
    if (dialogId !== activeDialogId) {
      incrementUnreadCount(dialogId)
    }
  }

  return (
    <div>
      {Object.keys(dialogMessages).map(dialogId => (
        <DialogItem 
          key={dialogId}
          unreadCount={unreadCounts[dialogId] || 0}
          onClick={() => handleDialogSwitch(dialogId)}
        />
      ))}
    </div>
  )
}
```
