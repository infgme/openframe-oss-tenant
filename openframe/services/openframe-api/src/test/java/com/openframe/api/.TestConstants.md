A utility class providing standardized test constants for authentication and API testing within the OpenFrame platform.

## Key Components

- **TEST_CLIENT_ID** - Standard client identifier for test scenarios
- **TEST_CLIENT_SECRET** - Client secret for authentication testing
- **TEST_ACCESS_TOKEN** - Mock access token for API authorization tests
- **TEST_REFRESH_TOKEN** - Mock refresh token for token renewal testing
- **TEST_AUTH_CODE** - Authorization code for OAuth flow testing

## Usage Example

```java
import com.openframe.api.TestConstants;

@Test
public void testOAuthFlow() {
    // Use predefined test constants for consistent testing
    AuthRequest request = new AuthRequest()
        .clientId(TestConstants.TEST_CLIENT_ID)
        .clientSecret(TestConstants.TEST_CLIENT_SECRET)
        .authorizationCode(TestConstants.TEST_AUTH_CODE);
    
    TokenResponse response = authService.exchangeCodeForTokens(request);
    
    assertThat(response.getAccessToken()).isNotNull();
}

@Test
public void testTokenRefresh() {
    RefreshRequest request = new RefreshRequest()
        .clientId(TestConstants.TEST_CLIENT_ID)
        .refreshToken(TestConstants.TEST_REFRESH_TOKEN);
    
    TokenResponse response = authService.refreshToken(request);
    assertThat(response.getAccessToken()).isNotNull();
}
```

The class follows the utility pattern with a private constructor to prevent instantiation and provides static final constants for consistent test data across the test suite.
