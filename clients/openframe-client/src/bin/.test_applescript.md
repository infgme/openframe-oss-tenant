A test utility for validating AppleScript command execution with administrator privileges, specifically testing the command format used in OpenFrame's privilege escalation functionality.

## Key Components

- **apple_script formatting**: Constructs properly escaped AppleScript commands for administrator privilege elevation
- **Command execution**: Uses `osascript` to execute AppleScript commands via system process
- **Status validation**: Tests command success/failure and provides detailed error reporting
- **Quote escaping**: Demonstrates proper string escaping for AppleScript syntax

## Usage Example

```rust
use std::process::Command;

// Test AppleScript command formatting
let command = "echo";
let args = ["Hello from elevated privileges!"];

let apple_script = format!(
    "do shell script \"{} {}\" with administrator privileges with prompt \"OpenFrame requires administrator privileges\"",
    command, args[0]
);

// Execute the AppleScript
let result = Command::new("osascript")
    .arg("-e")
    .arg(apple_script)
    .status();

match result {
    Ok(status) if status.success() => println!("SUCCESS: Format works correctly!"),
    Ok(status) => println!("FAILURE: Exit code: {:?}", status.code()),
    Err(e) => println!("FAILURE: Error: {}", e),
}
```

This test validates the critical AppleScript formatting used in OpenFrame's `run_as_admin` functionality, ensuring proper quote escaping and administrator privilege prompting work correctly on macOS systems.
