A React component that renders dynamic tab content for device views by resolving and displaying the appropriate tab component based on the active tab selection.

## Key Components

- **DeviceTabContent**: Main functional component that handles tab content rendering
- **DeviceTabContentProps**: TypeScript interface defining component props
- **getTabComponent**: External function imported from device-tabs module for tab component resolution

## Usage Example

```typescript
import { DeviceTabContent } from './device-tab-content'

function DevicePanel() {
  const [activeTab, setActiveTab] = useState('overview')
  const deviceData = { id: 'device-123', name: 'Server-01' }

  return (
    <div>
      <DeviceTabContent 
        activeTab={activeTab}
        device={deviceData}
      />
    </div>
  )
}
```

The component leverages the OpenFrame TabContent wrapper and dynamically resolves tab components through the `getTabComponent` utility, passing device data as props to the rendered tab component. This pattern enables flexible, modular device tab management within the Flamingo platform.
