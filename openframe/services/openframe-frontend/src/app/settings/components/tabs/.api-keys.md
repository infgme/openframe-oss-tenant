A React component that provides a comprehensive interface for managing API keys with CRUD operations, status management, and integrated documentation access.

## Key Components

- **ApiKeysTab**: Main component rendering the API keys management interface
- **Table Columns**: Configured columns for name, status, key ID, usage statistics, creation date, and expiration
- **Modal Management**: State handlers for create, edit, details, regenerate, and disable operations
- **Action Buttons**: Header actions for API documentation and key creation
- **Row Actions**: Per-key actions menu with edit, regenerate, and enable/disable options

## Usage Example

```typescript
import { ApiKeysTab } from './api-keys'

export default function ApiKeysPage() {
  return (
    <div className="container mx-auto">
      <ApiKeysTab />
    </div>
  )
}
```

The component integrates with the `useApiKeys` hook for data management and displays API keys in a sortable table format. It includes comprehensive modal dialogs for all key operations, from creation to regeneration, with proper confirmation flows for destructive actions. The interface also provides direct access to Swagger API documentation and displays key usage statistics and expiration dates.
