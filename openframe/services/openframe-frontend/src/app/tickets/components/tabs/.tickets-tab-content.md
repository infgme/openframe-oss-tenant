A client-side React component that dynamically renders tab content for the tickets section based on the active tab selection, with error handling for invalid tabs.

## Key Components

- **TicketsTabContent**: Main component that takes an `activeTab` prop and renders the corresponding tab component
- **getTabComponent**: Utility function from OpenFrame core that resolves tab configurations to actual components
- **TICKETS_TABS**: Configuration object defining available ticket tabs
- **Error State**: Fallback UI displayed when an invalid tab is selected

## Usage Example

```typescript
import { TicketsTabContent } from './tickets-tab-content'

function TicketsPage() {
  const [activeTab, setActiveTab] = useState('open-tickets')

  return (
    <div>
      <nav>
        <button onClick={() => setActiveTab('open-tickets')}>
          Open Tickets
        </button>
        <button onClick={() => setActiveTab('closed-tickets')}>
          Closed Tickets
        </button>
      </nav>
      
      <TicketsTabContent activeTab={activeTab} />
    </div>
  )
}
```

The component provides a consistent 400px minimum height container and gracefully handles invalid tab selections with a user-friendly error message. It leverages OpenFrame's tab system for dynamic component resolution, making it easy to extend with new ticket-related tabs.
