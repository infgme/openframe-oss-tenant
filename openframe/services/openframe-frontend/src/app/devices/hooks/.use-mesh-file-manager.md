A React hook that manages file operations for MeshCentral devices, providing a complete interface for remote file system access with upload/download capabilities, clipboard operations, and search functionality.

## Key Components

- **File Manager Interface**: Wraps `MeshCentralFileManager` for device file system operations
- **Connection Management**: Handles WebSocket connections with state tracking and error handling  
- **File Operations**: Complete CRUD operations (create, read, update, delete) for files and folders
- **Transfer Progress**: Real-time upload/download progress tracking with cancellation support
- **Clipboard System**: Copy/cut/paste operations for file management workflows
- **Search Functionality**: File search with async result handling and cancellation
- **Selection Management**: Multi-file selection with batch operations support

## Usage Example

```typescript
import { useMeshFileManager } from './use-mesh-file-manager'

function FileManagerComponent({ deviceId }: { deviceId: string }) {
  const {
    files,
    currentPath,
    selectedFiles,
    connectionState,
    loading,
    uploadProgress,
    downloadProgress,
    navigateToPath,
    createFolder,
    uploadFile,
    downloadFile,
    deleteItems,
    copyToClipboard,
    pasteFiles,
    searchFiles,
    selectFile,
    handleFileAction
  } = useMeshFileManager({
    meshcentralAgentId: deviceId,
    isRemote: true,
    onError: (error) => console.error('File manager error:', error)
  })

  const handleFileUpload = async (file: File) => {
    try {
      await uploadFile(file)
    } catch (error) {
      console.error('Upload failed:', error)
    }
  }

  const handleSearch = async (query: string) => {
    if (query.trim()) {
      await searchFiles(query)
    }
  }

  if (connectionState === 'connecting') {
    return <div>Connecting to file system...</div>
  }

  return (
    <div>
      <div>Current Path: {currentPath}</div>
      <div>Connection: {connectionState}</div>
      
      {uploadProgress && (
        <div>Uploading: {uploadProgress.fileName} ({uploadProgress.progress}%)</div>
      )}
      
      <input 
        type="file" 
        onChange={(e) => e.target.files?.[0] && handleFileUpload(e.target.files[0])} 
      />
      
      <input 
        type="text" 
        placeholder="Search files..." 
        onChange={(e) => handleSearch(e.target.value)} 
      />
      
      <div>
        {files.map(file => (
          <div 
            key={file.id}
            onClick={() => selectFile(file.id, !selectedFiles.includes(file.id))}
          >
            {file.name} ({file.type})
          </div>
        ))}
      </div>
    </div>
  )
}
```

The hook manages connection lifecycle automatically, handles React Strict Mode double initialization, and provides comprehensive error handling with toast notifications. It supports both server and remote file system access through MeshCentral's WebRTC-based file transfer protocol.
