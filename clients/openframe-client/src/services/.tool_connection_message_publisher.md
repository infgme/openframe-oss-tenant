A specialized publisher service that publishes tool connection messages to NATS topics for specific machines, handling agent tool registration events across the Flamingo platform.

## Key Components

- **ToolConnectionMessagePublisher** - Main publisher struct that wraps a NATS message publisher
- **publish()** - Async method to publish tool connection events with machine ID, agent tool ID, and tool type
- **build_topic_name()** - Creates machine-specific topic names in format `machine.{machine_id}.tool-connection`
- **build_message()** - Constructs `ToolConnectionMessage` instances from provided parameters

## Usage Example

```rust
use crate::services::tool_connection_message_publisher::ToolConnectionMessagePublisher;
use crate::services::nats_message_publisher::NatsMessagePublisher;

// Initialize the publisher
let nats_publisher = NatsMessagePublisher::new(nats_client);
let tool_publisher = ToolConnectionMessagePublisher::new(nats_publisher);

// Publish a tool connection event
let result = tool_publisher.publish(
    "machine-123".to_string(),
    "agent-tool-456".to_string(),
    "monitoring".to_string()
).await;

match result {
    Ok(_) => println!("Tool connection published successfully"),
    Err(e) => eprintln!("Failed to publish: {}", e),
}
```
