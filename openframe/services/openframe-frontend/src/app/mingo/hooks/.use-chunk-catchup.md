Custom React hook that provides client-side chunk catchup functionality for retrieving and processing missed message chunks in real-time conversations, built on top of the core OpenFrame chunk catchup system.

## Key Components

- **`useChunkCatchup`** - Main hook that wraps the core chunk catchup functionality with API integration
- **`fetchChunks`** - Callback function that retrieves chunks from the dialog chunks API endpoint
- **Type exports** - Re-exports `ChunkData`, `NatsMessageType`, and `UseChunkCatchupReturn` from the core package

## Usage Example

```typescript
import { useChunkCatchup } from './use-chunk-catchup'
import type { ChunkData, NatsMessageType } from './use-chunk-catchup'

function ChatComponent() {
  const { isConnected, error } = useChunkCatchup({
    dialogId: 'dialog-123',
    onChunkReceived: (chunk: ChunkData, messageType: NatsMessageType) => {
      console.log('Received chunk:', chunk)
      // Process the chunk data (add to messages, update UI, etc.)
    }
  })

  if (error) {
    return <div>Error loading chat: {error}</div>
  }

  return (
    <div>
      <div>Status: {isConnected ? 'Connected' : 'Connecting...'}</div>
      {/* Chat messages UI */}
    </div>
  )
}
```

The hook automatically fetches missed chunks for admin chat types and handles real-time message synchronization through the NATS messaging system, ensuring users see all messages even if they were offline or experienced connection issues.
