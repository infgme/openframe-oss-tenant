A data transfer object that encapsulates complete contact information for an organization, including contact persons and both physical and mailing addresses.

## Key Components

- **contacts** - List of `ContactPersonDto` objects representing individual contacts
- **physicalAddress** - `AddressDto` object for the organization's physical location
- **mailingAddress** - `AddressDto` object for mailing correspondence
- **mailingAddressSameAsPhysical** - Boolean flag indicating if mailing address matches physical address

## Usage Example

```java
// Create contact information for an organization
ContactInformationDto contactInfo = ContactInformationDto.builder()
    .contacts(List.of(
        ContactPersonDto.builder()
            .name("John Smith")
            .email("john.smith@company.com")
            .build()
    ))
    .physicalAddress(AddressDto.builder()
        .street("123 Main St")
        .city("Anytown")
        .state("CA")
        .zipCode("12345")
        .build())
    .mailingAddressSameAsPhysical(true)
    .build();

// Access contact information
List<ContactPersonDto> contacts = contactInfo.getContacts();
boolean sameAddress = contactInfo.getMailingAddressSameAsPhysical();
```

This DTO uses Lombok annotations for automatic generation of getters, setters, constructors, and builder pattern, making it ideal for API responses and data transfer operations in the OpenFrame platform.
