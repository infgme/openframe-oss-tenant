A simple DTO class that represents filter options for device filtering operations, containing a value-label pair with an optional count field.

## Key Components

- **`value`** - String representing the internal filter value
- **`label`** - String representing the display text for the filter option
- **`count`** - Integer indicating how many items match this filter option (optional)

## Usage Example

```java
// Creating a device filter option for device status
DeviceFilterOption statusFilter = DeviceFilterOption.builder()
    .value("ACTIVE")
    .label("Active Devices")
    .count(42)
    .build();

// Creating a filter option without count
DeviceFilterOption typeFilter = new DeviceFilterOption(
    "LAPTOP", 
    "Laptop Computers", 
    null
);

// Using in a filter dropdown context
List<DeviceFilterOption> statusOptions = Arrays.asList(
    new DeviceFilterOption("ACTIVE", "Active", 15),
    new DeviceFilterOption("INACTIVE", "Inactive", 3),
    new DeviceFilterOption("MAINTENANCE", "Under Maintenance", 2)
);
```

This class is typically used in UI components like dropdowns or filter panels where users need to see both human-readable labels and the count of items for each filter option, while the system uses the internal value for actual filtering operations.
