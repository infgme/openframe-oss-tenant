A mock service that simulates streaming chat responses with AI assistant capabilities, including tool execution demonstrations for testing chat interfaces.

## Key Components

- **MockChatService** - Main service class providing simulated chat responses
- **responses** - Array of predefined IT support responses for different scenarios
- **streamResponse()** - Primary method that yields message segments as an async generator
- **streamResponseWithTool()** - Demonstrates tool execution workflow with Fleet MDM integration
- **streamResponseWithError()** - Testing method that randomly simulates connection errors

## Usage Example

```typescript
import { MockChatService } from './mockChatService'

const chatService = new MockChatService()

// Stream a basic response
for await (const segment of chatService.streamResponse("My computer is running slow")) {
  if (segment.type === 'text') {
    console.log(segment.text)
  } else if (segment.type === 'tool_execution') {
    console.log('Tool execution:', segment.data.type)
  }
}

// Test with error simulation
try {
  for await (const segment of chatService.streamResponseWithError("Help me")) {
    // Handle segments
  }
} catch (error) {
  console.error('Chat error:', error.message)
}
```

The service intelligently matches user messages to appropriate responses (slow → performance diagnostics, password → recovery protocol) and randomly demonstrates tool execution with realistic Fleet MDM queries and results.
