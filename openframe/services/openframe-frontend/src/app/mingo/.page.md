A Next.js page component that provides the main Mingo AI chat interface with real-time messaging, dialog management, and NATS subscription handling for background dialog updates.

## Key Components

- **Dialog Management**: Creates, selects, and manages chat dialogs with URL-based routing
- **Real-time Messaging**: Handles streaming AI responses and background message updates via NATS
- **State Management**: Coordinates multiple stores for dialog details and background messages
- **Subscription Management**: Maintains persistent NATS connections for active dialogs
- **Welcome Messages**: Automatically adds greeting messages for new dialogs

The component features a two-panel layout with a sidebar for dialog history and a main area for the active conversation, with persistent background subscriptions for real-time updates across all dialogs.
