API client for managing user invitations in the OpenFrame platform, providing methods for inviting users, accepting invitations, and handling invitation conflicts.

## Key Components

- **inviteUser()** - Sends a user invitation and expects successful creation (201 status)
- **attemptInviteUser()** - Attempts to invite a user but expects a conflict response (409 status) 
- **acceptInvitation()** - Accepts an invitation using an unauthorized request spec
- **attemptAcceptInvitation()** - Attempts to accept an invitation but expects a conflict (409 status)
- **revokeInvitation()** - Deletes/revokes an invitation by ID

The class uses RestAssured for HTTP requests with different authorization specs - authorized requests for sending/revoking invitations, and unauthorized requests for accepting invitations (since the user may not be authenticated yet).

## Usage Example

```java
// Send an invitation to a new user
InvitationRequest inviteRequest = new InvitationRequest("user@example.com", "ADMIN");
Invitation invitation = InvitationApi.inviteUser(inviteRequest);

// Accept an invitation (typically done by the invited user)
AcceptInvitationRequest acceptRequest = new AcceptInvitationRequest(invitation.getToken(), "password123");
AcceptInvitationResponse response = InvitationApi.acceptInvitation(acceptRequest);

// Revoke an invitation if needed
InvitationApi.revokeInvitation(invitation.getId());

// Handle expected conflicts when testing duplicate invitations
InvitationConflictResponse conflict = InvitationApi.attemptInviteUser(inviteRequest);
```
