A React hook that manages user invitations with support for pagination, status tracking, and invitation lifecycle operations (send, resend, revoke).

## Key Components

- **InvitationRecord**: TypeScript interface defining invitation data structure with status, expiration, and role information
- **useInvitations**: Main hook providing paginated invitation data and mutation functions
- **InvitationStatus**: Enum defining PENDING and EXPIRED states
- **API Functions**: Async functions for fetching, revoking, resending, and creating invitations
- **Query Keys**: Structured keys for React Query cache management

## Usage Example

```typescript
import { useInvitations, InvitationStatus } from './use-invitations'

function InvitationManager() {
  const {
    invitations,
    isLoading,
    page,
    totalPages,
    hasNext,
    revokeInvitation,
    resendInvitation,
    inviteUsers
  } = useInvitations(0, 10) // page 0, size 10

  const handleInviteUsers = async () => {
    try {
      await inviteUsers(['user1@example.com', 'user2@example.com'])
      console.log('Invitations sent successfully')
    } catch (error) {
      console.error('Failed to send invitations:', error)
    }
  }

  return (
    <div>
      {invitations.map((invitation) => (
        <div key={invitation.id}>
          <span>{invitation.email}</span>
          <span>{invitation.status}</span>
          <button onClick={() => resendInvitation(invitation.id)}>
            Resend
          </button>
          <button onClick={() => revokeInvitation(invitation.id)}>
            Revoke
          </button>
        </div>
      ))}
      <button onClick={handleInviteUsers}>Send New Invitations</button>
    </div>
  )
}
```

The hook automatically handles error messaging through toast notifications and cache invalidation for real-time updates across the application.
