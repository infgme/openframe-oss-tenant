A React component that provides a modern authentication login interface with support for multiple SSO providers and tenant discovery.

## Key Components

- **AuthLoginSection** - Main authentication component with SSO provider selection
- **TenantInfo** - Interface for tenant information (name and domain)
- **SSOProvider** - Interface for SSO provider configuration with enabled state
- **AuthLoginSectionProps** - Component props including authentication handlers and state

## Features

- **OpenFrame SSO Priority** - Displays OpenFrame SSO as the primary authentication option
- **Multiple SSO Providers** - Supports Google, Microsoft, Slack, GitHub and custom providers
- **Provider Filtering** - Separates OpenFrame SSO from standard providers for better UX
- **Loading States** - Visual feedback during authentication attempts
- **Responsive Design** - Mobile-friendly layout with proper spacing and typography

## Usage Example

```typescript
import { AuthLoginSection } from './login-section'

function LoginPage() {
  const handleSSO = async (provider: string) => {
    // Handle SSO authentication
    await authenticateWithProvider(provider)
  }

  const handleBack = () => {
    // Navigate back to previous step
    router.back()
  }

  return (
    <AuthLoginSection
      email="user@example.com"
      tenantInfo={{ tenantName: "Acme Corp", tenantDomain: "acme.com" }}
      hasDiscoveredTenants={true}
      availableProviders={['openframe-sso', 'google', 'microsoft']}
      onSSO={handleSSO}
      onBack={handleBack}
      isLoading={false}
    />
  )
}
```
