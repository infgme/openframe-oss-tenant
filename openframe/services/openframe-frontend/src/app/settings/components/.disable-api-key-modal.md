A React modal component that prompts users to confirm disabling/deactivating an API key with a warning message and confirmation flow.

## Key Components

- **DisableApiKeyModalProps** - Interface defining modal props including open state, close handler, API key name, and confirm action
- **DisableApiKeyModal** - Main modal component with header, confirmation message, and action buttons

## Usage Example

```typescript
import { DisableApiKeyModal } from './disable-api-key-modal'

function ApiKeyManagement() {
  const [showDisableModal, setShowDisableModal] = useState(false)
  const [selectedKeyName, setSelectedKeyName] = useState('')

  const handleDisableKey = async () => {
    // Disable API key logic
    await disableApiKey(selectedKeyName)
    setShowDisableModal(false)
  }

  return (
    <>
      <Button 
        onClick={() => {
          setSelectedKeyName('Production Key')
          setShowDisableModal(true)
        }}
      >
        Disable Key
      </Button>

      <DisableApiKeyModal
        isOpen={showDisableModal}
        onClose={() => setShowDisableModal(false)}
        apiKeyName={selectedKeyName}
        onConfirm={handleDisableKey}
      />
    </>
  )
}
```

The modal uses OpenFrame's design system components and features a destructive action pattern with error styling on the confirm button to emphasize the action's impact.
