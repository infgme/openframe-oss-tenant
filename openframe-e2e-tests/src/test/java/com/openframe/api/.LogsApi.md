A GraphQL API client for managing log data operations in the OpenFrame platform, providing methods to retrieve, filter, and search system logs.

## Key Components

- **`getLogFilters()`** - Retrieves available log filter options
- **`getLogs()`** - Fetches all log events without filters
- **`getLogs(LogFilterInput filter)`** - Retrieves filtered log events based on criteria
- **`searchLogs(String search)`** - Performs text-based search across log data
- **`getLogDetails(LogEvent logEvent)`** - Fetches detailed information for a specific log event

## Usage Example

```java
// Get available filter options
LogFilters filters = LogsApi.getLogFilters();

// Retrieve all logs
List<LogEvent> allLogs = LogsApi.getLogs();

// Filter logs by specific criteria
LogFilterInput filter = new LogFilterInput();
filter.setToolType("authentication");
filter.setEventType("login_attempt");
List<LogEvent> filteredLogs = LogsApi.getLogs(filter);

// Search logs by text
List<LogEvent> searchResults = LogsApi.searchLogs("error 500");

// Get detailed information for a specific log
LogEvent event = allLogs.get(0);
LogDetails details = LogsApi.getLogDetails(event);
```

The API uses GraphQL endpoints with REST-assured for HTTP communication and automatically handles authentication through `getAuthorizedSpec()`. All methods return strongly-typed DTOs for easy integration with OpenFrame's logging system.
