A test implementation of Spring Security's AuthenticationManager that always throws a BadCredentialsException, designed for testing authentication failure scenarios in the OpenFrame API.

## Key Components

- **TestAuthenticationManager**: Implements `AuthenticationManager` interface
- **authenticate()**: Always throws `BadCredentialsException` regardless of input credentials
- Used for unit testing authentication failure paths and error handling

## Usage Example

```java
// In test configuration
@TestConfiguration
public class TestSecurityConfig {
    
    @Bean
    @Primary
    public AuthenticationManager testAuthenticationManager() {
        return new TestAuthenticationManager();
    }
}

// In unit tests
@Test
public void shouldHandleAuthenticationFailure() {
    AuthenticationManager authManager = new TestAuthenticationManager();
    
    assertThrows(BadCredentialsException.class, () -> {
        authManager.authenticate(
            new UsernamePasswordAuthenticationToken("user", "password")
        );
    });
}
```

This utility class is particularly useful for testing security configurations, authentication filters, and exception handling without requiring actual credential validation or external authentication services.
