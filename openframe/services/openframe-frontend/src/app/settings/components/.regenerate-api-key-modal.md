A React modal component that provides a confirmation dialog for regenerating API keys with loading state management and user feedback.

## Key Components

- **RegenerateApiKeyModal**: Main modal component that handles API key regeneration confirmation
- **RegenerateApiKeyModalProps**: TypeScript interface defining component props including callbacks and state
- **Loading state management**: Prevents multiple simultaneous regeneration attempts
- **Confirmation workflow**: Clear warning messaging about key invalidation

## Usage Example

```typescript
import { RegenerateApiKeyModal } from './regenerate-api-key-modal'

function ApiKeyManagement() {
  const [showModal, setShowModal] = useState(false)
  
  const handleRegenerate = async () => {
    try {
      await apiService.regenerateKey('my-api-key')
      // Handle success (refresh keys, show notification, etc.)
    } catch (error) {
      // Handle error
    } finally {
      setShowModal(false)
    }
  }

  return (
    <>
      <Button onClick={() => setShowModal(true)}>
        Regenerate Key
      </Button>
      
      <RegenerateApiKeyModal
        isOpen={showModal}
        onClose={() => setShowModal(false)}
        apiKeyName="Production API Key"
        onConfirm={handleRegenerate}
      />
    </>
  )
}
```

The modal automatically resets its loading state when closed and provides clear visual feedback during the regeneration process. The component emphasizes the destructive nature of the action by highlighting the API key name in warning colors and explaining the immediate impact.
