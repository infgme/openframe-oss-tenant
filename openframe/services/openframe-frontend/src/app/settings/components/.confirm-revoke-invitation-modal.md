A confirmation modal component for revoking user invitations, providing a destructive action dialog with clear warning messaging and styled buttons.

## Key Components

- **ConfirmRevokeInvitationModal** - Main modal component that wraps AlertDialog functionality
- **ConfirmRevokeInvitationModalProps** - Interface defining component props including open state, email, and confirmation callback
- **AlertDialog components** - Uses OpenFrame's UI components for consistent modal structure and styling

## Usage Example

```typescript
import { ConfirmRevokeInvitationModal } from './confirm-revoke-invitation-modal'
import { useState } from 'react'

function InvitationManagement() {
  const [showRevokeModal, setShowRevokeModal] = useState(false)
  const [selectedEmail, setSelectedEmail] = useState('')

  const handleRevokeInvitation = async () => {
    try {
      await revokeInvitationAPI(selectedEmail)
      setShowRevokeModal(false)
      // Refresh invitation list
    } catch (error) {
      console.error('Failed to revoke invitation:', error)
    }
  }

  return (
    <>
      <button onClick={() => setShowRevokeModal(true)}>
        Revoke Invitation
      </button>
      
      <ConfirmRevokeInvitationModal
        open={showRevokeModal}
        onOpenChange={setShowRevokeModal}
        userEmail={selectedEmail}
        onConfirm={handleRevokeInvitation}
      />
    </>
  )
}
```

The modal features destructive styling with a red "Revoke Invitation" button, highlights the user's email in error color, and uses the OpenFrame design system's typography and spacing tokens for consistent appearance.
