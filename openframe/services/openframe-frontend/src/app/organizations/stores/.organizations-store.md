A Zustand store that manages organization data state for the Flamingo MSP platform, including client organization entries, search functionality, and loading states with immutable updates.

## Key Components

**Interfaces:**
- `OrganizationEntry` - Complete organization data structure with contact info, tier, industry, MRR, and contract details
- `OrganizationsState` - Store state interface with organizations array, search term, loading/error states, and action methods

**Store:**
- `useOrganizationsStore` - Main Zustand store with devtools and Immer middleware for immutable state updates

**Selectors:**
- `selectOrganizations`, `selectSearch`, `selectIsLoading`, `selectError` - Typed selector functions for accessing specific state slices

## Usage Example

```typescript
import { useOrganizationsStore, selectOrganizations } from './organizations-store'

// In a React component
function OrganizationsList() {
  const { 
    organizations, 
    search, 
    isLoading, 
    setOrganizations, 
    setSearch 
  } = useOrganizationsStore()
  
  // Using selectors
  const orgs = useOrganizationsStore(selectOrganizations)
  
  // Update organizations data
  const handleLoadOrganizations = (data: OrganizationEntry[]) => {
    setOrganizations(data)
  }
  
  // Filter organizations
  const filteredOrgs = organizations.filter(org => 
    org.name.toLowerCase().includes(search.toLowerCase())
  )
  
  return (
    <div>
      <input 
        value={search} 
        onChange={(e) => setSearch(e.target.value)}
        placeholder="Search organizations..."
      />
      {isLoading ? 'Loading...' : filteredOrgs.map(org => 
        <div key={org.id}>{org.name} - {org.tier}</div>
      )}
    </div>
  )
}
```
