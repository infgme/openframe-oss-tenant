A data transfer object that represents the response for user authentication status and profile information, typically returned from a "who am I" or current user endpoint.

## Key Components

- **User user** - The authenticated user's profile information
- **boolean authenticated** - Flag indicating whether the user is currently authenticated
- **Lombok annotations** - `@Data`, `@AllArgsConstructor`, `@NoArgsConstructor`, `@Builder` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a response for an authenticated user
MeResponse response = MeResponse.builder()
    .user(currentUser)
    .authenticated(true)
    .build();

// Creating a response for an unauthenticated request
MeResponse unauthenticatedResponse = MeResponse.builder()
    .user(null)
    .authenticated(false)
    .build();

// In a REST controller
@GetMapping("/api/me")
public ResponseEntity<MeResponse> getCurrentUser() {
    if (isUserAuthenticated()) {
        User currentUser = userService.getCurrentUser();
        return ResponseEntity.ok(
            MeResponse.builder()
                .user(currentUser)
                .authenticated(true)
                .build()
        );
    }
    
    return ResponseEntity.ok(
        MeResponse.builder()
            .authenticated(false)
            .build()
    );
}
```

This DTO is commonly used in authentication flows to provide both user details and authentication status in a single response object.
