A React hook that provides approval request management functionality with approve/reject actions and status tracking using TanStack Query mutations.

## Key Components

- **ApprovalRequestAction** - Type defining the structure for approval request actions (requestId and approve boolean)
- **approveRequest()** - API function that sends approval/rejection requests to the backend
- **useApprovalRequests()** - Main hook that returns approval handlers and status information
- **handleApproveRequest()** - Helper function to approve a request with optional success/error callbacks
- **handleRejectRequest()** - Helper function to reject a request with optional success/error callbacks

## Usage Example

```typescript
import { useApprovalRequests } from './use-approval-requests'
import { APPROVAL_STATUS } from '../constants'

function ApprovalComponent({ requestId }: { requestId: string }) {
  const { 
    handleApproveRequest, 
    handleRejectRequest, 
    isLoading, 
    error 
  } = useApprovalRequests()

  const onApprove = () => {
    handleApproveRequest(requestId, {
      onSuccess: (status) => {
        console.log('Request approved:', status)
        // Handle success (e.g., show notification, update UI)
      },
      onError: (error) => {
        console.error('Approval failed:', error.message)
      }
    })
  }

  const onReject = () => {
    handleRejectRequest(requestId, {
      onSuccess: (status) => {
        console.log('Request rejected:', status)
      },
      onError: (error) => {
        console.error('Rejection failed:', error.message)
      }
    })
  }

  return (
    <div>
      <button onClick={onApprove} disabled={isLoading}>
        Approve
      </button>
      <button onClick={onReject} disabled={isLoading}>
        Reject
      </button>
      {error && <p className="error">{error}</p>}
    </div>
  )
}
```

The hook automatically manages loading states and provides both high-level handler functions and direct access to the underlying TanStack Query mutation for advanced use cases.
