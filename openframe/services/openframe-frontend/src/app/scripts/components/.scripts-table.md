A React table component for displaying, filtering, and managing scripts in the OpenFrame platform.

## Key Components

- **ScriptsTable**: Main component that renders a paginated, filterable table of scripts
- **UIScriptEntry**: TypeScript interface defining the script data structure
- **URL State Management**: Persists search, filters, and pagination state in URL parameters
- **Advanced Filtering**: Multi-column filtering with badges for shell types, OS platforms, and categories
- **Debounced Search**: Real-time search with 300ms debounce to reduce API calls
- **Responsive Design**: Progressive column hiding on smaller screens (sm, md, lg, xl breakpoints)

## Usage Example

```typescript
import { ScriptsTable } from './scripts-table'

function ScriptsPage() {
  return (
    <div className="container mx-auto">
      <ScriptsTable />
    </div>
  )
}

// The component automatically handles:
// - URL state: /scripts?search=deploy&shellType=bash&page=2
// - Filtering by shell type, platform, category, and added-by
// - Pagination with 10 items per page
// - Row actions: Edit, View Details, Run Script
// - Responsive column visibility
```

The component integrates with the `useScripts` hook for data fetching and provides a comprehensive interface for script management with search, filtering, pagination, and navigation to script details and execution pages.
