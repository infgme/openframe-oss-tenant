A React hook that provides lazy organization lookup functionality, fetching and caching organization names only when needed to avoid blocking initial renders.

## Key Components

- **useOrganizationLookup** - Main hook that manages organization data fetching and caching
- **OrganizationLookupState** - Interface defining the hook's internal state structure
- **fetchOrganizationNames** - Callback function that lazily fetches organization data via GraphQL
- **getOrganizationName** - Utility function to retrieve organization name by ID
- **GET_ORGANIZATIONS_MIN_QUERY** - GraphQL query for fetching minimal organization data

## Usage Example

```typescript
function OrganizationDisplay({ organizationId }: { organizationId: string }) {
  const { 
    getOrganizationName, 
    fetchOrganizationNames, 
    isLoading, 
    hasFetched 
  } = useOrganizationLookup()

  // Trigger fetch when component mounts or organizationId changes
  useEffect(() => {
    if (!hasFetched) {
      fetchOrganizationNames()
    }
  }, [fetchOrganizationNames, hasFetched])

  const orgName = getOrganizationName(organizationId)

  if (isLoading) return <span>Loading...</span>
  if (!orgName) return <span>Unknown Organization</span>
  
  return <span>{orgName}</span>
}
```

The hook implements smart caching to prevent duplicate API calls and uses a ref to track pending requests, ensuring efficient data fetching across multiple component instances.
