A React component that manages the main settings interface with tab-based navigation and URL synchronization for the Flamingo platform.

## Key Components

- **`SettingsView`** - Main component that orchestrates the settings interface
- **Tab Management** - Handles 6 different settings tabs: AI Settings, Architecture, Company & Users, API Keys, SSO Configuration, and Profile
- **URL Synchronization** - Keeps the active tab state synchronized with URL parameters for bookmarking and navigation
- **State Management** - Uses React hooks to manage tab state and handle browser navigation events

## Usage Example

```typescript
import { SettingsView } from './settings-view'

// Use in your app layout or page component
function SettingsPage() {
  return (
    <div className="container mx-auto p-6">
      <h1 className="text-2xl font-bold mb-6">Settings</h1>
      <SettingsView />
    </div>
  )
}

// URL examples:
// /settings - loads default 'ai-settings' tab
// /settings?tab=profile - loads profile tab
// /settings?tab=api-keys - loads API keys tab
```

The component automatically validates tab IDs against available tabs, falls back to the default AI Settings tab for invalid URLs, and maintains browser history for proper back/forward navigation. It renders a horizontal tab navigation bar with corresponding content panels below.
