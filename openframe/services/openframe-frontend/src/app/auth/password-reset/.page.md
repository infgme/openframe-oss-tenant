A React component for handling password reset functionality with token-based authentication and comprehensive form validation.

## Key Components

- **PasswordResetPage** - Main component that manages password reset flow using URL token
- **Form State Management** - Handles password, confirm password, and loading states
- **Validation Logic** - Implements password strength requirements and matching validation
- **API Integration** - Calls `authApiClient.confirmPasswordReset()` to process reset
- **Toast Notifications** - Provides user feedback for success/error states
- **AuthLayout** - Wraps the form in consistent authentication page styling

## Usage Example

This component is accessed via a URL with a reset token parameter:

```typescript
// Typical URL pattern
// /auth/password-reset?token=abc123def456

// Key validation features:
// - Minimum 8 character password requirement
// - Password confirmation matching
// - Real-time validation feedback
// - Token presence validation

// API call structure:
const response = await authApiClient.confirmPasswordReset({
  token: "reset-token-from-url",
  newPassword: "user-entered-password"
})
```

The component automatically validates the reset token on mount, provides real-time form validation with visual feedback, and redirects users to the login page after successful password reset. Form submission is disabled until all validation criteria are met, ensuring a secure password reset experience.
