A React component that defines table column configurations for the devices table view in the OpenFrame platform.

## Key Components

- **`getDeviceTableColumns`** - Main function that returns column configurations for the devices table
- **`getDeviceTableRowActions`** - Returns render function for device action dropdowns
- **`OrganizationCell`** - Component for rendering organization information with optional images
- **Device Status Column** - Displays status tags with last seen timestamps
- **OS Type Column** - Shows OS badges with filtering capabilities
- **Device Name Column** - Renders device icons and display names

## Usage Example

```typescript
import { getDeviceTableColumns } from './devices-table-columns'

// Basic column configuration
const columns = getDeviceTableColumns()

// With device filters and organization images
const columns = getDeviceTableColumns(
  {
    statuses: [{ value: 'ONLINE', label: 'Online' }],
    osTypes: [{ value: 'Windows', label: 'Windows' }],
    organizationIds: [{ value: 'org1', label: 'Org Name' }]
  },
  { 'https://example.com/logo.png': '/cached/logo.png' }
)

// With action handlers
const rowActions = getDeviceTableRowActions(() => {
  console.log('Device action completed')
})
```

The component supports responsive design with different column visibility at various breakpoints, filterable columns, and integrates with the OpenFrame design system for consistent styling and icons.
