Central configuration system for OpenFrame applications that manages app types, navigation, UI components, and platform-specific settings. Provides type-safe configuration interfaces and utility functions for controlling application behavior across different deployment contexts.

## Key Components

### Core Types
- **`AppType`** - Defines supported application types (`'openframe-auth'` | `'openframe-dashboard'`)
- **`AppConfig`** - Complete application configuration interface including branding, SEO, navigation, and UI settings
- **`NavigationMenuItem`** & **`NavigationSection`** - Type-safe navigation structure definitions
- **`FooterConfig`** - Footer layout and content configuration

### Configuration Functions
- **`getCurrentAppType()`** - Retrieves the current application type from runtime environment
- **`getAppConfig()`** - Returns configuration object for current or specified app type
- **`getPlatformAssetPaths()`** - Generates platform-specific asset URLs
- **`generateStructuredData()`** - Creates JSON-LD structured data for SEO

### Utility Functions
- **`shouldShowComponent()`** - Conditionally shows UI components based on configuration
- **`isRouteAllowed()`** - Validates route access against allowed/restricted route lists
- **`shouldShowAdminMenu()`** - Controls admin menu visibility based on user role

## Usage Example

```typescript
import { getAppConfig, getCurrentAppType, shouldShowComponent } from './app-config'

// Get current app configuration
const config = getAppConfig()

// Check current app type
const appType = getCurrentAppType() // 'openframe-auth' | 'openframe-dashboard'

// Conditionally render UI components
const showUserMenu = shouldShowComponent('userMenu')
const showSearchBar = shouldShowComponent('searchBar')

// Access configuration properties
const brandColors = config.brandColors
const navigationItems = config.navigation.headerMenu

// Generate structured data for SEO
const { schemaLd, organizationLd } = generateStructuredData(config)

// Get platform assets
const assetPaths = getPlatformAssetPaths()
console.log(assetPaths.favicon) // "/assets/openframe/favicon.ico"
```
