A modal dialog component for handling password reset requests in the Flamingo platform. It provides a user-friendly interface to collect email addresses and trigger password reset email delivery with proper validation and error handling.

## Key Components

- **ForgotPasswordModal** - Main modal component with email input and submission logic
- **ForgotPasswordModalProps** - Interface defining component props including open state, change handler, and optional default email
- **Form handling** - Email validation, submission state management, and API integration
- **Toast notifications** - Success and error feedback using the OpenFrame toast system
- **Keyboard support** - Enter key submission for improved accessibility

## Usage Example

```typescript
import { ForgotPasswordModal } from './forgot-password-modal'

function LoginPage() {
  const [showForgotPassword, setShowForgotPassword] = useState(false)
  const [userEmail, setUserEmail] = useState('')

  return (
    <>
      <button onClick={() => setShowForgotPassword(true)}>
        Forgot Password?
      </button>
      
      <ForgotPasswordModal
        open={showForgotPassword}
        onOpenChange={setShowForgotPassword}
        defaultEmail={userEmail}
      />
    </>
  )
}
```

The modal automatically handles the password reset API call, displays appropriate success/error messages, and resets its state when closed. It integrates with the OpenFrame design system for consistent styling and uses the `authApiClient` for secure authentication operations.
