A data transfer object representing the response structure when an invitation operation encounters a conflict or error condition.

## Key Components

- **code** - String field containing the error/conflict code identifier
- **message** - String field containing the human-readable error/conflict description
- **Lombok annotations** - `@Data`, `@Builder`, `@AllArgsConstructor`, `@NoArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating a conflict response using builder pattern
InvitationConflictResponse conflict = InvitationConflictResponse.builder()
    .code("INVITATION_ALREADY_EXISTS")
    .message("An invitation for this email address already exists")
    .build();

// Using in REST controller error handling
@PostMapping("/invitations")
public ResponseEntity<?> createInvitation(@RequestBody InvitationRequest request) {
    try {
        // invitation creation logic
        return ResponseEntity.ok(invitation);
    } catch (InvitationConflictException e) {
        InvitationConflictResponse response = InvitationConflictResponse.builder()
            .code(e.getErrorCode())
            .message(e.getMessage())
            .build();
        return ResponseEntity.status(HttpStatus.CONFLICT).body(response);
    }
}

// Example JSON output
{
    "code": "USER_ALREADY_MEMBER",
    "message": "User is already a member of this organization"
}
```
