A React hook that manages selection and fetching of Mingo AI dialog conversations, handling both dialog metadata and paginated message loading with automatic state synchronization.

## Key Components

- **useMingoDialogSelection()** - Main hook that orchestrates dialog selection workflow
- **dialogQuery** - TanStack Query for fetching selected dialog details
- **messagesQuery** - Infinite query for paginated message loading with admin message filtering
- **selectDialogMutation** - Mutation that handles dialog selection with proper state reset
- **State synchronization effects** - React effects that sync query results with Zustand store

## Usage Example

```typescript
import { useMingoDialogSelection } from './hooks/use-mingo-dialog-selection'

function DialogInterface() {
  const {
    selectDialog,
    isSelectingDialog,
    isLoadingDialog,
    isLoadingMessages,
    dialogError,
    messagesError,
    refetchDialog,
    refetchMessages
  } = useMingoDialogSelection()

  const handleDialogClick = (dialogId: string) => {
    selectDialog(dialogId)
  }

  if (dialogError || messagesError) {
    return <div>Error: {dialogError || messagesError}</div>
  }

  return (
    <div>
      <button 
        onClick={() => handleDialogClick('dialog-123')}
        disabled={isSelectingDialog}
      >
        {isLoadingDialog ? 'Loading...' : 'Select Dialog'}
      </button>
      {isLoadingMessages && <div>Loading messages...</div>}
    </div>
  )
}
```

The hook automatically fetches all available message pages, filters for admin messages only, and synchronizes the complete dataset with the global dialog store for seamless state management across components.
