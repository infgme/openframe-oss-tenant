A client-side rendered dashboard page component that dynamically loads its content with a loading state for optimal performance.

## Key Components

- **Dashboard**: Default exported page component that renders the dashboard
- **DashboardContent**: Dynamically imported component with SSR disabled
- **AppLayout**: Layout wrapper component for consistent page structure
- **DashboardLoading**: Loading component shown during dynamic import

## Usage Example

```typescript
// This page component automatically handles:
// 1. Client-side rendering with 'use client'
// 2. Dynamic loading of dashboard content
// 3. Loading state management
// 4. Layout consistency

// The dynamic import reduces initial bundle size
const DashboardContent = nextDynamic(
  () => import('./components/dashboard-content'),
  { 
    ssr: false,  // Disable server-side rendering
    loading: () => <DashboardLoading />  // Show loading during import
  }
)

// Usage in Next.js app directory structure:
// app/dashboard/page.tsx exports this component
export default function Dashboard() {
  return (
    <AppLayout>
      <DashboardContent />
    </AppLayout>
  )
}
```

The component uses Next.js dynamic imports to code-split the dashboard content, improving initial page load times while maintaining a smooth user experience with the loading component.
