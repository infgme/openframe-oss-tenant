Test base class for unauthorized API endpoint testing, providing common setup and teardown functionality for tests that verify endpoints without authentication.

## Key Components

- **UnauthorizedTest** - Abstract base class for unauthorized endpoint tests
- **config()** - Static setup method that configures RestAssured and opens MongoDB connection
- **close()** - Static teardown method that closes the database connection

## Usage Example

```java
public class UserEndpointUnauthorizedTest extends UnauthorizedTest {

    @Test
    public void testGetUserWithoutAuth_ShouldReturn401() {
        given()
            .when()
            .get("/api/users/123")
            .then()
            .statusCode(401);
    }

    @Test
    public void testCreateUserWithoutAuth_ShouldReturn401() {
        given()
            .contentType("application/json")
            .body("{\"name\":\"John\",\"email\":\"john@test.com\"}")
            .when()
            .post("/api/users")
            .then()
            .statusCode(401);
    }
}
```

This base class ensures consistent test environment setup across all unauthorized endpoint tests, automatically handling RestAssured configuration and database connection lifecycle management. Test classes extending this base can focus solely on testing unauthorized access scenarios without worrying about infrastructure setup.
