A React component that displays system logs in a filterable, paginated table format with support for embedded mode and detailed log viewing.

## Key Components

- **LogsTable**: Main component with filtering, pagination, and modal support
- **LogsTableRef**: Ref interface exposing refresh functionality
- **UILogEntry**: TypeScript interface for transformed log data
- **LogsTableProps**: Component props including deviceId and embedded mode
- **Table Integration**: Uses OpenFrame UI components with cursor pagination
- **LogInfoModal**: Side panel for detailed log viewing

## Usage Example

```typescript
import { LogsTable, type LogsTableRef } from './logs-table'

// Basic usage - full page layout
function LogsPage() {
  return <LogsTable />
}

// Embedded mode for device-specific logs
function DeviceLogsSection({ deviceId }: { deviceId: string }) {
  const logsTableRef = useRef<LogsTableRef>(null)
  
  const handleRefresh = () => {
    logsTableRef.current?.refresh()
  }
  
  return (
    <div>
      <button onClick={handleRefresh}>Refresh Logs</button>
      <LogsTable 
        ref={logsTableRef}
        deviceId={deviceId} 
        embedded={true} 
      />
    </div>
  )
}
```

The component transforms raw log entries into a structured UI format with filtering by severity, tool type, and organization. It supports both standalone and embedded modes, with cursor-based pagination and real-time log details fetching through a modal interface.
