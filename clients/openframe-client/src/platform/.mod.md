This module serves as the main utilities module, organizing and re-exporting common utility functions for directory management, file operations, permissions handling, and platform-specific functionality.

## Key Components

**Cross-Platform Modules:**
- `DirectoryManager` - Directory creation and management utilities
- `PermissionUtils` & `Permissions` - File and directory permission handling
- `Capability` - Permission capability definitions
- `remove_directory_with_retry` - Robust directory removal with retry logic

**Windows-Specific Modules:**
- `LockingProcess` & file lock utilities - Windows file lock detection and management
- `get_powershell_path` - PowerShell executable location detection
- Windows cleanup functionality

**Error Types:**
- `DirectoryError` - Directory operation errors
- `PermissionError` - Permission-related errors

## Usage Example

```rust
use crate::utils::{
    DirectoryManager, PermissionUtils, Permissions,
    remove_directory_with_retry
};

// Directory management
let dir_manager = DirectoryManager::new();
dir_manager.ensure_directory_exists("/path/to/dir")?;

// Permission handling
let perms = Permissions::new()
    .read(true)
    .write(true)
    .execute(false);
PermissionUtils::set_permissions("/path/to/file", perms)?;

// Safe directory removal
remove_directory_with_retry("/temp/dir").await?;

#[cfg(target_os = "windows")]
{
    use crate::utils::{get_powershell_path, is_file_in_use_error};

    }
}
```
