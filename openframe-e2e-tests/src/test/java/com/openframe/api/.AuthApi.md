A utility class that provides static methods for handling OAuth 2.0 authentication flows in the OpenFrame API testing framework, supporting multi-tenant login, token management, and session handling.

## Key Components

- **OAuth Flow Methods**: `startOAuthFlow()`, `initiateAuthorization()`, `submitCredentials()`, `getAuthorizationCode()`, `exchangeCodeForTokens()` - Complete OAuth 2.0 PKCE flow implementation
- **Token Management**: `refresh()` (overloaded), `attemptRefresh()` - Handle access token refresh using refresh tokens
- **Session Management**: `logout()` (overloaded) - Terminate user sessions and clear tokens
- **Helper Methods**: `getAuthQueryParams()` - Build OAuth 2.0 authorization parameters with PKCE support

## Usage Example

```java
// Start OAuth flow for a tenant
AuthParts authParts = new AuthParts("tenant123", "user@example.com", "password");
Response oauthResponse = AuthApi.startOAuthFlow(authParts);

// Complete authentication flow
Response authResponse = AuthApi.initiateAuthorization(authParts);
Response credResponse = AuthApi.submitCredentials(authParts);
Response codeResponse = AuthApi.getAuthorizationCode(authParts);
Response tokenResponse = AuthApi.exchangeCodeForTokens(authParts);

// Refresh tokens
Map<String, String> cookies = tokenResponse.getCookies();
Map<String, String> newCookies = AuthApi.refresh(user, cookies);

// Logout user
AuthApi.logout(user, cookies);
```

The class uses RestAssured for HTTP requests and follows OAuth 2.0 PKCE (Proof Key for Code Exchange) standards with support for multi-tenant authentication flows.
