A simple data transfer object (DTO) for handling invitation requests, containing only the invitee's email address.

## Key Components

- **InvitationRequest**: Main DTO class for invitation requests
- **email**: String field to store the invitee's email address
- **Lombok annotations**: Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Creating an invitation request using builder pattern
InvitationRequest request = InvitationRequest.builder()
    .email("user@example.com")
    .build();

// Creating with constructor
InvitationRequest request2 = new InvitationRequest("admin@flamingo.run");

// Accessing the email
String inviteeEmail = request.getEmail();
```

This DTO is typically used in REST API endpoints for sending invitations to new users or team members within the OpenFrame platform.
