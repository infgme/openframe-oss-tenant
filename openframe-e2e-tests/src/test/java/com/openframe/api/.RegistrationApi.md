REST API client wrapper for user registration operations in the OpenFrame MSP platform test automation suite.

## Key Components

- **registerUser()** - Performs successful user registration with 200 status code validation
- **attemptRegistration()** - Handles failed registration attempts expecting 400 status code for error testing
- **REGISTER** - OAuth registration endpoint constant (`sas/oauth/register`)

## Usage Example

```java
// Successful user registration
UserRegistrationRequest newUser = new UserRegistrationRequest(
    "john.doe@example.com", 
    "SecurePass123", 
    "John Doe"
);
UserRegistrationResponse response = RegistrationApi.registerUser(newUser);

// Test registration failure scenarios
UserRegistrationRequest invalidUser = new UserRegistrationRequest(
    "invalid-email", 
    "weak", 
    ""
);
ErrorResponse error = RegistrationApi.attemptRegistration(invalidUser);
```

This API wrapper uses RestAssured for HTTP requests and leverages the unauthorized request specification from `RequestSpecHelper` since registration endpoints don't require authentication. Both methods automatically deserialize responses to their respective DTO objects for easy assertion in test cases.
