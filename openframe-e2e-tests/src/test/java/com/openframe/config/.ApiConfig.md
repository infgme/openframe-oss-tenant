Configuration utility class that manages the base URL for API endpoints in the OpenFrame platform, supporting environment variables and system properties with fallback to localhost.

## Key Components

- **DEFAULT_BASE_URL**: Static constant defining the default localhost URL (`https://localhost/`)
- **baseUrl**: Private static field for caching the resolved base URL
- **getBaseUrl()**: Static method that retrieves and caches the base URL from various sources

## Configuration Priority

The base URL is resolved in the following priority order:
1. System property: `api.base.url`
2. Environment variable: `API_BASE_URL`
3. Default fallback: `https://localhost/`

## Usage Example

```java
// Get the configured base URL
String apiBase = ApiConfig.getBaseUrl();

// Build complete endpoint URLs
String usersEndpoint = ApiConfig.getBaseUrl() + "api/v1/users";
String healthCheck = ApiConfig.getBaseUrl() + "health";

// The URL is automatically normalized with trailing slash
// Environment: API_BASE_URL=https://api.flamingo.run
// Result: https://api.flamingo.run/
```

## Environment Configuration

```bash
# Set via environment variable
export API_BASE_URL=https://api.flamingo.run

# Set via system property
java -Dapi.base.url=https://staging.flamingo.run MyApp
```

The class automatically ensures URLs end with a trailing slash and uses lazy initialization with caching for performance. Logging is provided via SLF4J for debugging configuration resolution.
