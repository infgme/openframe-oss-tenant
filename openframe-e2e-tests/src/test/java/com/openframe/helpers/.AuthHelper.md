Provides authentication utilities for user management and session handling in the OpenFrame platform, including user persistence and cookie management for maintaining authenticated sessions.

## Key Components

- **saveUser()** - Persists a registered user to the configured user file
- **getUser()** - Lazy-loads and returns the current user from storage
- **getCookies()** - Retrieves authentication cookies, automatically logging in if needed
- **setCookies()** - Updates the current session cookies
- **Static fields** - Caches user instance and authentication cookies for performance

## Usage Example

```java
// Save a newly registered user
User newUser = new User("username", "email@example.com");
AuthHelper.saveUser(newUser);

// Get current user (lazy-loaded from file)
User currentUser = AuthHelper.getUser();

// Get authentication cookies (auto-login if needed)
Map<String, String> sessionCookies = AuthHelper.getCookies();

// Update session cookies after refresh
Map<String, String> refreshedCookies = // ... from auth refresh
AuthHelper.setCookies(refreshedCookies);
```

The helper automatically handles the authentication flow when cookies are requested but not available, streamlining session management across the application.
