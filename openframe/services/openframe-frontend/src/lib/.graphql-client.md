A GraphQL client configuration utility that handles schema introspection for URL state management in the OpenFrame frontend, with authentication support for both production and development environments.

## Key Components

- **GRAPHQL_ENDPOINT**: Dynamic GraphQL endpoint URL that adapts to current environment
- **initializeGraphQLIntrospection()**: Main function to fetch and cache GraphQL schema after authentication
- **clearGraphQLCache()**: Utility to clear cached schema data for debugging
- **isGraphQLSchemaLoaded()**: Status checker for schema availability

## Usage Example

```typescript
import { 
  initializeGraphQLIntrospection, 
  isGraphQLSchemaLoaded,
  clearGraphQLCache 
} from './graphql-client'

// Initialize after user authentication
useEffect(() => {
  if (isAuthenticated) {
    initializeGraphQLIntrospection()
  }
}, [isAuthenticated])

// Check if schema is ready
if (isGraphQLSchemaLoaded()) {
  // URL state hooks can now use schema introspection
  console.log('GraphQL schema ready for URL state management')
}

// Clear cache when needed (debugging)
clearGraphQLCache()

// Force fresh schema fetch
await initializeGraphQLIntrospection(true)
```

The client automatically handles DevTicket authentication mode by reading access tokens from localStorage and includes proper error handling to ensure URL state hooks continue working even if introspection fails.
