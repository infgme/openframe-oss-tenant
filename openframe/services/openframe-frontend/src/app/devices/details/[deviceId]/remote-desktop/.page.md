This React page component provides a complete remote desktop interface for connecting to and controlling remote devices through MeshCentral integration, featuring session management, display switching, and remote control capabilities.

## Key Components

- **RemoteDesktopPage**: Main component handling remote desktop sessions with device connectivity
- **MeshControlClient**: Manages authentication and control commands to MeshCentral server
- **MeshDesktop**: Handles desktop rendering and user input capture on HTML canvas
- **MeshTunnel**: Establishes WebSocket tunnel for real-time desktop streaming
- **RemoteSettingsModal**: Configuration panel for display quality and input settings
- **Device Management**: Fetches device details and handles legacy backward compatibility
- **Connection State Management**: Tracks tunnel states with automatic reconnection logic
- **Power Actions**: Remote power control (wake, sleep, reset, shutdown)
- **Keyboard Input**: Key combination sending and Ctrl+Alt+Del support
- **Multi-Display Support**: Display detection and switching capabilities

## Usage Example

```typescript
// Navigate to remote desktop for a device
router.push(`/remote-desktop/${deviceId}`)

// With legacy device data (backward compatibility)
const deviceData = { 
  id: "device123", 
  meshcentralAgentId: "agent456",
  hostname: "workstation-01" 
}
router.push(`/remote-desktop/${deviceId}?deviceData=${encodeURIComponent(JSON.stringify(deviceData))}`)

// Key combination handling
const sendWinM = () => sendKeyCombo([0x5B, 0x4D]) // Win+M
const sendCtrlW = () => sendKeyCombo([0x11, 0x57]) // Ctrl+W

// Power management
await sendPower('reset') // Reset device
await sendPower('sleep') // Put to sleep
```

The component provides a full-featured remote desktop experience with connection status monitoring, settings management, and comprehensive device control capabilities within the Flamingo platform.
