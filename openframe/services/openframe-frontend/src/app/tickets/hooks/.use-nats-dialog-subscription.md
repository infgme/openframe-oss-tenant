A React hook that manages NATS WebSocket subscriptions for dialog-based real-time messaging with authentication and environment-aware configuration.

## Key Components

- **`useNatsDialogSubscription`** - Main hook that wraps the core NATS subscription with client-specific authentication and URL building
- **`getApiBaseUrl()`** - Helper function to determine the API base URL from environment or window location
- **`getAccessToken()`** - Helper function to retrieve authentication token from localStorage
- **`ADMIN_TOPICS`** - Predefined message types for admin-level dialog communications
- **Environment Integration** - Automatic token refresh listening and dev ticket observer support

## Usage Example

```typescript
import { useNatsDialogSubscription } from './use-nats-dialog-subscription'

function ChatDialog({ dialogId }: { dialogId: string }) {
  const { isConnected, isSubscribed, error } = useNatsDialogSubscription({
    enabled: true,
    dialogId,
    onEvent: (payload, messageType) => {
      console.log('Received message:', payload, 'Type:', messageType)
      // Handle incoming messages
    },
    onConnect: () => {
      console.log('Connected to NATS')
    },
    onDisconnect: () => {
      console.log('Disconnected from NATS')
    }
  })

  if (error) return <div>Connection error: {error}</div>
  if (!isConnected) return <div>Connecting...</div>
  
  return <div>Chat ready ({isSubscribed ? 'subscribed' : 'subscribing'})</div>
}
```

The hook automatically handles authentication token changes, builds appropriate WebSocket URLs based on the environment, and subscribes to admin message topics for real-time dialog communication.
