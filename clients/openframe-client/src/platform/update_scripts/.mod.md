This module provides platform-specific update script functionality for Windows and macOS operating systems using conditional compilation.

## Key Components

### Platform-Specific Modules
- **`windows`** - Windows-specific update functionality (conditionally compiled)
- **`macos`** - macOS-specific update functionality (conditionally compiled)

### Public Exports
- **`UPDATE_SCRIPT_WINDOWS`** - Windows update script (Windows only)
- **`UPDATE_SCRIPT_MACOS`** - macOS update script (macOS only) 
- **`UPDATER_PLIST_TEMPLATE`** - macOS plist template for updater configuration (macOS only)

## Usage Example

This module automatically exposes the appropriate platform-specific constants based on the target operating system:

```rust
// On Windows, this will be available:
#[cfg(target_os = "windows")]
use crate::updater::UPDATE_SCRIPT_WINDOWS;

// On macOS, these will be available:
#[cfg(target_os = "macos")]
use crate::updater::{UPDATE_SCRIPT_MACOS, UPDATER_PLIST_TEMPLATE};

// Platform-agnostic usage
fn get_update_script() -> &'static str {
    #[cfg(target_os = "windows")]
    return UPDATE_SCRIPT_WINDOWS;
    
    #[cfg(target_os = "macos")]
    return UPDATE_SCRIPT_MACOS;
}
```

The conditional compilation ensures that only the relevant platform code is included in the final binary, reducing size and preventing compilation errors on unsupported platforms.
