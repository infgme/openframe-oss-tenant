API client for managing devices in the OpenFrame MSP platform. Provides methods to query, filter, and manage device inventory through GraphQL and REST endpoints.

## Key Components

- **getDeviceHostnames()** - Retrieves device hostnames using GraphQL filters
- **getDeviceIds()** - Fetches device machine IDs with filtering support
- **getDevice()** - Gets complete device details by machine ID
- **getDevices()** - Returns filtered list of devices
- **updateDeviceStatus()** - Updates device status via REST PATCH
- **archiveDevice() / deleteDevice()** - Convenience methods for status updates
- **searchDevice()** - Searches devices with text query and filters
- **getDeviceFilters()** - Retrieves available filter options

## Usage Example

```java
// Get all device hostnames for Windows machines
DeviceFilterInput filter = new DeviceFilterInput();
filter.setOperatingSystem("Windows");
List<String> hostnames = DeviceApi.getDeviceHostnames(filter);

// Retrieve full device details
Machine device = DeviceApi.getDevice("machine-123");

// Archive a device
DeviceApi.archiveDevice(device);

// Search for devices by name
Machine foundDevice = DeviceApi.searchDevice(filter, "server01");

// Get available filter options
DeviceFilters filters = DeviceApi.getDeviceFilters();
```

The API handles both GraphQL queries for complex data retrieval and REST operations for device state management, with automatic authorization headers and response validation.
