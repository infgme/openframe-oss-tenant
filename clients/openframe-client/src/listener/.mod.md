A module declaration file that exposes listener components for handling various system update events in the Flamingo platform.

## Key Components

The module provides three main listener components:

- **`ToolInstallationMessageListener`** - Handles tool installation status messages and events
- **`OpenFrameClientUpdateListener`** - Manages OpenFrame client update notifications and processing
- **`ToolAgentUpdateListener`** - Processes tool agent update events and state changes

## Usage Example

```rust
use crate::listeners::{
    ToolInstallationMessageListener,
    OpenFrameClientUpdateListener,
    ToolAgentUpdateListener
};

// Initialize listeners for different event types
let tool_listener = ToolInstallationMessageListener::new();
let client_listener = OpenFrameClientUpdateListener::new();
let agent_listener = ToolAgentUpdateListener::new();

// Start listening for events
tool_listener.start().await?;
client_listener.start().await?;
agent_listener.start().await?;
```

This module serves as the entry point for event-driven communication between the Flamingo platform components, enabling real-time updates for tool installations, client synchronization, and agent state management across the MSP infrastructure.
