Custom React hook for managing OpenFrame authentication tokens in localStorage with error handling and logging.

## Key Components

- **`useTokenStorage()`** - Main hook providing token management functions
- **`ACCESS_TOKEN_KEY`** - Constant for access token localStorage key (`of_access_token`)
- **`REFRESH_TOKEN_KEY`** - Constant for refresh token localStorage key (`of_refresh_token`)
- **Token operations** - Store, retrieve, and clear tokens with try-catch error handling
- **Header parsing** - Extract tokens from HTTP response headers

## Usage Example

```typescript
import { useTokenStorage } from './use-token-storage'

function LoginComponent() {
  const {
    storeAccessToken,
    storeRefreshToken,
    storeTokensFromHeaders,
    getAccessToken,
    getRefreshToken,
    clearTokens
  } = useTokenStorage()

  // Store tokens manually
  const handleLogin = async (credentials: LoginData) => {
    const response = await fetch('/api/login', {
      method: 'POST',
      body: JSON.stringify(credentials)
    })
    
    // Extract and store tokens from response headers
    const { accessToken, refreshToken } = storeTokensFromHeaders(response.headers)
  }

  // Get current tokens
  const currentAccessToken = getAccessToken()
  const currentRefreshToken = getRefreshToken()

  // Clear tokens on logout
  const handleLogout = () => {
    clearTokens()
  }
}
```

The hook uses the `of_` prefix for all localStorage keys to namespace OpenFrame tokens and includes comprehensive error handling with console logging for debugging authentication flows.
