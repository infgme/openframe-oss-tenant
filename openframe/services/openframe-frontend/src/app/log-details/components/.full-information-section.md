A React component that displays comprehensive log entry information in a structured, card-based layout with formatted key-value pairs and tool type badges.

## Key Components

- **FullInformationSection** - Main component that renders log details in an organized information card
- **LogEntry** - TypeScript interface defining the structure of log data including tool events, severity, timestamps, and metadata
- **FullInformationSectionProps** - Component props interface accepting optional log details
- **formatTimestamp** - Utility function that safely converts timestamp strings to ISO format

## Usage Example

```typescript
import { FullInformationSection } from './full-information-section'

const logEntry = {
  toolEventId: "evt_123456789",
  eventType: "error",
  ingestDay: "2024-01-15",
  toolType: "monitoring",
  severity: "high",
  userId: "user_abc123",
  deviceId: "device_xyz789",
  timestamp: "2024-01-15T10:30:00Z",
  message: "Connection timeout",
  details: "Network connectivity issue detected"
}

export default function LogDetailsPage() {
  return (
    <div className="p-6">
      <FullInformationSection logDetails={logEntry} />
    </div>
  )
}
```

The component uses OpenFrame's design system components (`InfoRow`, `ToolBadge`) and handles empty states gracefully. It displays essential log metadata including event IDs, severity levels, timestamps, and user/device associations in a visually organized card format with proper spacing and typography.
