A utility class providing helper methods for creating test objects, JSON serialization, and reflection-based field manipulation in OpenFrame API tests.

## Key Components

- **asJsonString()** - Converts objects to JSON strings using Jackson ObjectMapper
- **setPrivateField()** - Uses reflection to set private field values for testing
- **createTestClient()** - Creates mock OAuthClient instances with test configuration
- **createTestToken()** - Generates OAuthToken objects with realistic expiry times
- **createTestTokenResponse()** - Builds TokenResponse DTOs for OAuth flow testing
- **createTestAuthResponse()** - Creates AuthorizationResponse objects for authorization code flow

## Usage Example

```java
// Convert test data to JSON for API requests
String jsonPayload = TestUtils.asJsonString(myTestObject);

// Set private fields for unit testing
MyService service = new MyService();
TestUtils.setPrivateField(service, "apiKey", "test-key");

// Create OAuth test fixtures
OAuthClient testClient = TestUtils.createTestClient();
OAuthToken testToken = TestUtils.createTestToken();

// Generate response objects for mock endpoints
TokenResponse mockResponse = TestUtils.createTestTokenResponse();
AuthorizationResponse authResponse = TestUtils.createTestAuthResponse();
```

This class centralizes common test utilities, particularly for OAuth-related testing scenarios, reducing code duplication across test suites and providing consistent test data generation.
