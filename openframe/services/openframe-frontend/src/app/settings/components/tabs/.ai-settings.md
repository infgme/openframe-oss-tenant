This component provides a comprehensive AI configuration interface for managing AI providers, models, and policy templates within the OpenFrame platform.

## Key Components

- **AISettingsTab** - Main React component for AI configuration management
- **Provider Configuration** - Manages AI providers (Anthropic, OpenAI, Google Gemini) with API keys and model selection
- **Policy Management** - Handles AI policy templates including custom policy creation and rule modifications
- **Edit State Management** - Tracks configuration changes with snapshot functionality for save/cancel operations
- **Custom Policy System** - Enables creation of custom AI policies based on existing templates with override capabilities

## Usage Example

```typescript
import { AISettingsTab } from './ai-settings'

// Basic usage in a settings page
function SettingsPage() {
  return (
    <div className="settings-container">
      <h1>AI Configuration</h1>
      <AISettingsTab />
    </div>
  )
}

// The component handles:
// - AI provider selection (Anthropic, OpenAI, Google)
// - Model configuration for each provider
// - Policy template management
// - Custom policy creation with rule overrides
// - Edit mode with save/cancel functionality
```

The component integrates with multiple hooks (`useAIConfiguration`, `useAIPolicies`) and provides a complete interface for managing AI settings, from basic provider configuration to advanced policy customization. It supports creating custom policies based on existing templates, managing approval levels for different permission categories, and maintaining configuration state with proper error handling and loading states.
