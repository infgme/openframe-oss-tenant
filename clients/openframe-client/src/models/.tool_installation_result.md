A simple data structure representing the result of a tool installation operation, containing the identifier of the successfully installed tool agent.

## Key Components

- **ToolInstallationResult**: A serializable struct that holds the result data from a tool installation process
- **tool_agent_id**: String field containing the unique identifier of the installed tool agent

## Usage Example

```rust
use serde_json;

// Creating a tool installation result
let result = ToolInstallationResult {
    tool_agent_id: "agent_12345".to_string(),
};

// Serializing to JSON
let json = serde_json::to_string(&result)?;
// Output: {"tool_agent_id":"agent_12345"}

// Deserializing from JSON
let parsed: ToolInstallationResult = serde_json::from_str(&json)?;
println!("Installed tool agent: {}", parsed.tool_agent_id);
```

This struct is typically used as a return type for installation functions or API responses to communicate which tool agent was successfully installed in the system.
