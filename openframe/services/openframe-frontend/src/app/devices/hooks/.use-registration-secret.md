A React hook that manages fetching and state for agent registration secrets from the Flamingo platform API.

## Key Components

- **`useRegistrationSecret`** - Main hook that fetches the active registration secret key
- **State Management** - Tracks `initialKey`, `isLoading`, and `error` states
- **`fetchActiveSecret`** - Callback function that handles API requests with error handling
- **Auto-fetch** - Automatically loads the registration secret on component mount

## Usage Example

```typescript
import { useRegistrationSecret } from './use-registration-secret'

function RegistrationSecretManager() {
  const { initialKey, isLoading, error, refetch } = useRegistrationSecret()

  if (isLoading) {
    return <div>Loading registration secret...</div>
  }

  if (error) {
    return (
      <div>
        <p>Error: {error}</p>
        <button onClick={refetch}>Retry</button>
      </div>
    )
  }

  return (
    <div>
      <p>Registration Key: {initialKey}</p>
      <button onClick={refetch}>Refresh Secret</button>
    </div>
  )
}
```

The hook integrates with the OpenFrame toast system for error notifications and provides a clean API for components that need to display or manage agent registration secrets. The secret is automatically fetched when the component mounts and can be manually refreshed using the `refetch` function.
