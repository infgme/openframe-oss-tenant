A React component that displays detailed information about a specific log entry, including status, timestamp, device information, and full log details with navigation and copy functionality.

## Key Components

- **LogDetailsView**: Main component that renders the complete log details page
- **getSeverityVariant**: Helper function that maps log severity levels to UI variant types
- **Custom header content**: Includes back navigation button and copy log details action
- **Status display**: Shows severity tag and formatted timestamp
- **Log summary card**: Displays message, tool badge, and event type
- **Device info section**: Conditionally rendered device information
- **Full information and details sections**: Comprehensive log data display

## Usage Example

```typescript
import { LogDetailsView } from './log-details-view'

// Used in a page component or routing context
function LogPage({ params }: { params: { logId: string } }) {
  const { logId, ingestDay, toolType, eventType, timestamp } = params
  
  return (
    <LogDetailsView
      logId={logId}
      ingestDay={ingestDay}
      toolType={toolType}
      eventType={eventType}
      timestamp={timestamp}
    />
  )
}

// The component handles:
// - Loading states with DetailLoader
// - Error states with fallback UI
// - Navigation back to logs page
// - Copying log details to clipboard
// - Responsive layout for mobile and desktop
```

The component automatically fetches log details on mount and provides comprehensive error handling, loading states, and user interactions for viewing and copying log information.
