A React component that displays detailed information for a specific log entry, including device details, status information, and full log data with copy functionality.

## Key Components

- **LogDetailsView**: Main component that renders a detailed view of a log entry with status, timestamps, device information, and full log details
- **getSeverityVariant**: Helper function that maps log severity levels to appropriate status tag variants (success, warning, error, info, critical)
- **Custom Header**: Includes navigation back button and copy log details action
- **Responsive Layout**: Mobile-first design with collapsible sections and responsive spacing

## Usage Example

```typescript
import { LogDetailsView } from './log-details-view'

// Render log details page
function LogPage() {
  return (
    <LogDetailsView 
      logId="log-123"
      ingestDay="2024-01-15"
      toolType="monitoring"
      eventType="alert"
      timestamp="2024-01-15T10:30:00Z"
    />
  )
}

// The component will automatically:
// - Fetch log details using the provided parameters
// - Display loading state while fetching
// - Show error state if log not found
// - Render full log details with copy functionality
// - Include device information if available
```

The component provides comprehensive log viewing with automatic data fetching, error handling, and user-friendly actions like copying log details to clipboard. It uses the OpenFrame design system components and follows responsive design patterns for optimal viewing across devices.
