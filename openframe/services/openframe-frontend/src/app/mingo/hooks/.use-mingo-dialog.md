A React hook for managing Mingo AI dialog operations, providing functionality to create chat sessions and send messages with automatic error handling and state management.

## Key Components

- **useMingoDialog()** - Main hook that returns dialog management functions and state
- **createDialogMutation** - TanStack Query mutation for creating new admin dialogs
- **sendMessageMutation** - TanStack Query mutation for sending messages to dialogs
- **State Management** - Tracks current dialog ID using both state and ref for persistence
- **Error Handling** - Comprehensive error handling with toast notifications

## Usage Example

```typescript
import { useMingoDialog } from './hooks/use-mingo-dialog'

function ChatInterface() {
  const {
    currentDialogId,
    isCreatingDialog,
    isSendingMessage,
    createDialog,
    sendMessage,
    resetDialog,
    hasDialog
  } = useMingoDialog()

  const handleSendMessage = async (content: string) => {
    const success = await sendMessage(content)
    if (success) {
      console.log('Message sent successfully')
    }
  }

  const handleNewChat = async () => {
    const dialogId = await createDialog()
    if (dialogId) {
      console.log('New dialog created:', dialogId)
    }
  }

  return (
    <div>
      {!hasDialog && (
        <button onClick={handleNewChat} disabled={isCreatingDialog}>
          Start New Chat
        </button>
      )}
      
      <button 
        onClick={() => handleSendMessage("Hello Mingo!")}
        disabled={isSendingMessage}
      >
        Send Message
      </button>
      
      <button onClick={resetDialog}>Reset Dialog</button>
    </div>
  )
}
```

The hook automatically handles dialog creation when sending messages to non-existent dialogs, manages loading states, and provides toast notifications for errors. It integrates with TanStack Query for caching and state synchronization.
