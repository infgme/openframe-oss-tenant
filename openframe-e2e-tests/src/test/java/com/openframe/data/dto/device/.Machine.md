A data transfer object (DTO) representing a machine/device in the OpenFrame system, containing comprehensive device information including hardware specifications, network details, operating system information, and associated components.

## Key Components

- **Device Identification**: `id`, `machineId`, `hostname`, `displayName`, `serialNumber`
- **Network Information**: `ip`, `macAddress` 
- **Hardware Details**: `manufacturer`, `model`, `type` (DeviceType enum)
- **Operating System**: `osUuid`, `osType`, `osVersion`, `osBuild`, `timezone`
- **Status & Monitoring**: `status` (DeviceStatus enum), `lastSeen`, `agentVersion`
- **Organization**: `organizationId` for multi-tenant support
- **Associations**: `tags`, `toolConnections`, `installedAgents` collections
- **Timestamps**: `registeredAt`, `updatedAt` for audit tracking

## Usage Example

```java
Machine machine = Machine.builder()
    .machineId("DESK-001")
    .hostname("john-laptop")
    .displayName("John's Laptop")
    .ip("192.168.1.100")
    .macAddress("00:11:22:33:44:55")
    .manufacturer("Dell")
    .model("Latitude 7420")
    .osType("Windows")
    .osVersion("11")
    .status(DeviceStatus.ONLINE)
    .type(DeviceType.WORKSTATION)
    .organizationId("org-123")
    .agentVersion("2.1.0")
    .tags(List.of(new Tag("department", "IT")))
    .build();

// Access device information
String deviceInfo = machine.getDisplayName() + " (" + machine.getIp() + ")";
boolean isOnline = machine.getStatus() == DeviceStatus.ONLINE;
```
