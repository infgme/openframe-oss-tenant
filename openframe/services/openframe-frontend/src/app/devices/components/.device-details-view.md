A React component that renders a comprehensive device details page for the Flamingo MSP platform, providing device management capabilities including remote control, shell access, file management, and tabbed information display.

## Key Components

- **DeviceDetailsView** - Main component that orchestrates the entire device details interface
- **Device Action Buttons** - Remote control, shell access, and file management with platform-specific options
- **DeviceInfoSection** - Displays core device information and status
- **TabNavigation** - Provides tabbed interface for hardware, software, and other device details
- **Modal Components** - RemoteShellModal and ScriptsModal for interactive device operations
- **URL Parameter Handling** - Processes action parameters for deep linking and navigation

## Usage Example

```typescript
import { DeviceDetailsView } from './device-details-view'

// Render device details page
function DevicePage({ params }: { params: { id: string } }) {
  return <DeviceDetailsView deviceId={params.id} />
}

// With URL action parameters (from device table dropdown)
// /devices/details/device123?action=runScript
// /devices/details/device123?action=remoteShell&shellType=powershell
```

The component automatically handles device data fetching, platform-specific shell options (CMD/PowerShell for Windows, bash for others), and provides appropriate action buttons based on device capabilities. It includes responsive design with modal overlays and maintains URL synchronization for tab navigation and deep linking.
