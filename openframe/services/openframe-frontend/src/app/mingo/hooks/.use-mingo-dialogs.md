Custom React hook that fetches and manages paginated Mingo dialogs with real-time unread message counts using infinite scrolling and background message synchronization.

## Key Components

- **`transformToDialogItem()`** - Transforms raw dialog nodes into standardized dialog items with unread counts
- **`useMingoDialogs()`** - Main hook that provides infinite query functionality for dialogs with search and pagination support
- **Background message integration** - Syncs with `useMingoBackgroundMessagesStore` for real-time unread counts
- **GraphQL integration** - Uses TanStack Query with GraphQL endpoint for data fetching

## Usage Example

```typescript
import { useMingoDialogs } from './hooks/use-mingo-dialogs'

function DialogsList() {
  const { 
    dialogs, 
    isLoading, 
    hasNextPage, 
    fetchNextPage,
    refetch 
  } = useMingoDialogs({
    search: 'customer support',
    limit: 50,
    enabled: true
  })

  if (isLoading) return <div>Loading dialogs...</div>

  return (
    <div>
      {dialogs.map(dialog => (
        <div key={dialog.id}>
          <h3>{dialog.title}</h3>
          <span>{dialog.unreadMessagesCount} unread</span>
          <time>{dialog.timestamp.toLocaleDateString()}</time>
        </div>
      ))}
      {hasNextPage && (
        <button onClick={() => fetchNextPage()}>
          Load More
        </button>
      )}
    </div>
  )
}
```
