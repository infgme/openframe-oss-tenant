Configuration constants class that centralizes user-related configuration values for the OpenFrame application, including default passwords and file paths.

## Key Components

- **CORRECT_PASSWORD**: Static constant defining the default/expected user password (`"Password123!"`)
- **NEW_PASSWORD**: Static constant for updated password value (`"Password124!"`)
- **USER_FILE**: Static constant specifying the user data file path (`"user.json"`)
