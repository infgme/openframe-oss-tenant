A serializable message structure for communicating tool agent update information, including version details and download configurations for the update process.

## Key Components

- **ToolAgentUpdateMessage** - Main struct containing update metadata
  - `tool_agent_id: String` - Unique identifier for the tool agent
  - `version: String` - Target version for the update
  - `download_configurations: Vec<DownloadConfiguration>` - List of download configurations needed for the update

## Usage Example

```rust
use tool_agent_update_message::ToolAgentUpdateMessage;
use download_configuration::DownloadConfiguration;

// Create an update message
let update_message = ToolAgentUpdateMessage {
    tool_agent_id: "monitoring-agent-001".to_string(),
    version: "2.1.4".to_string(),
    download_configurations: vec![
        DownloadConfiguration {
            url: "https://releases.example.com/agent-2.1.4.tar.gz".to_string(),
            checksum: "sha256:abc123...".to_string(),
        }
    ],
};

// Serialize to JSON for transmission
let json = serde_json::to_string(&update_message)?;

// Deserialize from JSON
let parsed: ToolAgentUpdateMessage = serde_json::from_str(&json)?;
```

The struct uses camelCase serialization for JSON compatibility and supports both serialization and deserialization through serde, making it suitable for API communication and configuration management in distributed tool agent systems.
