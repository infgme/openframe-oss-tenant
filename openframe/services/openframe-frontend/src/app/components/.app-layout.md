A React component that provides the main application layout wrapper with authentication-aware routing and navigation for the OpenFrame platform.

## Key Components

- **`AppLayout`** - Main export that wraps content in Suspense with skeleton fallback
- **`AppLayoutInner`** - Core layout logic handling authentication state and routing
- **`AppShell`** - Authenticated layout with sidebar navigation and header
- **`ContentLoading`** - Loading component for page transitions

## Authentication Flow

The component handles different tenant modes and authentication states:

- **OSS Tenant Mode**: Redirects unauthenticated users to `/auth`
- **SaaS Tenant Mode**: Checks authentication via `/me` endpoint with fallback overlay
- **Auth-Only Mode**: Renders children directly without layout chrome

## Usage Example

```typescript
import { AppLayout } from './app-layout'

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <AppLayout mainClassName="custom-main-class">
      {children}
    </AppLayout>
  )
}

// For pages that need custom main styling
function CustomPage() {
  return (
    <AppLayout mainClassName="p-0 overflow-hidden">
      <div>Custom page content</div>
    </AppLayout>
  )
}
```

The component integrates with the auth store, navigation config, and core OpenFrame components to provide a unified application shell with proper loading states and error handling.
