A modal dialog component that confirms resending user invitations, displaying the recipient email and invalidation warning before proceeding.

## Key Components

- **ConfirmResendInvitationModal** - Main modal component using AlertDialog primitives
- **ConfirmResendInvitationModalProps** - TypeScript interface defining component props
- **AlertDialog components** - Imported UI primitives for modal structure and styling

## Usage Example

```typescript
import { ConfirmResendInvitationModal } from './confirm-resend-invitation-modal'
import { useState } from 'react'

function UserManagement() {
  const [isResendModalOpen, setIsResendModalOpen] = useState(false)
  
  const handleResendInvitation = async () => {
    try {
      await resendUserInvitation('user@example.com')
      setIsResendModalOpen(false)
      // Show success notification
    } catch (error) {
      console.error('Failed to resend invitation:', error)
    }
  }

  return (
    <>
      <button onClick={() => setIsResendModalOpen(true)}>
        Resend Invitation
      </button>
      
      <ConfirmResendInvitationModal
        open={isResendModalOpen}
        onOpenChange={setIsResendModalOpen}
        userEmail="user@example.com"
        onConfirm={handleResendInvitation}
      />
    </>
  )
}
```

The component features custom styling with the OpenFrame design system, highlighting the user email in warning color and providing clear cancel/confirm actions. The modal explains that resending will invalidate the previous invitation.
