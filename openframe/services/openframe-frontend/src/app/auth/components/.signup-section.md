A React component that provides a complete user registration form interface for creating new organizations in the OpenFrame platform, supporting both email/password signup and SSO authentication methods.

## Key Components

- **AuthSignupSection** - Main registration form component with personal details, organization info, and authentication options
- **RegisterRequest** - TypeScript interface defining the structure of registration data
- **AuthSignupSectionProps** - Component props interface including callbacks and configuration
- **Form validation** - Email format validation, password confirmation, and required field checking
- **SSO integration** - Conditional rendering of social authentication providers
- **Responsive layout** - Mobile-first design with flexible form layouts

## Usage Example

```typescript
import { AuthSignupSection } from './signup-section'

function RegistrationPage() {
  const handleSubmit = (data: RegisterRequest) => {
    console.log('Registration data:', data)
    // Process registration
  }

  const handleSSO = (provider: string) => {
    console.log('SSO provider selected:', provider)
    // Handle SSO authentication
  }

  return (
    <AuthSignupSection
      orgName="My Organization"
      domain="my-org.com"
      accessCode="invite-123"
      email="user@example.com"
      onSubmit={handleSubmit}
      onSSO={handleSSO}
      onBack={() => console.log('Go back')}
      isLoading={false}
    />
  )
}
```

The component handles form state management, input validation, keyboard navigation (Enter key submission), and conditional rendering based on app deployment mode (SaaS shared vs. self-hosted).
