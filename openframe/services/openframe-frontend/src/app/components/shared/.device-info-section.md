A reusable React component that displays device information in a standardized card format, supporting both fetched device data and pre-loaded device objects from external sources.

## Key Components

- **DeviceInfoSection**: Main component that renders device information using the DeviceCard UI component
- **useDeviceDetails**: Hook for fetching device data when not provided via props
- **DeviceCard**: UI component from the core library for consistent device display
- **StatusTag**: Component for displaying device status with appropriate styling
- **DeviceDetailsButton**: Action component for navigating to detailed device view

## Usage Example

```typescript
import { DeviceInfoSection } from './device-info-section'

// With device ID (will fetch data)
function LogEntry() {
  return (
    <DeviceInfoSection 
      deviceId="device-123" 
      userId="user-456" 
    />
  )
}

// With pre-loaded device data (no fetch needed)
function DialogContent() {
  const device = {
    id: 'device-123',
    displayName: 'John\'s Laptop',
    status: 'online',
    osType: 'windows'
  }
  
  return (
    <DeviceInfoSection 
      device={device}
      deviceId="device-123"
    />
  )
}
```

The component intelligently handles both scenarios: fetching device data when only an ID is provided, or using pre-loaded device data to avoid unnecessary API calls. It includes loading states, error handling, and consistent styling with the Flamingo design system.
