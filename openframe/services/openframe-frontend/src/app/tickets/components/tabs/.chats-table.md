A React component that renders a paginated table of chat dialogs with filtering, search, and archive functionality for both current and archived chats.

## Key Components

- **`ChatsTable`** - Main table component that displays dialogs with cursor pagination and filtering
- **`CurrentChats`** - Wrapper component for displaying active/current chat dialogs  
- **`ArchivedChats`** - Wrapper component for displaying archived chat dialogs

### Props
- `isArchived: boolean` - Determines whether to show current or archived chats

### Core Features
- Cursor-based pagination with next/reset navigation
- Real-time search functionality with debounced input
- Status-based filtering (current chats only)
- Bulk archive action for resolved dialogs
- Organization name lookup with lazy loading
- Row click navigation to individual chat details

## Usage Example

```typescript
import { CurrentChats, ArchivedChats, ChatsTable } from './chats-table'

// Use wrapper components for tab navigation
function ChatTabs() {
  const [activeTab, setActiveTab] = useState('current')
  
  return (
    <div>
      <TabNavigation>
        <Tab onClick={() => setActiveTab('current')}>Current</Tab>
        <Tab onClick={() => setActiveTab('archived')}>Archived</Tab>
      </TabNavigation>
      
      {activeTab === 'current' ? <CurrentChats /> : <ArchivedChats />}
    </div>
  )
}

// Or use directly with custom props
function CustomChatTable() {
  return <ChatsTable isArchived={false} />
}
```

The component integrates with the dialogs store for state management and provides a responsive table interface with mobile-friendly filtering options.
