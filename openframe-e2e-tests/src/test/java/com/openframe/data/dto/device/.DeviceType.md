This enum defines the standard device types supported by the OpenFrame platform for device categorization and management across MSP operations.

## Key Components

- **DESKTOP** - Traditional desktop computers
- **LAPTOP** - Portable laptop computers  
- **SERVER** - Physical server hardware
- **MOBILE_DEVICE** - Smartphones and mobile devices
- **TABLET** - Tablet computers and similar devices
- **NETWORK_DEVICE** - Routers, switches, firewalls, and network equipment
- **IOT_DEVICE** - Internet of Things devices and sensors
- **VIRTUAL_MACHINE** - Virtualized computer instances
- **CONTAINER_HOST** - Systems running containerized applications
- **OTHER** - Fallback category for uncategorized devices

## Usage Example

```java
// Categorizing devices in inventory management
DeviceType serverType = DeviceType.SERVER;
DeviceType laptopType = DeviceType.LAPTOP;

// Using in device creation or filtering
public class Device {
    private DeviceType type;
    private String name;
    
    public Device(String name, DeviceType type) {
        this.name = name;
        this.type = type;
    }
    
    public boolean isNetworkEquipment() {
        return type == DeviceType.NETWORK_DEVICE;
    }
    
    public boolean requiresRegularUpdates() {
        return type == DeviceType.DESKTOP || 
               type == DeviceType.LAPTOP || 
               type == DeviceType.SERVER;
    }
}
```

This enum is commonly used throughout the OpenFrame platform for device inventory management, monitoring configurations, and automated IT support workflows.
