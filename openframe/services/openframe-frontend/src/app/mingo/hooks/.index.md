A collection of React hooks that provide dialog management, message processing, and AI interaction capabilities for the Mingo AI assistant system.

## Key Components

- **`useMingoDialog`** - Hook for managing individual Mingo AI dialog sessions
- **`useMingoDialogs`** - Hook for managing multiple concurrent dialog sessions
- **`useMingoDialogSelection`** - Hook for handling dialog selection and switching between conversations
- **`useProcessedMessages`** - Hook for processing and formatting AI messages with streaming support
- **`useChunkCatchup`** - Hook for handling chunked message delivery and catch-up mechanisms

## Usage Example

```typescript
import { 
  useMingoDialog, 
  useMingoDialogs, 
  useProcessedMessages 
} from './hooks'

function ChatInterface() {
  // Manage multiple dialog sessions
  const { dialogs, createDialog, removeDialog } = useMingoDialogs()
  
  // Get current active dialog
  const { messages, sendMessage, isLoading } = useMingoDialog(activeDialogId)
  
  // Process messages with streaming

      ))}
    </div>
  )
}
```

These hooks work together to provide a complete AI chat interface with support for multiple conversations, real-time message streaming, and intelligent message processing for the Flamingo platform's Mingo AI assistant.
