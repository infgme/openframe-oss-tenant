Main dashboard layout component that orchestrates all dashboard sections with conditional rendering based on app mode.

## Key Components

- **DashboardContent** - Default exported React component that renders the complete dashboard layout
- **OnboardingSection** - User onboarding and setup guidance section
- **DevicesOverviewSection** - Device management and monitoring overview
- **ChatsOverviewSection** - Chat/communication overview (SaaS mode only)
- **OrganizationsOverviewSection** - Organization management section
- **isSaasTenantMode** - Utility function to determine if app is running in SaaS tenant mode

## Usage Example

```typescript
import { lazy, Suspense } from 'react'

// Lazy load dashboard content with loading skeleton
const DashboardContent = lazy(() => import('./dashboard-content'))

function DashboardPage() {
  return (
    <Suspense fallback={<DashboardSkeleton />}>
      <DashboardContent />
    </Suspense>
  )
}
```

The component is designed for dynamic imports to enable code splitting and loading states. It conditionally shows the chats section only in SaaS tenant mode, while other sections are always displayed. All sections are arranged in a vertical layout with consistent spacing.
