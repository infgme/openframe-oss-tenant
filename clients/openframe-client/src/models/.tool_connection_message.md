A simple data structure representing a message used to establish or reference connections between tools and agents in the Flamingo platform.

## Key Components

- **`ToolConnectionMessage`** - Main struct that encapsulates tool connection information
- **`tool_type`** - String identifier specifying the type of tool being connected
- **`agent_tool_id`** - Unique identifier linking the tool to a specific agent

## Usage Example

```rust
use serde_json;

// Create a new tool connection message
let connection_msg = ToolConnectionMessage {
    tool_type: "monitoring".to_string(),
    agent_tool_id: "agent_123_monitoring_tool".to_string(),
};

// Serialize to JSON (camelCase format)
let json = serde_json::to_string(&connection_msg)?;
// Output: {"toolType":"monitoring","agentToolId":"agent_123_monitoring_tool"}

// Deserialize from JSON
let deserialized: ToolConnectionMessage = serde_json::from_str(&json)?;
```

The struct uses `#[serde(rename_all = "camelCase")]` to ensure JSON serialization follows camelCase naming conventions, making it compatible with JavaScript frontends and API standards.
