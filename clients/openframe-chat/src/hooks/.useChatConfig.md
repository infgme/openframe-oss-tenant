Custom React hook that manages chat configuration settings, specifically loading and updating quick action buttons for chat interfaces.

## Key Components

- **`QuickAction` interface**: Defines structure for chat quick actions with `id` and `text` fields
- **`useChatConfig()` hook**: Main hook that manages quick actions state and provides update functionality
- **`quickActions` state**: Array of available quick action items loaded from JSON config
- **`updateQuickActions()` function**: Method to update the quick actions list

## Usage Example

```typescript
import { useChatConfig } from './useChatConfig'

function ChatInterface() {
  const { quickActions, updateQuickActions } = useChatConfig()
  
  const handleAddAction = () => {
    const newAction = { id: '4', text: 'New Quick Reply' }
    updateQuickActions([...quickActions, newAction])
  }
  
  return (
    <div>
      <div className="quick-actions">
        {quickActions.map(action => (
          <button key={action.id} onClick={() => sendMessage(action.text)}>
            {action.text}
          </button>
        ))}
      </div>
      <button onClick={handleAddAction}>Add Quick Action</button>
    </div>
  )
}
```

The hook automatically loads quick actions from a JSON configuration file on mount and provides a clean interface for managing chat shortcuts that can be easily extended to support persistence via local storage or API calls.
