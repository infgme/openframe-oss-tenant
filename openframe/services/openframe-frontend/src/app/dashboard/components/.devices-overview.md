A React component that displays a dashboard overview section for device statistics, showing total device count and breakdown of online/offline devices with interactive cards.

## Key Components

- **DevicesOverviewSection**: Main functional component that renders the devices overview dashboard section
- **useDevicesOverview**: Custom hook that provides device statistics and percentages
- **DashboardInfoCard**: Reusable card component from the core library for displaying metrics
- **Default export**: Also exports the component as default for flexible importing

## Usage Example

```typescript
import { DevicesOverviewSection } from './devices-overview'
// or
import DevicesOverviewSection from './devices-overview'

function Dashboard() {
  return (
    <div className="dashboard">
      <DevicesOverviewSection />
    </div>
  )
}
```

The component automatically fetches device data through the `useDevicesOverview` hook and renders:
- A section header with total device count
- Grid layout of info cards showing online/offline device statistics
- Progress indicators and clickable links to filtered device views
- Responsive design that adapts from 1 column on mobile to 3 columns on large screens
