A modal component for editing user profile information with form fields for name updates and read-only display of email and role.

## Key Components

- **EditProfileModal** - Main modal component with form inputs for first name and last name
- **EditProfileModalProps** - TypeScript interface defining component props including user data, save handler, and modal state
- **Form State Management** - Local state for firstName and lastName with automatic sync when modal opens
- **Save/Cancel Handlers** - Callback functions for form submission and modal dismissal

## Usage Example

```typescript
import { EditProfileModal } from './edit-profile-modal'
import { useState } from 'react'

function ProfilePage() {
  const [isModalOpen, setIsModalOpen] = useState(false)
  const [isSaving, setIsSaving] = useState(false)
  
  const user = {
    firstName: 'John',
    lastName: 'Doe', 
    email: 'john@example.com',
    roles: ['Admin']
  }

  const handleSave = async (data: { firstName: string; lastName: string }) => {
    setIsSaving(true)
    try {
      // Update user profile via API
      await updateUserProfile(data)
    } finally {
      setIsSaving(false)
    }
  }

  return (
    <EditProfileModal
      isOpen={isModalOpen}
      onClose={() => setIsModalOpen(false)}
      user={user}
      onSave={handleSave}
      isSaving={isSaving}
    />
  )
}
```

The component provides a clean two-column layout with editable name fields and read-only email/role display, featuring proper form validation states and loading indicators.
