A React component that provides a tabbed interface for managing policies and queries within the OpenFrame platform, utilizing the core tab navigation system with URL synchronization.

## Key Components

- **PoliciesAndQueriesView** - Main component that renders a tabbed interface for policies and queries management
- **TabNavigation** - Core navigation component from `@flamingo-stack/openframe-frontend-core`
- **TabContent** - Content container that dynamically renders the active tab component
- **POLICIES_AND_QUERIES_TABS** - Configuration object defining available tabs (imported from local tabs module)

## Usage Example

```typescript
import { PoliciesAndQueriesView } from './policies-and-queries-view'

// Basic implementation in a page component
export default function PoliciesPage() {
  return (
    <div className="container mx-auto p-4">
      <h1 className="text-2xl font-bold mb-4">IT Policies & Queries</h1>
      <PoliciesAndQueriesView />
    </div>
  )
}

// The component automatically handles:
// - Tab state management
// - URL synchronization (e.g., /policies?tab=queries)
// - Dynamic tab content rendering
// - Default "policies" tab selection
```

The component leverages the OpenFrame core tab system to create a seamless user experience with browser history integration and dynamic content loading based on the active tab selection.
