A React component that renders a paginated, searchable, and filterable table of scripts with URL state management for a Flamingo/OpenFrame MSP platform.

## Key Components

- **ScriptsTable**: Main component providing complete script management interface
- **UIScriptEntry**: TypeScript interface defining script data structure
- **URL State Management**: Search, filters, and pagination persist in browser URL
- **Real-time Search**: Debounced search input with 300ms delay
- **Advanced Filtering**: Multi-select filters for shell type, source tool, and category
- **Table Actions**: Row-level actions for edit, view details, and script execution

## Usage Example

```typescript
import { ScriptsTable } from './scripts-table'

// Basic usage - component handles all state internally
export function ScriptsPage() {
  return (
    <div>
      <ScriptsTable />
    </div>
  )
}

// The component automatically provides:
// - Search functionality with URL persistence
// - Filterable columns (shellType, addedBy, category)
// - Pagination with client-side rendering
// - Row actions (Edit, Details, Run Script)
// - Responsive design with column hiding
```

The component integrates with the OpenFrame UI kit and handles script data transformation from backend APIs, providing a complete script management interface with persistent URL state for bookmarking and sharing filtered views.
