Vite configuration file for the Flamingo/OpenFrame platform that sets up React development environment with environment variable handling and build optimization.

## Key Components

- **React Plugin**: Enables React JSX transformation and Fast Refresh
- **Environment Variables**: Exposes `NODE_ENV` and `NEXT_PUBLIC_APP_TYPE` to the client
- **Build Configuration**: Optimized output settings with assets bundling
- **Development Server**: Local server configuration on port 3003
- **Environment Prefixes**: Allows access to VITE_, TAURI_, and NEXT_PUBLIC_ variables

## Usage Example

```typescript
// The config automatically injects these environment variables
const appType = process.env.NEXT_PUBLIC_APP_TYPE // 'flamingo' by default
const nodeEnv = process.env.NODE_ENV // 'development' or 'production'

// Start development server
// npm run dev (starts on http://127.0.0.1:3003)

// Build for production
// npm run build (outputs to ./dist with assets in ./dist/assets)
```

The configuration supports both Flamingo MSP platform and OpenFrame unified interface builds through the `NEXT_PUBLIC_APP_TYPE` environment variable, with optimized settings for Tauri desktop app integration.
