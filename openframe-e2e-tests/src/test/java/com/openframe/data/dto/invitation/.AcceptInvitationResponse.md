A DTO (Data Transfer Object) that represents the response returned after successfully accepting an invitation, containing complete user profile information including authentication details and metadata.

## Key Components

- **User Identity Fields**: `id`, `email`, `firstName`, `lastName`, `fullName`
- **Authorization Data**: `roles` (List of UserRole), `status` (UserStatus), `tenantId`
- **Authentication Details**: `passwordHash`, `emailVerified`, `loginProvider`, `externalUserId`
- **Timestamps**: `createdAt`, `updatedAt`, `lastLogin`
- **Lombok Annotations**: `@Data`, `@Builder`, `@NoArgsConstructor`, `@AllArgsConstructor` for automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Building a response after successful invitation acceptance
AcceptInvitationResponse response = AcceptInvitationResponse.builder()
    .id("user-123")
    .email("john.doe@company.com")
    .firstName("John")
    .lastName("Doe")
    .fullName("John Doe")
    .roles(Arrays.asList(UserRole.TECHNICIAN))
    .status(UserStatus.ACTIVE)
    .tenantId("tenant-456")
    .emailVerified(true)
    .createdAt(LocalDateTime.now())
    .updatedAt(LocalDateTime.now())
    .lastLogin(Instant.now())
    .build();

// Returning from REST endpoint
@PostMapping("/invitations/{id}/accept")
public ResponseEntity<AcceptInvitationResponse> acceptInvitation(@PathVariable String id) {
    AcceptInvitationResponse response = invitationService.acceptInvitation(id);
    return ResponseEntity.ok(response);
}
```
