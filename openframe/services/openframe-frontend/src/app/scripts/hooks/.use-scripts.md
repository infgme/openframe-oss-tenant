A React hook for managing scripts data fetching and filtering with optimized loading states and smart refetching logic.

## Key Components

- **`useScripts`** - Main hook that provides scripts state management and API operations
- **`ScriptsFilterInput`** - Interface for filtering scripts by shell type, author, and category
- **State Management** - Integration with `useScriptsStore` for centralized scripts state
- **Smart Loading** - Prevents flash of empty state with immediate skeleton display
- **Filter Tracking** - Optimized refetching that only occurs when filters actually change

## Usage Example

```typescript
import { useScripts } from './hooks/use-scripts'

function ScriptsView() {
  const filters = {
    shellType: ['powershell', 'bash'],
    category: ['automation'],
    addedBy: ['user123']
  }
  
  const {
    scripts,
    search,
    isLoading,
    error,
    fetchScripts,
    searchScripts,
    refreshScripts,
    markInitialLoadDone
  } = useScripts(filters)

  useEffect(() => {
    const loadInitialData = async () => {
      await fetchScripts('', filters)
      markInitialLoadDone() // Prevents unnecessary refetches
    }
    loadInitialData()
  }, [])

  return (
    <div>
      <SearchInput onSearch={searchScripts} />
      <RefreshButton onClick={refreshScripts} />
      {isLoading ? <Skeleton /> : <ScriptsList scripts={scripts} />}
    </div>
  )
}
```

The hook provides intelligent caching and prevents infinite refetch loops while maintaining responsive UI states and proper error handling with toast notifications.
