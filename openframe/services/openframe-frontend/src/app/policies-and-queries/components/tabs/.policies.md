A React component that renders the main policies listing page with search, filtering, and table functionality for managing IT security policies.

## Key Components

- **Policies Component**: Main page component that displays a searchable, filterable table of policies
- **State Management**: Handles search terms, table filters, and debounced search functionality
- **Navigation Actions**: Provides routing to policy details and creation pages
- **Table Integration**: Uses reusable table columns and row actions from separate modules
- **Custom Hooks**: Leverages `usePolicies` for data management and `useDebounce` for search optimization

## Usage Example

```typescript
import { Policies } from './policies'

// Main policies page route component
export default function PoliciesPage() {
  return <Policies />
}

// The component handles:
// - Policy listing with search and filters
// - Navigation to individual policy details
// - Creating new policies via "Add Policy" button
// - Debounced search (300ms delay)
// - Automatic data fetching and refresh
```

The component follows OpenFrame's ListPageLayout pattern with integrated Table functionality, providing a consistent user experience for managing policy data with built-in loading states, error handling, and responsive design.
