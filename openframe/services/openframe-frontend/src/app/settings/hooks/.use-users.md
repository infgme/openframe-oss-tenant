A custom React hook for managing user data operations including fetching, pagination, and deletion with React Query integration and error handling.

## Key Components

- **`UserRecord`** - Type definition for user entities with properties like id, email, name, roles, and status
- **`UserStatus`** - Enum defining user states (ACTIVE, DELETED)
- **`PagedUsersResponse`** - Type for paginated API responses with metadata
- **`useUsers`** - Main hook providing user data management functionality
- **`usersQueryKeys`** - Query key factory for React Query cache management
- **`fetchUsers`** / **`deleteUserApi`** - API functions for user operations

## Usage Example

```typescript
import { useUsers, UserStatus } from './hooks/use-users'

function UsersPage() {
  const {
    users,
    isLoading,
    error,
    page,
    totalPages,
    hasNext,
    hasPrevious,
    deleteUser,
    refetch
  } = useUsers(0, 10) // page 0, size 10

  const handleDeleteUser = (userId: string) => {
    deleteUser(userId, {
      onSuccess: () => {
        console.log('User deleted successfully')
      },
      onError: (error) => {
        console.error('Delete failed:', error)
      }
    })
  }

  if (isLoading) return <div>Loading...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      {users.map(user => (
        <div key={user.id}>
          {user.email} - {user.status}
          <button onClick={() => handleDeleteUser(user.id)}>
            Delete
          </button>
        </div>
      ))}
      <div>Page {page + 1} of {totalPages}</div>
    </div>
  )
}
```
