A React modal component for creating and editing API keys with form validation and toast notifications.

## Key Components

- **CreateApiKeyModal** - Main modal component that handles both create and edit modes
- **CreateApiKeyModalProps** - TypeScript interface defining component props including callbacks for CRUD operations
- **Form Fields** - Name (required), description (optional), and expiration date (optional) inputs
- **State Management** - Local state for form data, submission status, and validation
- **Toast Integration** - Success/error notifications using the OpenFrame toast system

## Usage Example

```typescript
import { CreateApiKeyModal } from './create-api-key-modal'

function ApiKeyManagement() {
  const [showModal, setShowModal] = useState(false)
  
  const handleCreate = async (data: { name: string; description?: string; expiresAt?: string | null }) => {
    // API call to create key
    const response = await fetch('/api/keys', { method: 'POST', body: JSON.stringify(data) })
    return response.json()
  }
  
  const handleCreated = async ({ apiKeyId, fullKey }: { apiKeyId: string; fullKey: string }) => {
    // Handle successful creation (e.g., show the full key to user)
    console.log('Created API key:', apiKeyId, fullKey)
  }
  
  return (
    <CreateApiKeyModal
      isOpen={showModal}
      onClose={() => setShowModal(false)}
      create={handleCreate}
      onCreated={handleCreated}
      mode="create"
    />
  )
}
```

The component supports both create and edit modes, handles form validation (name is required), and provides proper error handling with toast notifications. It automatically resets form state when closed and can format expiration dates for datetime-local inputs.
