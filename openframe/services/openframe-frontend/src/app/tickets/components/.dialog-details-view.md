A React component for displaying detailed dialog views with real-time chat functionality, status management, and approval workflows in the Flamingo platform.

## Key Components

- **DialogDetailsView**: Main component handling dialog visualization and interactions
- **Real-time Processing**: NATS subscription for live message updates with chunk-based processing
- **Status Management**: Dialog status controls (put on hold, resolve) with optimistic updates
- **Chat Interface**: Dual chat system for client and admin communications with typing indicators
- **Approval System**: Interactive approval request handling with status tracking
- **Message Processing**: Historical message processing with error handling and segmentation

## Usage Example

```typescript
import { DialogDetailsView } from './dialog-details-view'

// Basic usage in a dialog detail page
function DialogPage({ params }: { params: { id: string } }) {
  return (
    <div className="container mx-auto">
      <DialogDetailsView dialogId={params.id} />
    </div>
  )
}

// With error boundary and loading states
function ProtectedDialogView({ dialogId }: { dialogId: string }) {
  return (
    <ErrorBoundary>
      <Suspense fallback={<DetailLoader />}>
        <DialogDetailsView dialogId={dialogId} />
      </Suspense>
    </ErrorBoundary>
  )
}
```

The component automatically handles data fetching, real-time subscriptions, message processing, and provides interactive controls for dialog management. It integrates with the Flamingo dialog system to provide a comprehensive interface for support ticket management with AI assistance from both Fae (client-facing) and Mingo (admin-facing) assistants.
