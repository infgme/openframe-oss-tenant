A data structure representing the response from agent authentication, containing access and refresh tokens with expiration information.

## Key Components

- **AgentTokenResponse** - Main struct containing authentication token data
  - `access_token` - Bearer token for API authorization
  - `refresh_token` - Token used to obtain new access tokens
  - `token_type` - Type of token (typically "Bearer")
  - `expires_in` - Optional token lifetime in seconds

## Usage Example

```rust
use serde_json;

// Deserialize from JSON response
let json_response = r#"
{
    "accessToken": "eyJhbGciOiJIUzI1NiIs...",
    "refreshToken": "def502004a8b7c...",
    "tokenType": "Bearer",
    "expiresIn": 3600
}
"#;

let token_response: AgentTokenResponse = serde_json::from_str(json_response)?;

// Use the tokens
println!("Access token expires in: {:?} seconds", token_response.expires_in);
println!("Token type: {}", token_response.token_type);

// Serialize back to JSON
let serialized = serde_json::to_string(&token_response)?;
```

The struct uses camelCase serialization to match standard OAuth2 JSON response formats, making it compatible with external authentication services.
