A React hook that processes admin chat messages for the Mingo AI assistant, handling message formatting, error processing, and approval request management with real-time status updates.

## Key Components

- **useProcessedMessages** - Main hook that transforms raw admin messages into processed chat format
- **handleApprove/handleReject** - Callback functions for managing approval request workflows
- **processedData** - Memoized computation that filters and formats messages
- **approvalStatuses** - State management for tracking approval request statuses
- **pendingApprovals** - Extracted pending approval segments for separate UI handling

## Usage Example

```typescript
import { useProcessedMessages } from './use-processed-messages'

function MingoChat() {
  const { 
    messages, 
    pendingApprovals, 
    assistantType,
    handleApprove, 
    handleReject 
  } = useProcessedMessages()

  return (
    <div>
      {messages.map(message => (
        <ChatMessage 
          key={message.id}
          content={message.content}
          role={message.role}
          timestamp={message.timestamp}
        />
      ))}
      
      {pendingApprovals.map(approval => (
        <ApprovalRequest
          key={approval.requestId}
          onApprove={() => handleApprove(approval.requestId)}
          onReject={() => handleReject(approval.requestId)}
        />
      ))}
    </div>
  )
}
```

The hook integrates with the Mingo dialog store, processes historical messages with error handling, and provides a clean interface for managing chat interactions and approval workflows in the OpenFrame platform.
