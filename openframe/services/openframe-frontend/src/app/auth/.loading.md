A loading skeleton component that precisely mirrors the authentication page layout with matching dimensions and spacing for seamless user experience during auth page transitions.

## Key Components

- **AuthLoading**: Default export component that renders a full-screen loading skeleton
- **Layout Structure**: Two-column responsive layout (50% left auth forms, 50% right benefits)
- **Form Skeletons**: Create organization and login form placeholders with exact field dimensions
- **Benefits Section**: Right-side skeleton matching logo and benefit cards layout

## Usage Example

```typescript
import AuthLoading from './loading'

// Used automatically by Next.js as loading UI for auth routes
export default function AuthPage() {
  return (
    <Suspense fallback={<AuthLoading />}>
      <AuthContent />
    </Suspense>
  )
}

// Or manually in components
function AuthWrapper() {
  const [isLoading, setIsLoading] = useState(true)
  
  if (isLoading) {
    return <AuthLoading />
  }
  
  return <AuthForm />
}
```

The component uses Tailwind CSS classes and ODS design tokens to create pixel-perfect skeleton placeholders that match the actual auth page structure, including form fields with 60px height, proper gap spacing, and responsive breakpoints.
