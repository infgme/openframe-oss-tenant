Utility functions for converting MeshCentral file system data structures to UI-compatible formats and handling file operations in the OpenFrame file manager.

## Key Components

- **convertFileEntryToItem()** - Transforms MeshCentral FileEntry objects to UI FileItem format with proper path normalization
- **convertFileEntriesToItems()** - Batch conversion of FileEntry arrays to FileItem arrays
- **formatFileSize()** - Converts byte values to human-readable size strings (B, KB, MB, GB, TB)
- **formatDate()** - Transforms Unix timestamps to MM/DD/YYYY HH:MM format
- **sanitizePath()** - Prevents path traversal attacks by filtering out dangerous path components
- **Path utilities** - Helper functions for path manipulation (getParentPath, joinPath, getFileExtension)
- **isDownloadable()** - Determines if a file type supports download operations

## Usage Example

```typescript
import { convertFileEntryToItem, formatFileSize, sanitizePath } from './file-manager-utils'
import type { FileEntry } from '@lib/meshcentral/file-manager-types'

// Convert MeshCentral data to UI format
const meshEntry: FileEntry = {
  n: 'document.pdf',
  t: 3, // file type
  s: 2048576, // size in bytes
  d: Date.now() // timestamp
}

const uiItem = convertFileEntryToItem(meshEntry, '/home/user')
// Result: { id: '/home/user/document.pdf', name: 'document.pdf', type: 'file', size: '2 MB', ... }

// Format file size
const sizeString = formatFileSize(1536) // "1.5 KB"

// Sanitize user input path
const safePath = sanitizePath('../../../etc/passwd') // ""
```
