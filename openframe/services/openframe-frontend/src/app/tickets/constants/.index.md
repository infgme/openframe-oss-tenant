A comprehensive constants file that defines all the essential enums, types, and configuration values for the Flamingo chat/dialog system, providing type-safe constants for status management, API endpoints, and network configuration.

## Key Components

- **Dialog Management**: `DIALOG_STATUS` enum for tracking conversation states (ON_HOLD, RESOLVED, ACTIVE)
- **Message System**: `MESSAGE_TYPE` enum covering text, tool execution, approvals, and system messages
- **User Types**: `OWNER_TYPE` and `CHAT_TYPE` for distinguishing between clients, admins, and AI assistants
- **Assistant Configuration**: `ASSISTANT_CONFIG` defining Fae (client AI) and Mingo (admin AI) assistants
- **API Endpoints**: `API_ENDPOINTS` constants for chat API routes
- **Network Configuration**: `NETWORK_CONFIG` with timeouts, reconnection settings, and message limits
- **Storage & Messaging**: `STORAGE_KEYS` for token management and `NATS_TOPICS` for pub/sub messaging

## Usage Example

```typescript
import { DIALOG_STATUS, MESSAGE_TYPE, ASSISTANT_CONFIG, type DialogStatus, type MessageType } from './index'

// Type-safe status checking
const isDialogActive = (status: DialogStatus) => status === DIALOG_STATUS.ACTIVE

// Message type validation
const isToolExecution = (type: MessageType) => 
  type === MESSAGE_TYPE.EXECUTING_TOOL || type === MESSAGE_TYPE.EXECUTED_TOOL

// Assistant configuration
const faeConfig = ASSISTANT_CONFIG.FAE
console.log(`Assistant: ${faeConfig.name} (${faeConfig.type})`)

// API endpoint usage
fetch(API_ENDPOINTS.SEND_MESSAGE, {
  method: 'POST',
  body: JSON.stringify({ message: 'Hello' })
})
```
