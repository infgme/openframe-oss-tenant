A React hook that provides functionality to update organization data via API calls, reusing the same request schema as organization creation.

## Key Components

- **`useUpdateOrganization`** - Main hook that returns an `updateOrganization` function
- **`updateOrganization`** - Async function that sends PUT request to update organization by ID
- **`CreateOrganizationRequest`** - Imported type definition for the request payload structure

## Usage Example

```typescript
import { useUpdateOrganization } from './use-update-organization'

function EditOrganizationForm({ organizationId }: { organizationId: string }) {
  const { updateOrganization } = useUpdateOrganization()
  
  const handleSubmit = async (formData: CreateOrganizationRequest) => {
    try {
      const updatedOrganization = await updateOrganization(organizationId, {
        name: formData.name,
        description: formData.description,
        // ... other organization fields
      })
      console.log('Organization updated:', updatedOrganization)
    } catch (error) {
      console.error('Failed to update organization:', error.message)
    }
  }

  return (
    // Your form JSX here
    <form onSubmit={handleSubmit}>
      {/* Form fields */}
    </form>
  )
}
```

The hook uses `useCallback` for performance optimization and follows REST conventions with PUT method for updates. Error handling includes both API error messages and fallback HTTP status codes.
