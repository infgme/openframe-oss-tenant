A Data Transfer Object (DTO) representing a script entity in the OpenFrame platform, containing metadata and configuration for executable scripts across different platforms.

## Key Components

- **Script Metadata**: `id`, `name`, `description`, `category`, `filename`
- **Execution Properties**: `shell`, `args`, `scriptBody`, `defaultTimeout`, `runAsUser`
- **Platform Support**: `supportedPlatforms`, `syntax`
- **Security & Environment**: `scriptHash`, `envVars`, `hidden`
- **User Preferences**: `favorite`, `scriptType`

## Usage Example

```java
Script deploymentScript = Script.builder()
    .name("Deploy Application")
    .description("Automated deployment script")
    .shell("bash")
    .scriptBody("#!/bin/bash\necho 'Deploying...'")
    .scriptType("deployment")
    .defaultTimeout(300)
    .supportedPlatforms(List.of("linux", "macos"))
    .runAsUser(true)
    .envVars(List.of("APP_ENV=production", "DEBUG=false"))
    .category("automation")
    .favorite(false)
    .hidden(false)
    .build();

// JSON serialization will use snake_case for annotated fields
// e.g., scriptType -> script_type, defaultTimeout -> default_timeout
```

The class uses Lombok annotations for automatic getter/setter generation and Jackson annotations for proper JSON serialization with snake_case naming conventions for API compatibility.
