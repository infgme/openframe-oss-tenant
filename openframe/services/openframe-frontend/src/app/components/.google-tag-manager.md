A client-side React component that integrates Google Tag Manager (GTM) into Next.js applications with proper script loading and fallback support.

## Key Components

- **GoogleTagManager** - Main component that renders GTM scripts and initializes the data layer
- **GoogleTagManagerProps** - Interface defining optional `containerId` prop
- **Runtime configuration integration** - Uses `runtimeEnv.gtmContainerId()` for default container ID
- **DataLayer initialization** - Sets up `window.dataLayer` for GTM event tracking
- **Noscript fallback** - Provides iframe-based tracking for users with JavaScript disabled

## Usage Example

```typescript
import { GoogleTagManager } from '@/components/google-tag-manager'

// Use with default container ID from runtime config
export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html>
      <head>
        <GoogleTagManager />
      </head>
      <body>{children}</body>
    </html>
  )
}

// Use with custom container ID
export function CustomAnalytics() {
  return <GoogleTagManager containerId="GTM-CUSTOM123" />
}
```

The component follows GTM best practices by loading the script with `afterInteractive` strategy to avoid blocking page rendering, initializing the global dataLayer array, and providing a noscript iframe fallback for accessibility and comprehensive tracking coverage.
