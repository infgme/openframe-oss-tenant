Configuration module that defines tab items and utilities for a policies and queries interface in the OpenFrame frontend.

## Key Components

- **`POLICIES_AND_QUERIES_TABS`** - Array of `TabItem` objects defining the policies and queries tabs with their icons and components
- **`getPoliciesAndQueriesTab()`** - Function to retrieve a specific tab by ID
- **`getTabComponent()`** - Function to extract the React component from a tab by ID

## Usage Example

```typescript
import { 
  POLICIES_AND_QUERIES_TABS, 
  getPoliciesAndQueriesTab, 
  getTabComponent 
} from './policies-and-queries-tabs'

// Render all tabs in a tab navigation
const TabNavigation = () => (
  <div>
    {POLICIES_AND_QUERIES_TABS.map(tab => (
      <button key={tab.id}>
        <tab.icon />
        {tab.label}
      </button>
    ))}
  </div>
)

// Get a specific tab and render its component
const TabContent = ({ activeTabId }: { activeTabId: string }) => {
  const Component = getTabComponent(activeTabId)
  return Component ? <Component /> : <div>Tab not found</div>
}

// Check if a tab exists
const policiesTab = getPoliciesAndQueriesTab('policies')
if (policiesTab) {
  console.log(`Found tab: ${policiesTab.label}`)
}
```

The module exports a pre-configured set of tabs for managing IT policies and security queries, commonly used in MSP dashboards for compliance and monitoring workflows.
