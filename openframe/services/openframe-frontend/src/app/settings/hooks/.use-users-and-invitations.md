A React hook that provides a unified interface for managing both users and invitations data, combining them into a single normalized format for easier UI rendering and management.

## Key Components

- **`UnifiedUserRecord`** - Normalized type that represents both users and invitations with consistent interface
- **`RecordType`** - Enum distinguishing between user and invitation records
- **`useUsersAndInvitations`** - Main hook combining user and invitation data with unified operations
- **Helper functions** - `userToUnified()` and `invitationToUnified()` for data transformation
- **Re-exported types** - Convenient access to underlying user and invitation types

## Usage Example

```typescript
import { useUsersAndInvitations, RecordType } from './use-users-and-invitations'

function UserManagementTable() {
  const {
    records,
    isLoading,
    deleteUser,
    revokeInvitation,
    refetchAll,
    combinedTotalElements
  } = useUsersAndInvitations(0, 20)

  if (isLoading) return <div>Loading...</div>

  return (
    <div>
      <h2>Total Records: {combinedTotalElements}</h2>
      {records.map(record => (
        <div key={record.id}>
          <span>{record.email} ({record.type})</span>
          {record.type === RecordType.USER ? (
            <button onClick={() => deleteUser.mutate(record.id)}>
              Delete User
            </button>
          ) : (
            <button onClick={() => revokeInvitation.mutate(record.id)}>
              Revoke Invitation
            </button>
          )}
        </div>
      ))}
    </div>
  )
}
```

The hook automatically handles pagination, loading states, and provides both individual and combined mutation operations for comprehensive user/invitation management.
