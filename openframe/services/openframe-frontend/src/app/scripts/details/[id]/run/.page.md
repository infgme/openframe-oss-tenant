A Next.js page component that renders a script execution interface for a specific script identified by URL parameters.

## Key Components

- **RunScriptPage**: Main page component that extracts script ID from URL params and renders the script execution interface
- **AppLayout**: Layout wrapper component providing consistent page structure
- **RunScriptView**: Core component handling script execution functionality
- **useParams**: Next.js hook for accessing dynamic route parameters

## Usage Example

This page is typically accessed through a dynamic route like `/scripts/[id]/run`:

```typescript
// URL: /scripts/abc123/run
// The component will extract "abc123" as the script ID

function RunScriptPage() {
  const params = useParams<{ id?: string }>()
  const id = typeof params?.id === 'string' ? params.id : ''
  
  return (
    <AppLayout>
      <RunScriptView scriptId={id} />
    </AppLayout>
  )
}
```

The page safely handles cases where the ID parameter might be undefined or an array by providing a fallback empty string. The extracted script ID is passed to `RunScriptView` which manages the actual script execution interface and functionality.
