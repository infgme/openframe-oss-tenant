Service class for managing chat API operations and dialog state in the Flamingo platform. Handles dialog creation, message sending, and maintains reactive state subscriptions for authentication tokens and API URLs.

## Key Components

- **ChatApiService** - Main service class providing chat API functionality
- **createDialog()** - Creates new chat dialogs and manages dialog ID state
- **sendMessage()** - Sends messages to existing chat dialogs
- **onDialogIdUpdate()** - Reactive subscription system for dialog ID changes
- **Token Integration** - Automatic token refresh and API URL management via tokenService
- **Debug Mode** - Optional detailed error logging and network debugging

## Usage Example

```typescript
import { ChatApiService } from './chatApiService'

// Initialize the service with debug mode
const chatService = new ChatApiService(true)

// Subscribe to dialog ID changes
const unsubscribe = chatService.onDialogIdUpdate((dialogId) => {
  console.log('Current dialog:', dialogId)
})

// Create a new chat dialog
const dialogId = await chatService.createDialog()

// Send a message to the dialog
await chatService.sendMessage({
  dialogId,
  content: 'Hello, I need help with my server setup',
  chatType: 'CLIENT_CHAT'
})

// Clean up subscriptions when done
chatService.destroy()
unsubscribe()
```

The service automatically handles authentication via the tokenService integration and provides error handling with optional debug logging for network operations.
