HTTP client for downloading platform-specific tool agent files from the OpenFrame/Flamingo platform's tool agent service.

## Key Components

- **ToolAgentFileClient**: Main struct that wraps an HTTP client and base URL for making requests
- **new()**: Constructor that initializes the client with HTTP client and base URL
- **get_tool_agent_file()**: Downloads tool agent files for the current platform (Windows/Mac)

## Usage Example

```rust
use reqwest::Client;
use tool_agent_file_client::ToolAgentFileClient;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let http_client = Client::new();
    let base_url = "https://api.flamingo.com".to_string();
    
    let client = ToolAgentFileClient::new(http_client, base_url);
    
    // Download tool agent file for current platform
    let asset_id = "tool-agent-v1.2.0".to_string();
    let file_bytes = client.get_tool_agent_file(asset_id).await?;
    
    // Save or process the downloaded file
    std::fs::write("tool-agent", file_bytes)?;
    
    Ok(())
}
```

The client automatically detects the operating system and requests the appropriate binary (Windows or Mac). It handles HTTP errors gracefully and returns the file content as bytes for further processing or storage.
