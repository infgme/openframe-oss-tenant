A Spring Boot application entry point that initializes the OpenFrame streaming service with Kafka integration and multi-package component scanning.

## Key Components

- **StreamApplication** - Main Spring Boot application class with Kafka enabled
- **@EnableKafka** - Enables Kafka consumer and producer functionality
- **@ComponentScan** - Scans multiple OpenFrame packages for Spring components
- **main()** - Standard Spring Boot application entry point

## Usage Example

```java
// Run the application
public class StreamApplication {
    public static void main(String[] args) {
        SpringApplication.run(StreamApplication.class, args);
    }
}

// The application will automatically scan and wire components from:
// - com.openframe.stream (streaming services)
// - com.openframe.data (data layer components)
// - com.openframe.kafka.producer (Kafka producer beans)
```

## Configuration

The application uses Spring Boot's auto-configuration with Kafka enabled by default. Component scanning includes:

- **stream** - Core streaming logic and services
- **data** - Data access and persistence components  
- **kafka.producer** - Message publishing infrastructure

This setup provides a complete streaming microservice foundation for the OpenFrame platform's real-time data processing capabilities.
