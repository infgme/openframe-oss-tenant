A React component that displays comprehensive device information in a structured card layout with error handling for missing device data.

## Key Components

- **DeviceInfoSection** - Main component that renders device information in a responsive grid layout
- **OSTypeLabel** - Imported component from core library for displaying formatted OS type labels
- **Device** - TypeScript interface defining the device data structure
- **DeviceInfoSectionProps** - Interface defining component props with nullable device parameter

## Usage Example

```typescript
import { DeviceInfoSection } from './device-info-section'
import { Device } from '../types/device.types'

const ExampleUsage = () => {
  const deviceData: Device = {
    type: 'Desktop',
    manufacturer: 'Dell',
    model: 'OptiPlex 7090',
    serialNumber: 'ABC123XYZ',
    hostname: 'workstation-01',
    organization: 'Acme Corp',
    platform: 'Windows',
    os_version: '11',
    updatedAt: '2024-01-15T10:30:00Z',
    needs_reboot: false,
    macAddress: '00:11:22:33:44:55'
  }

  return (
    <div>
      <h2>Device Information</h2>
      <DeviceInfoSection device={deviceData} />
    </div>
  )
}
```

The component features a three-section layout displaying hardware specs, network information, and operating system details. It gracefully handles missing data by showing "Unknown" placeholders and includes proper date formatting for timestamps. The responsive design adapts from single-column on mobile to four-column grid on larger screens.
