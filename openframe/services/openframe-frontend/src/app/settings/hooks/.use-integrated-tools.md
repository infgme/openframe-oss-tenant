A React hook for fetching and managing integrated tools data from the OpenFrame GraphQL API with built-in loading states and error handling.

## Key Components

- **`useIntegratedTools()`** - Main hook that provides tools data and fetching functionality
- **`IntegratedTool`** - TypeScript interface defining the structure of integrated tool objects
- **`ToolUrl`**, **`ApiKey`**, **`Credentials`** - Supporting type definitions for tool configuration
- **`GET_INTEGRATED_TOOLS_QUERY`** - GraphQL query string for fetching tools with filtering and search capabilities

## Usage Example

```typescript
import { useIntegratedTools } from './hooks/use-integrated-tools'

function ToolsManagement() {
  const { tools, isLoading, error, fetchIntegratedTools } = useIntegratedTools()

  // Fetch all enabled tools
  const loadTools = async () => {
    await fetchIntegratedTools({ enabled: true, category: null }, '')
  }

  // Search for specific tools
  const searchTools = async (query: string) => {
    await fetchIntegratedTools({ enabled: true }, query)
  }

  if (isLoading) return <div>Loading tools...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      {tools.map(tool => (
        <div key={tool.id}>
          <h3>{tool.name}</h3>
          <p>{tool.description}</p>
          <span>Category: {tool.category}</span>
        </div>
      ))}
    </div>
  )
}
```

The hook automatically manages loading states, error handling with toast notifications, and provides a clean interface for fetching integrated tools with optional filtering and search parameters.
