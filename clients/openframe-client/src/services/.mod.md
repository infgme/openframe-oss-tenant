This module serves as the main service layer aggregator for the Flamingo agent, organizing and exposing all core services required for agent functionality, communication, and management within the OpenFrame platform.

## Key Components

The module exports 27 distinct services organized into functional categories:

**Authentication & Configuration:**
- `AgentAuthService` - Handles agent authentication
- `AgentConfigurationService` - Manages agent configuration
- `InitialConfigurationService` - Handles initial setup configuration
- `AgentRegistrationService` - Manages agent registration process
- `LocalTlsConfigProvider` - Provides TLS configuration
- `EncryptionService` - Handles data encryption/decryption
- `SharedTokenService` - Manages shared authentication tokens

**Tool Management:**
- `ToolInstallationService` - Installs new tools
- `ToolUninstallService` - Removes installed tools
- `ToolKillService` - Terminates running tools
- `InstalledToolsService` - Tracks installed tools
- `ToolRunManager` - Manages tool execution

**Communication & Messaging:**
- `NatsConnectionManager` - Manages NATS connections
- `NatsMessagePublisher` - Publishes messages via NATS
- `ToolConnectionMessagePublisher` - Publishes tool connection events
- `InstalledAgentMessagePublisher` - Publishes agent installation events
- `MachineHeartbeatPublisher` - Publishes system heartbeats

**Updates & Maintenance:**
- `OpenFrameClientUpdateService` - Updates OpenFrame client
- `ToolAgentUpdateService` - Updates tool agents
- `UpdateStateService` - Manages update state
- `UpdateCleanupService` - Cleans up after updates
- `UpdateHandlerService` - Orchestrates update processes
- `GithubDownloadService` - Downloads updates from GitHub

## Usage Example

```rust
use crate::services::{
    AgentAuthService,
    ToolInstallationService,
    NatsConnectionManager
};

// Initialize core services
let auth_service = AgentAuthService::new(config);
let tool_service = ToolInstallationService::new();
let nats_manager = NatsConnectionManager::new(connection_config);

// Services work together for agent operations
let authenticated = auth_service.authenticate().await?;
if authenticated {
    tool_service.install_tool("monitoring-tool").await?;
}
```
