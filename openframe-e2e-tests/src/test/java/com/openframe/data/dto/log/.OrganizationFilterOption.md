A simple data transfer object (DTO) that represents an organization option for filtering purposes in the log module. This class encapsulates basic organization identification data used in filter dropdowns and selection interfaces.

## Key Components

- **id** - Unique identifier for the organization
- **name** - Display name of the organization
- **Lombok annotations** - Provides automatic generation of getters, setters, constructors, and builder pattern

## Usage Example

```java
// Create using builder pattern
OrganizationFilterOption option = OrganizationFilterOption.builder()
    .id("org-123")
    .name("Acme Corporation")
    .build();

// Create using constructor
OrganizationFilterOption option2 = new OrganizationFilterOption("org-456", "TechCorp Inc");

// Access properties
String orgId = option.getId();
String orgName = option.getName();

// Use in filter collections
List<OrganizationFilterOption> filterOptions = Arrays.asList(
    new OrganizationFilterOption("org-1", "Company A"),
    new OrganizationFilterOption("org-2", "Company B")
);
```

This DTO is typically used in REST API responses for populating organization filter dropdowns in the OpenFrame UI, allowing users to filter logs by specific organizations.
