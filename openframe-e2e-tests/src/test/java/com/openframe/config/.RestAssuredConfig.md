Configures REST Assured for automated API testing in the OpenFrame platform. Sets up base URI, logging, SSL handling, and HTTP client timeouts for consistent test execution.

## Key Components

- **`configure()`** - Default configuration using ApiConfig base URL
- **`configure(String baseUrl)`** - Custom base URL configuration with comprehensive settings
- **`DEFAULT_TIMEOUT`** - 60-second timeout constant for HTTP connections

## Configuration Features

- **Logging**: Enables request/response logging on validation failures
- **SSL**: Relaxed HTTPS validation for testing environments
- **Timeouts**: Connection and socket timeouts set to 60 seconds
- **Base URI**: Automatic base URL setup from ApiConfig

## Usage Example

```java
// Basic setup using default configuration
RestAssuredConfig.configure();

// Custom base URL setup
RestAssuredConfig.configure("https://api.staging.openframe.ai");

// After configuration, use REST Assured normally
given()
    .when()
    .get("/health")
    .then()
    .statusCode(200);
```

The configuration applies globally to all REST Assured requests in the test suite, ensuring consistent behavior across API tests.
