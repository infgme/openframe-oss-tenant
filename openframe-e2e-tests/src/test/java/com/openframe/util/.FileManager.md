A utility class that provides simple file-based persistence for Java objects using JSON serialization. It handles reading and writing objects to files with automatic JSON conversion.

## Key Components

- **save(String, Object)** - Serializes any object to JSON and writes it to a specified file
- **read(String, Class<T>)** - Reads JSON content from a file and deserializes it to the specified type
- **objectMapper** - Static Jackson ObjectMapper instance for JSON processing

## Usage Example

```java
// Save an object to file
User user = new User("john", "john@example.com");
FileManager.save("user.json", user);

// Read object from file
User loadedUser = FileManager.read("user.json", User.class);

// Works with any serializable object
List<String> items = Arrays.asList("item1", "item2", "item3");
FileManager.save("items.json", items);
List<String> loadedItems = FileManager.read("items.json", List.class);
```

**Note:** The class uses runtime exceptions for error handling, so IOException instances are wrapped in RuntimeException. Files are created in the current working directory unless an absolute path is provided.
