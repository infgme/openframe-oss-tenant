A React hook that provides client-side chunk catchup functionality for real-time chat dialogs, wrapping the core chunk catchup logic with API integration for fetching missed message chunks.

## Key Components

- **`useChunkCatchup`** - Main hook that manages chunk synchronization for chat dialogs
- **`fetchChunks`** - Callback function that retrieves chunks from the API with optional sequence filtering
- **Type exports** - Re-exports core types (`ChunkData`, `NatsMessageType`, `UseChunkCatchupReturn`)
- **`UseChunkCatchupOptions`** - Interface defining the required dialog ID and chunk handler

## Usage Example

```typescript
import { useChunkCatchup, type ChunkData, type NatsMessageType } from './use-chunk-catchup'

function ChatDialog({ dialogId }: { dialogId: string }) {
  const handleChunkReceived = useCallback((chunk: ChunkData, messageType: NatsMessageType) => {
    console.log('Received chunk:', chunk)
    // Process the chunk (update UI, store in state, etc.)
  }, [])

  const { 
    isConnected, 
    connectionStatus, 
    lastProcessedSequence 
  } = useChunkCatchup({
    dialogId,
    onChunkReceived: handleChunkReceived
  })

  return (
    <div>
      <div>Connection: {isConnected ? 'Connected' : 'Disconnected'}</div>
      <div>Last sequence: {lastProcessedSequence}</div>
      {/* Chat messages UI */}
    </div>
  )
}
```

The hook automatically handles both CLIENT and ADMIN chat types, fetches missed chunks from the API, and provides real-time synchronization status for chat dialogs.
