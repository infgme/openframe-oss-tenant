A custom React hook collection that provides real-time statistics for different dashboard sections by fetching and aggregating data from various GraphQL endpoints.

## Key Components

- **`useDevicesOverview`** - Fetches device statistics including total count, active/inactive devices with percentages
- **`useChatsOverview`** - Retrieves chat/ticket statistics with status counts, resolution times, and Fae ratings  
- **`useLogsOverview`** - Aggregates log data by severity levels (info, warning, critical) with distribution percentages

## Usage Example

```typescript
import { useDevicesOverview, useChatsOverview, useLogsOverview } from './use-dashboard-stats'

function DashboardStats() {
  const devices = useDevicesOverview()
  const chats = useChatsOverview() 
  const logs = useLogsOverview()

  if (devices.isLoading) {
    return <div>Loading devices...</div>
  }

  return (
    <div className="dashboard-stats">
      <div>
        <h3>Devices</h3>
        <p>Total: {devices.total}</p>
        <p>Active: {devices.active} ({devices.activePercentage}%)</p>
        <p>Inactive: {devices.inactive} ({devices.inactivePercentage}%)</p>
      </div>

      <div>
        <h3>Support Tickets</h3>
        <p>Total: {chats.total}</p>
        <p>Active: {chats.active} ({chats.activePercentage}%)</p>
        <p>Avg Resolve Time: {chats.avgResolveTime}</p>
        <p>Avg Fae Rating: {chats.avgFaeRate}</p>
      </div>

      <div>
        <h3>System Logs</h3>
        <p>Total: {logs.total}</p>
        <p>Info: {logs.info} ({logs.infoPercentage}%)</p>
        <p>Warning: {logs.warning} ({logs.warningPercentage}%)</p>
        <p>Critical: {logs.critical} ({logs.criticalPercentage}%)</p>
      </div>
    </div>
  )
}
```

Each hook handles GraphQL API calls with proper error handling and cleanup to prevent memory leaks. The hooks return computed statistics with percentage breakdowns for dashboard visualization components.
