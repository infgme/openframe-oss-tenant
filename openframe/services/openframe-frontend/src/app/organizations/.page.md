A client-side Next.js page component that displays an organizations management interface with a table view of organizations.

## Key Components

- **Organizations** - Default export function that renders the main organizations page
- **dynamic** - Export configuration forcing dynamic rendering
- **AppLayout** - Wrapper component providing consistent page layout
- **OrganizationsTable** - Table component displaying organization data

## Usage Example

```typescript
// This page is accessed via routing (e.g., /organizations)
// and renders automatically in Next.js applications

// The component structure:
function Organizations() {
  return (
    <AppLayout>
      <div className="space-y-6">
        <OrganizationsTable/>
      </div>
    </AppLayout>
  )
}
```

## Configuration

The page uses `'use client'` directive for client-side rendering and `dynamic = 'force-dynamic'` to ensure the page is always rendered on-demand rather than being statically generated. The layout includes spacing utilities (`space-y-6`) for consistent vertical spacing between elements.
