A React page component for creating and editing automation scripts in the Flamingo platform's tactical management system. The page supports multi-platform script configuration with syntax highlighting, argument management, and environment variable setup.

## Key Components

- **EditScriptPage** - Main page component for script creation/editing
- **ScriptData** - Interface defining script configuration structure
- **Platform Selection** - Multi-platform support toggle (Windows, Linux, macOS)
- **Code Editor** - Textarea with line numbering and scroll synchronization
- **Argument Manager** - Dynamic script argument configuration
- **Environment Variables** - Key-value pair management for script environment

## Usage Example

```typescript
import { EditScriptPage } from './edit-script-page'

// For creating new script
function NewScript() {
  return <EditScriptPage scriptId={null} />
}

// For editing existing script
function EditScript() {
  const scriptId = "script-123"
  return <EditScriptPage scriptId={scriptId} />
}

// Script data structure
const exampleScriptData: ScriptData = {
  name: 'System Health Check',
  shell: 'powershell',
  default_timeout: 90,
  args: [{ name: 'verbose', value: '' }],
  script_body: 'Get-ComputerInfo',
  run_as_user: false,
  env_vars: [{ name: 'LOG_LEVEL', value: 'INFO' }],
  description: 'Checks system health metrics',
  supported_platforms: ['windows', 'linux'],
  category: 'System Maintenance'
}
```

The component integrates with the tactical API client for CRUD operations and provides real-time validation, platform-specific shell type selection, and comprehensive script management functionality within the OpenFrame ecosystem.
