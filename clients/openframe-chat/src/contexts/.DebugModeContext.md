A React context provider that manages the application's debug mode state, synchronized with the Tauri backend system settings.

## Key Components

- **DebugModeContext** - React context for sharing debug mode state across components
- **DebugModeProvider** - Provider component that fetches initial debug state from Tauri backend and provides state management
- **useDebugMode** - Custom hook for accessing debug mode state and setter function with proper error handling
- **DebugModeContextType** - TypeScript interface defining the context structure with boolean state and setter

## Usage Example

```typescript
// Wrap your app with the provider
import { DebugModeProvider } from './DebugModeContext'

function App() {
  return (
    <DebugModeProvider>
      <MainApplication />
    </DebugModeProvider>
  )
}

// Use the hook in any component
import { useDebugMode } from './DebugModeContext'

function DeveloperTools() {
  const { debugMode, setDebugMode } = useDebugMode()
  
  return (
    <div>
      <label>
        <input 
          type="checkbox" 
          checked={debugMode}
          onChange={(e) => setDebugMode(e.target.checked)}
        />
        Enable Debug Mode
      </label>
      {debugMode && <AdvancedDebuggingPanel />}
    </div>
  )
}
```

The context automatically fetches the initial debug state from the Tauri backend via the `get_debug_mode` command and provides safe error handling with console logging for debugging purposes.
