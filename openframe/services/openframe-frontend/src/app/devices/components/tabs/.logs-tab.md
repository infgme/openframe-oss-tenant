A client-side React component that renders device-specific logs in a tabbed interface, providing filtered log viewing and refresh functionality for individual devices.

## Key Components

- **LogsTab**: Main component that filters and displays logs for a specific device
- **LogsTableRef**: Reference type for accessing LogsTable methods programmatically
- **deviceId resolution**: Uses `machineId` as primary identifier, falling back to `id`
- **URL-based refresh**: Monitors search parameters to trigger log table refreshes

## Usage Example

```typescript
// Basic usage in a device details page
import { LogsTab } from './logs-tab'

function DeviceDetailsPage() {
  const device = {
    id: 'device-123',
    machineId: 'machine-456',
    name: 'Server-01'
  }

  return (
    <div className="device-tabs">
      <LogsTab device={device} />
    </div>
  )
}

// Triggering refresh via URL parameter
function DeviceActions() {
  const router = useRouter()
  
  const refreshLogs = () => {
    router.push(`/devices/${deviceId}?refresh=${Date.now()}`)
  }

  return (
    <button onClick={refreshLogs}>
      Refresh Device Logs
    </button>
  )
}
```

The component automatically filters logs by device ID and provides embedded table functionality with programmatic refresh capabilities through URL parameters.
