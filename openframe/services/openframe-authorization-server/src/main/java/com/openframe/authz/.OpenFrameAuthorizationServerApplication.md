Main entry point for the OpenFrame authorization server, a Spring Boot microservice that provides centralized authentication and authorization for the Flamingo MSP platform.

## Key Components

- **@SpringBootApplication** - Enables auto-configuration and component scanning for Spring Boot
- **@EnableDiscoveryClient** - Registers the service with service discovery (Eureka/Consul) for microservice communication
- **@ComponentScan** - Scans multiple packages for Spring components including core, data, and notification modules
- **main()** method - Bootstrap entry point that starts the authorization server

## Usage Example

```java
// Run the authorization server
public class OpenFrameAuthorizationServerApplication {
    public static void main(String[] args) {
        // Starts embedded Tomcat server and initializes Spring context
        SpringApplication.run(OpenFrameAuthorizationServerApplication.class, args);
    }
}

// The application will scan these packages for Spring components:
// - com.openframe.authz (authorization logic)
// - com.openframe.core (shared core functionality)  
// - com.openframe.data (data access layer)
// - com.openframe.notification (notification services)
```

This service integrates with OpenFrame's unified MSP platform to handle OAuth2/JWT token management, user authentication, and authorization policies across all Flamingo microservices.
