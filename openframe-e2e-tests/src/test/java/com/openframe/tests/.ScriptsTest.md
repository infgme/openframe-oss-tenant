Test suite for OpenFrame script management operations, validating CRUD functionality for automated scripts in the MSP platform.

## Key Components

- **ScriptsTest** - Main test class extending `AuthorizedTest` for authenticated script operations
- **testListScripts()** - Validates script listing functionality and required properties
- **testGetScript()** - Tests individual script retrieval with body content verification
- **testAddScript()** - Validates script creation workflow using generated test data
- **testEditScript()** - Tests script modification and persistence

## Usage Example

```java
// Run the full script test suite
@Test
public void runScriptTests() {
    // Tests are currently disabled - remove @Disabled annotation to run
    ScriptsTest scriptTests = new ScriptsTest();
    
    // Each test validates different aspects:
    scriptTests.testListScripts();    // Verify scripts exist and have required fields
    scriptTests.testGetScript();      // Check individual script retrieval
    scriptTests.testAddScript();      // Test script creation
    scriptTests.testEditScript();     // Validate script updates
}

// Example script validation pattern
Script script = getScript(scriptId);
assertThat(script.getId()).isNotNull();
assertThat(script.getName()).isNotEmpty();
assertThat(script.getDescription()).isNotEmpty();
assertThat(script.getScriptBody()).isNotEmpty();
```

The test suite uses static imports from `ScriptApi` and `ScriptGenerator` for API operations and test data generation.
