A React component that dynamically renders settings tab content based on the active tab selection, providing error handling for missing tabs.

## Key Components

- **`SettingsTabContent`** - Main component that renders the appropriate tab content
- **`SettingsTabContentProps`** - Interface defining the component props with `activeTab` string
- **`getTabComponent`** - Utility function from OpenFrame core for retrieving tab components
- **`SETTINGS_TABS`** - Configuration object containing available settings tabs

## Usage Example

```typescript
import { SettingsTabContent } from './settings-tab-content'

function SettingsPage() {
  const [activeTab, setActiveTab] = useState('general')

  return (
    <div className="settings-page">
      <nav>
        <button onClick={() => setActiveTab('general')}>General</button>
        <button onClick={() => setActiveTab('security')}>Security</button>
      </nav>
      <SettingsTabContent activeTab={activeTab} />
    </div>
  )
}
```

The component uses the OpenFrame frontend core's tab management system to dynamically load and render settings tabs. It includes a fallback UI that displays a "Tab Not Found" message when an invalid tab is selected, ensuring graceful error handling. The rendered content maintains a minimum height of 400px for consistent layout.
