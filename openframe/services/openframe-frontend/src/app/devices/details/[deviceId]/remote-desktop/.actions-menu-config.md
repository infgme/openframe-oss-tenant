Configuration file for remote desktop actions menu that defines keyboard shortcuts, power management, and input controls for remote system interaction.

## Key Components

- **VK (Virtual Keys)**: Constant object mapping Windows virtual key codes for keyboard input
- **ActionHandlers**: Interface defining callback functions for system actions (keyboard shortcuts, power management, input control, notifications)
- **createActionsMenuGroups()**: Factory function that generates menu configuration with three main groups:
  - Keyboard shortcuts submenu (Ctrl+Alt+Del, Windows key combinations)
  - Power management actions (wake, sleep, reboot, shutdown)
  - Input toggle control

## Usage Example

```typescript
import { createActionsMenuGroups, ActionHandlers, VK } from './actions-menu-config'

const handlers: ActionHandlers = {
  sendCtrlAltDel: () => remoteDesktop.sendCtrlAltDel(),
  sendKeyCombo: (keys) => remoteDesktop.sendKeys(keys),
  sendPower: (action) => powerManager.execute(action),
  setEnableInput: (enabled) => setInputEnabled(enabled),
  toast: (options) => showToast(options)
}

const menuGroups = createActionsMenuGroups(handlers, true)
// Pass menuGroups to ActionsMenu component
```

The configuration supports common Windows shortcuts like Win+L (lock), Win+M (minimize all), Win+R (run dialog), and power operations, with toast notifications for user feedback.
