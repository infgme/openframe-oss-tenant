A comprehensive JUnit test suite for validating OpenFrame's logging system functionality, including log retrieval, filtering, search, and detail operations.

## Key Components

**Test Class**
- `LogsTest` - Extends `AuthorizedTest` for authenticated API testing
- Currently disabled pending device availability

**Test Methods**
- `testGetLogFilters()` - Validates log filter metadata retrieval
- `testListLogs()` - Tests basic log event listing with field validation
- `testGetLogDetails()` - Verifies detailed log information retrieval
- `testSearchLogs()` - Tests log search functionality by tool type
- `testFilterLogs()` - Validates filtering by severity and tool type combinations

**Dependencies**
- Uses LogsApi static methods for API operations
- Leverages LogGenerator for test data creation
- Employs AssertJ for fluent assertions

## Usage Example

```java
// Run all log-related tests (when enabled)
@Test
public void validateLoggingSystem() {
    // Test log filters are available
    LogFilters filters = getLogFilters();
    assertThat(filters.getToolTypes()).isNotEmpty();
    
    // Test log retrieval and validation
    List<LogEvent> logs = getLogs();
    assertThat(logs).allSatisfy(log -> {
        assertThat(log.getToolEventId()).isNotNull();
        assertThat(log.getEventType()).isNotEmpty();
    });
    
    // Test filtered log retrieval
    List<LogEvent> filtered = getLogs(
        severityAndToolFilter("ERROR", "MSP_TOOL")
    );
    assertThat(filtered).allMatch(log -> 
        log.getSeverity().equals("ERROR")
    );
}
```

The test suite ensures comprehensive validation of the OpenFrame logging API's core functionality with proper error handling and data integrity checks.
