Entry point for the authentication store module, providing centralized access to authentication state management functionality.

## Key Components

**Exported Hook:**
- `useAuthStore` - Main Zustand store hook for authentication state management

**Exported Types:**
- `AuthState` - Complete authentication state interface
- `User` - User entity type definition
- `UserImage` - User profile image type definition

**Exported Selectors:**
- `selectUser` - Retrieves current authenticated user
- `selectIsAuthenticated` - Boolean authentication status
- `selectAuthLoading` - Loading state for auth operations
- `selectAuthError` - Current authentication error state

## Usage Example

```typescript
import { 
  useAuthStore, 
  selectUser, 
  selectIsAuthenticated,
  type User 
} from './auth'

function ProfileComponent() {
  const user = useAuthStore(selectUser)
  const isAuthenticated = useAuthStore(selectIsAuthenticated)
  const isLoading = useAuthStore(selectAuthLoading)

  if (isLoading) return <div>Loading...</div>
  if (!isAuthenticated) return <div>Please log in</div>

  return <div>Welcome, {user?.name}</div>
}
```

This barrel export pattern provides a clean API surface for consuming authentication functionality throughout the application while maintaining type safety and selective imports.
