A React hook for processing real-time streaming messages from Mingo AI, managing both active and background dialog states with proper streaming message handling.

## Key Components

- **useMingoRealtimeProcessor**: Main hook that processes real-time chunks from NATS messaging system
- **processChunk**: Core function that routes chunks to appropriate dialog handlers
- **processDialogChunk**: Handles different action types (message_start, text, tool_execution, etc.)
- **DialogState**: Interface tracking streaming state per dialog
- **resetDialog**: Cleans up dialog state when needed
- **getDialogStreamingMessage**: Retrieves current streaming message for a dialog

## Usage Example

```typescript
import { useMingoRealtimeProcessor } from './use-mingo-realtime-processor'

function MingoChat() {
  const { processChunk, resetDialog, getDialogStreamingMessage } = useMingoRealtimeProcessor({
    activeDialogId: 'dialog-123',
    onActiveStreamStart: () => setIsStreaming(true),
    onActiveStreamEnd: () => setIsStreaming(false),
    onActiveError: (error) => showError(error),
    onBackgroundStreamStart: (dialogId) => markAsStreaming(dialogId),
    onBackgroundStreamEnd: (dialogId) => markAsComplete(dialogId),
    onBackgroundUnreadIncrement: (dialogId) => incrementUnread(dialogId),
  })

  // Process incoming NATS chunks
  const handleNatsMessage = (chunk: ChunkData, messageType: NatsMessageType) => {
    processChunk(chunk, messageType)
  }

  // Reset dialog when switching conversations
  const handleDialogSwitch = (newDialogId: string) => {
    resetDialog(currentDialogId)
  }

  return <div>/* Chat UI */</div>
}
```

The hook handles streaming text chunks, tool executions, approval requests, and admin messages, automatically routing them to either active dialog state or background message stores based on the current active dialog.
