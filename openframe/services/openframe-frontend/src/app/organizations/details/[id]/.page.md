A Next.js page component that serves as a wrapper for displaying organization details, extracting the organization ID from the URL parameters and rendering it within the application layout.

## Key Components

- **OrganizationDetailsPageWrapper**: Main page component that handles URL parameter extraction and renders the organization details view
- **AppLayout**: Layout wrapper component that provides consistent application structure
- **OrganizationDetailsView**: Child component responsible for displaying organization-specific information
- **useParams**: Next.js hook for accessing dynamic route parameters

## Usage Example

This page component is typically used in a Next.js app router structure for organization detail routes:

```typescript
// File structure: /app/organizations/[id]/page.tsx
// Handles routes like: /organizations/123, /organizations/org-uuid

// The component automatically extracts the 'id' parameter from URLs
// and passes it to OrganizationDetailsView for data fetching and display

// Example usage in routing:
// /organizations/company-123 -> id = "company-123"
// /organizations/456 -> id = "456"
// /organizations/ -> id = "" (empty string fallback)
```

The component includes type safety for the route parameters and provides a fallback empty string when the ID parameter is not available or not a string, ensuring robust handling of dynamic routes in the Flamingo platform's organization management interface.
