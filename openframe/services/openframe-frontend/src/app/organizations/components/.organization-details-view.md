This React component displays comprehensive organization details in a structured, card-based layout within the Flamingo OpenFrame platform. It provides a full-featured organization profile view with edit/delete capabilities and responsive design.

## Key Components

- **`OrganizationDetailsView`** - Main component that renders organization details with summary cards, contact information, and service configuration
- **`useOrganizationDetails`** - Custom hook for fetching organization data by ID
- **`useDeleteOrganization`** - Custom hook for organization deletion with confirmation
- **`DetailPageContainer`** - Layout wrapper with navigation and header actions
- **Contact Cards** - Three InfoCard components for Primary, Billing, and Technical contacts
- **Service Configuration** - Financial and contract information display
- **Feature Flag Integration** - Conditional rendering based on `organizationImages.displayEnabled()`

## Usage Example

```typescript
import { OrganizationDetailsView } from './organization-details-view'

// In a Next.js page or component
export default function OrganizationPage({ params }: { params: { id: string } }) {
  return <OrganizationDetailsView id={params.id} />
}

// The component handles:
// - Loading states with CardLoader
// - Error states with LoadError/NotFoundError
// - Navigation to edit form (/organizations/edit/[id])
// - Organization deletion with toast notifications
// - Responsive grid layout for organization data
```

The component features a comprehensive organization profile with business details, contact management, financial information, and service configuration, all styled with the OpenFrame design system (ODS) classes and supporting both light/dark themes.
