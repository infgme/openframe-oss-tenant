React component that renders dynamic tab content for the policies and queries section by delegating to appropriate sub-components based on the active tab.

## Key Components

- **PoliciesAndQueriesTabContent** - Main component that renders tab content using the OpenFrame core TabContent wrapper
- **PoliciesAndQueriesTabContentProps** - Interface defining the required activeTab string prop
- **getTabComponent** - Imported function that resolves the correct component for each tab

## Usage Example

```typescript
import { PoliciesAndQueriesTabContent } from './content'

// Render content for a specific tab
function MyPage() {
  const [currentTab, setCurrentTab] = useState('policies')
  
  return (
    <div>
      <PoliciesAndQueriesTabContent activeTab={currentTab} />
    </div>
  )
}

// Use with tab navigation
function TabContainer() {
  const activeTab = useSearchParams().get('tab') || 'default'
  
  return (
    <PoliciesAndQueriesTabContent activeTab={activeTab} />
  )
}
```

The component follows the OpenFrame pattern of using a centralized TabContent wrapper with dynamic component resolution, ensuring consistent tab behavior across the policies and queries interface while maintaining modularity through the tab component registry system.
