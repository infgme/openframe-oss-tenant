A utility class that provides pre-configured DeviceFilterInput objects for common device filtering scenarios in the OpenFrame platform. This generator simplifies the creation of device filters by offering convenient factory methods for different device status combinations.

## Key Components

- **onlineDevicesFilter()** - Creates filter for online devices only
- **offlineDevicesFilter()** - Creates filter for offline devices only  
- **statusAndOSDevicesFilter()** - Creates filter combining device status and OS type
- **listedDevicesFilter()** - Creates filter for all standard device statuses
- **filterDevicesByStatus()** - Creates filter for specified device statuses (varargs)

## Usage Example

```java
import com.openframe.data.generator.DeviceGenerator;
import com.openframe.data.dto.device.DeviceStatus;

// Get filter for online devices only
DeviceFilterInput onlineFilter = DeviceGenerator.onlineDevicesFilter();

// Filter by status and OS
DeviceFilterInput windowsActiveFilter = DeviceGenerator.statusAndOSDevicesFilter(
    DeviceStatus.ACTIVE, "Windows"
);

// Filter by multiple specific statuses
DeviceFilterInput customFilter = DeviceGenerator.filterDevicesByStatus(
    DeviceStatus.ONLINE, 
    DeviceStatus.ACTIVE, 
    DeviceStatus.MAINTENANCE
);

// Get comprehensive filter for all listed devices
DeviceFilterInput allListedFilter = DeviceGenerator.listedDevicesFilter();
```

This generator eliminates boilerplate code when creating device filters and ensures consistent filter configuration across the application.
