A React hook for fetching and managing organization details from the OpenFrame platform, providing comprehensive organization data including contact information, addresses, and contract details.

## Key Components

- **OrganizationDetails Interface**: Complete type definition for organization data including contacts, addresses, and financial information
- **useOrganizationDetails Hook**: Main hook that manages organization state and provides fetch functionality
- **fetchOrganizationById**: Async function to retrieve organization data by ID via GraphQL
- **formatAddress Helper**: Utility function to format address objects into readable strings

## Usage Example

```typescript
import { useOrganizationDetails } from './use-organization-details'

function OrganizationProfile({ organizationId }: { organizationId: string }) {
  const { organization, isLoading, error, fetchOrganizationById } = useOrganizationDetails()

  useEffect(() => {
    fetchOrganizationById(organizationId)
  }, [organizationId, fetchOrganizationById])

  if (isLoading) return <div>Loading organization details...</div>
  if (error) return <div>Error: {error}</div>
  if (!organization) return <div>Organization not found</div>

  return (
    <div>
      <h2>{organization.name}</h2>
      <p>Industry: {organization.industry}</p>
      <p>Employees: {organization.employees}</p>
      <p>MRR: ${organization.mrrUsd}</p>
      <div>
        <h3>Primary Contact</h3>
        <p>{organization.primary.name} - {organization.primary.email}</p>
      </div>
    </div>
  )
}
```

The hook handles error states with toast notifications and provides comprehensive organization data mapping from GraphQL responses to a structured TypeScript interface.
