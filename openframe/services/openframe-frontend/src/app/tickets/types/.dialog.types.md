Defines TypeScript interfaces and types for the dialog system that handles client-AI assistant conversations and support tickets in the OpenFrame platform.

## Key Components

**Core Types:**
- `DialogStatus` - Status states for dialogs (ACTIVE, ACTION_REQUIRED, ON_HOLD, RESOLVED, ARCHIVED)
- `Dialog` - Main dialog interface with metadata, ownership, and status tracking
- `Machine` - Client machine information for dialog context
- `DialogRating` - User feedback system for dialog resolution quality

**Message System:**
- `Message` - Individual message structure with ownership and typed data
- `MessageOwnerType` - Message sender types (CLIENT, ASSISTANT, ADMIN)
- `MessageDataType` - Message content types including tool execution and approvals

**Pagination:**
- `DialogConnection` & `MessageConnection` - Cursor-based pagination for dialogs and messages
- `CursorPageInfo` - Pagination metadata with next/previous page indicators

**Tool Integration:**
- `ExecutingToolData` & `ExecutedToolData` - Tool execution tracking with approval workflows
- `ApprovalRequestData` & `ApprovalResultData` - Admin approval system for sensitive operations
