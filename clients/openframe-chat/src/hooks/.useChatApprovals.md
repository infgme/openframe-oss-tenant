Custom React hook for managing chat approval workflows in the Flamingo AI platform. Handles approval request states, API interactions, and escalation flows for commands requiring technician approval.

## Key Components

- **State Management**: Tracks approval statuses, pending requests, and technician response states
- **API Methods**: `handleApproveRequest()` and `handleRejectRequest()` for approval decisions
- **Message Updates**: `updateApprovalStatusInMessages()` syncs approval states with chat messages
- **Escalation Handling**: Methods for managing escalated approvals requiring technician intervention
- **Cleanup**: `clearApprovals()` resets all approval-related state

## Usage Example

```typescript
import { useChatApprovals } from './useChatApprovals'

function ChatInterface() {
  const {
    approvalStatuses,
    awaitingTechnicianResponse,
    handleApproveRequest,
    handleRejectRequest,
    updateApprovalStatusInMessages,
    clearApprovals
  } = useChatApprovals()

  // Handle approval button clicks
  const onApprove = (requestId: string) => {
    handleApproveRequest(requestId)
  }

  // Update messages with approval status
  const updatedMessages = updateApprovalStatusInMessages(
    messages, 
    'req-123', 
    'approved'
  )

  // Clear all approvals on new chat
  const startNewChat = () => {
    clearApprovals()
  }
}
```

The hook integrates with the tokenService for authentication and provides a complete approval workflow system for AI-assisted technical operations.
